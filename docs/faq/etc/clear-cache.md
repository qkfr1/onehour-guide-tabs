# 캐시 삭제하기

<figure><img src="../../.gitbook/assets/스크린샷 2026-06-25 오후 5.48.25.png" alt="" width="563"><figcaption><p>"새 버전이 출시되었어요" 안내가 뜨거나, 새로고침을 해도 화면이 깨지거나 오류가 계속될 때 아래 순서대로 캐시를 비우면 해결됩니다.</p></figcaption></figure>

{% hint style="info" %}
## 왜 이런 일이 생기나요?

내 기기에 잠시 저장해 둔 옛 화면 정보가 새 버전과 맞지 않아 생기는 일시적인 현상이에요.&#x20;

캐시(임시 저장 정보)를 비우면 최신 화면으로 정상적으로 다시 불러옵니다.

캐시를 비우면 일부 사이트에서 다시 로그인이 필요할 수 있어요. 학습 데이터가 사라지는 건 아니니 안심하세요.
{% endhint %}



## PC Chrome (윈도우 · 맥 공통)

{% stepper %}
{% step %}
Chrome 앱에서 원아워를 엽니다.
{% endstep %}

{% step %}
오른쪽 위 '⋮ 더보기'를 누르고 '설정'으로 들어갑니다.
{% endstep %}

{% step %}
왼쪽 메뉴에서 '개인 정보 보호 및 보안'을 누르고 '인터넷 사용 기록 삭제'를 누릅니다.
{% endstep %}

{% step %}
기간을 '전체 기간'으로 선택합니다.
{% endstep %}

{% step %}
'쿠키 및 기타 사이트 데이터'와 '캐시된 이미지 또는 파일'에 체크합니다.
{% endstep %}

{% step %}
'인터넷 사용 기록 삭제'를 누릅니다.
{% endstep %}

{% step %}
Chrome을 닫았다가 다시 열고 원아워에 접속합니다.
{% endstep %}
{% endstepper %}

## 안드로이드 · 아이폰 · 아이패드 (Chrome)

{% stepper %}
{% step %}
Chrome 앱에서 원아워를 엽니다.
{% endstep %}

{% step %}
주소창 오른쪽 '⋮ 더보기'를 누릅니다.
{% endstep %}

{% step %}
'인터넷 사용 기록 삭제'를 누릅니다.
{% endstep %}

{% step %}
기간을 '전체 기간'으로 선택합니다.
{% endstep %}

{% step %}
'쿠키 및 사이트 데이터'와 '캐시된 이미지 및 파일'에 체크가 되어 있는지 확인합니다.
{% endstep %}

{% step %}
'데이터 삭제'를 누릅니다.
{% endstep %}

{% step %}
Chrome을 닫았다가 다시 열고 원아워에 접속합니다.
{% endstep %}
{% endstepper %}

## 아이폰 · 아이패드 (Safari)

{% stepper %}
{% step %}
'설정' 앱을 엽니다.
{% endstep %}

{% step %}
아래로 내려 '앱'을 누르고 'Safari'를 누릅니다.
{% endstep %}

{% step %}
'방문 기록 및 웹사이트 데이터 지우기'를 누릅니다.
{% endstep %}

{% step %}
안내 창에서 '방문 기록 및 데이터 지우기'를 눌러 확정합니다.
{% endstep %}

{% step %}
Safari를 완전히 닫았다가 다시 열고 원아워에 접속합니다.
{% endstep %}
{% endstepper %}

## 맥 Safari

{% stepper %}
{% step %}
화면 위 메뉴에서 'Safari'를 누르고 '설정'(또는 환경설정)을 엽니다.
{% endstep %}

{% step %}
'고급' 탭으로 이동해 맨 아래 '메뉴 막대에서 개발자용 메뉴 보기'에 체크합니다.
{% endstep %}

{% step %}
위 메뉴에 새로 생긴 '개발자용'을 누르고 '캐시 비우기'를 누릅니다.
{% endstep %}

{% step %}
Safari를 닫았다가 다시 열고 원아워에 접속합니다.
{% endstep %}
{% endstepper %}
