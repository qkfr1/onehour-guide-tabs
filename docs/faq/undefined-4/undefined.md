# 학생 클래스 이동 시 신규 클래스 과제를 부여하고 싶어요

안녕하세요, 원아워입니다.

학생을 다른 클래스로 이동시킬 때는 **기존 과제 정리 → 클래스 이동 → 신규 과제 배정** 순서로 진행해 주세요.

***

### 1단계: 기존 클래스에서 부여받은 미수행 과제 삭제하기

{% embed url="https://app.storylane.io/share/cdtmh6socnlx" %}

클래스를 이동하기 전, 기존 클래스에서 부여받은 미완료 과제를 먼저 정리해 주세요.

1. **대시보드 → 부여한 과제현황**에서 **기존 클래스**를 선택하고 **학생을 선택**합니다.
2. 캘린더에서 해당 학생에게 부여된 **미수행 과제**를 선택합니다.
3. **삭제** 버튼을 눌러 과제를 제거합니다.

***

### 2단계: 학생을 신규 클래스로 이동시키기

미수행 과제 정리가 완료되면 학생을 새로운 클래스로 이동시킵니다.

1. **클래스** 메뉴에서 학생의 현재 클래스를 선택합니다.
2. 이동할 학생을 선택한 뒤 **클래스 이동** 기능을 사용합니다.
3. 이동할 **신규 클래스**를 지정하여 이동을 완료합니다.

***

### 3단계: 신규 클래스 과제 배정하기

{% embed url="https://app.storylane.io/share/kojersecvmya" %}

클래스 이동이 완료되면 새 클래스의 과제를 배정해 주세요.

1. **대시보드 → 부여한 과제현황**에서 학생이 이동한 **신규 클래스**를 선택합니다.
2. 캘린더에서 학생에게 추가할 **과제를 선택**합니다.
3. **학생 추가하기** 버튼을 눌러 과제를 배정합니다.

***

> 💡 **TIP**: 기존 클래스의 미수행 과제를 삭제하지 않으면 학생에게 불필요한 과제가 남아있을 수 있으니, 반드시 1단계부터 순서대로 진행해 주세요!
