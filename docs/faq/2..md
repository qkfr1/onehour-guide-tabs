## 1. 수업 만들기&#x20;

원아워에서는 유튜브 영상 URL만 있으면 자동으로 수업이 만들어집니다.  \
학생 흥미와 현실 활용도가 높은 콘텐츠를 직접 고르실 수 있습니다.

<div><figure><img src="../.gitbook/assets/스크린샷 2026-05-04 오후 4.21.10.png" alt="" width="563"><figcaption></figcaption></figure> <figure><img src="../.gitbook/assets/스크린샷 2026-05-04 오후 4.29.02.png" alt="" width="563"><figcaption></figcaption></figure></div>

### 영상 고르실 때 확인할 점

* 길이는 5분 내외
* 발음이 또렷한 영상
* 너무 빠르거나 어려운 영상은 입문 단계에서 피하기

{% hint style="success" %}
추천 콘텐츠 세 가지

* **팝송**: 멜로디가 있어 연음과 리듬 훈련에 효과적
* **영화 예고편**: 1\~3분 분량으로 짧고 임팩트 있음
* **인터뷰 클립**: 자연스러운 회화 속도와 실제 표현 학습에 적합
{% endhint %}

### 수업 만드는 방법

유튜브 영상 URL을 입력하면 AI가 영상 속 소리를 자동 인식하여 학습할 문장과 단어를 추출하여 수업으로 만들어줍니다.&#x20;

자세한 방법은 👉 [유튜브 영상으로 수업 만들기](https://1hour.gitbook.io/guide/teacher/lesson-management/create) 가이드를 참고해 주세요.

## 2. 과제 만들기

> 수업이 준비되었다면 학생에게 배포할 과제를 설계할 차례입니다.&#x20;
>
> 소리 영어 학습에 가장 중요한 요소는 소리를 익숙하기 느끼도록 과제의 **분량, 반복 횟수, 배속**을 어떻게 정할지입니다.

#### 1단계. 과제 선택 — 학습 방식 정하기

소리영어 과제는 문장 학습 설정에서 **더빙**과 **동시통역**을 활용하고 배속과 쉐도잉 횟수를 지정합니다.&#x20;

| 항목     | 추천 설정 | 이유             |
| ------ | ----- | -------------- |
| 쉐도잉 횟수 | 3회 이상 | 한 번으로는 체화되지 않음 |
| 배속     | 1배속   | 원어민 속도로 학습     |

<figure><img src="../.gitbook/assets/스크린샷 2026-05-06 오후 2.16.30.png" alt="" width="563"><figcaption></figcaption></figure>



#### 2단계. 날짜 선택 — 분량과 일정 정하기

분할을 통해 문장 수를 너무 많지 않도록 조정하고, 반복할 날짜와 요일을 지정합니다.&#x20;

<figure><img src="../.gitbook/assets/스크린샷 2026-05-06 오후 2.21.26.png" alt="" width="563"><figcaption></figcaption></figure>

{% hint style="info" %}
### 과제 설계 요령

학생이 더빙 과제에 적응함에 따라 소리 영어에 익숙해질 수 있도록 같은 영상으로 단계를 바꿔가며 과제를 내주세요.

* **1단계**: 더빙 (0.75배속, 반복 3회) — 정확한 발음과 억양 익히기
* **2단계**: 더빙 (1.0배속, 반복 3회) — 원어민 속도에 맞춰 따라 하기
* **3단계**: 동시통역 — 영상 속도에 맞춰 실시간 쉐도잉 녹화

같은 영상을 1\~3단계까지 진행하시면 깊이 있게 체화할 수 있습니다.
{% endhint %}

자세한 방법은 👉 [과제 만들기](https://1hour.gitbook.io/guide/teacher/assignment-management/create) 가이드를 참고해 주세요.

## 3. 학생 과제 수행 (학습 지도)

> 과제를 배포했다면 학생이 첫 학습을 잘 시작할 수 있도록 안내가 필요합니다.&#x20;
>
> 초기에는 함께 과제를 학습하며 수행 방법을 익힐 수 있게 도와주세요.

### 더빙 과제는 이렇게 진행됩니다

<figure><img src="../.gitbook/assets/더빙 (1).gif" alt="" width="563"><figcaption></figcaption></figure>

1. 영상과 자막을 보며 소리 듣기
2. 따라 더빙하기
3. 발음 평가점수 확인하기 (다시 녹음하기)

{% hint style="info" %}
더빙 학습 전 학생이 꼭 알아야 하는 점

* **마이크 권한 허용** — 허용하지 않으면 과제를 수행할 수 없어요.
* **조용한 환경에서 녹음** — 소음이 크면 정확한 발음 체크가 불가하여 점수가 낮게 나옵니다.
* **점수에 너무 신경 쓰지 않기** — 반복하면 점차 올라갑니다.
{% endhint %}

## 4. 과제 확인 및 케어

> 학생이 학습을 시작하면 과제 관리에서 진행 상황을 확인하고 점수가 낮은 학생을 케어합니다.

<figure><img src="../.gitbook/assets/스크린샷 2026-05-06 오후 3.30.38.png" alt=""><figcaption></figcaption></figure>

{% hint style="success" %}
### 녹음 음성도 들어보세요

학습 영상은 **리포트**에서 확인하실 수 있습니다. 더빙 음성, 발음 점수, 문장별 평가 결과가 함께 제공됩니다.

학습 영상을 직접 들어보시면 점수만으로는 알 수 없는 부분까지 확인할 수 있습니다.
{% endhint %}

## FAQ

<details>

<summary>학생이 마이크가 켜지지 않는다고 합니다</summary>

브라우저에서 마이크 권한이 차단된 경우입니다.

* 브라우저 주소창 자물쇠 아이콘 클릭 → 마이크 권한 허용
* 페이지 새로고침

</details>

<details>

<summary>발음 점수가 이상하게 낮게 나옵니다</summary>

다음을 확인해 주세요.

* 주변 소음이 크지 않은지
* 마이크가 학생 입에서 멀지 않은지
* 영상 음성과 학생 음성이 겹쳐 녹음되지 않는지 (이어폰 사용 권장)

점검 후에도 점수가 계속 낮으면 기기 문제일 가능성이 있습니다. 학부모님께 **학습 기기를 지참하시고 고객센터로 문의**하시도록 안내해 주세요.

</details>

<details>

<summary>학부모가 "효과가 있나요?"라고 물어봅니다</summary>

소리영어는 단기간에 변화가 보이기 어려운 학습법입니다. 다음과 같이 설명해 보세요.

* **수치로**: 발음 점수 변화 추이를 리포트로 공유
* **음성으로**: 첫 더빙 음성과 최근 더빙 음성 비교
* **기간으로**: 한 콘텐츠를 깊이 학습하면 1\~2개월 후 변화가 나타납니다

학부모 알림 기능으로 진행 상황을 정기적으로 공유하실 수 있습니다.

</details>

<details>

<summary>학생이 학습을 안 합니다</summary>

다음을 점검해 보세요.

* 분량이 너무 많지 않은지 (하루 10문장 내외로 시작)
* 콘텐츠가 학생 흥미와 맞는지
* 첫 학습 점수가 너무 낮아 자신감을 잃지 않았는지

학습 동기가 낮은 학생은 학부모와 함께 짧은 일일 학습을 약속하는 것도 효과적입니다.

</details>
