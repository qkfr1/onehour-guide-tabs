# 책 워크시트로 랜덤 시험지 만들기

여러 변형문제 세트를 하나의 문제은행으로 모아, 원하는 수량만큼 랜덤으로 시험지를 출제하는 기능입니다.

## 이런 분께 추천해요

* 내신 시즌마다 변형문제 시험지를 여러 세트 만들어야 하는 분
* 학생마다 다른 구성의 모의고사를 제공하고 싶은 분
* 대량의 문제 풀에서 효율적으로 시험지를 뽑고 싶은 분

## 사용 방법

{% stepper %}
{% step %}
#### 변형문제 세트 준비하기

먼저 외부 지문별로 변형문제 세트를 만들어 둡니다.

예를 들어 지문 1번부터 8번까지, 각 세트당 약 20문제씩 구성하면 총 160문제 이상의 문제 풀을 확보할 수 있습니다.

<figure><img src="../../.gitbook/assets/image (25).png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
#### 문제은행용 책 만들기

1. **책** 메뉴로 이동합니다.
2. **책 생성** 버튼을 클릭합니다.
3. 책 이름을 입력합니다. (예: "내신 대비 문제은행")
4. 앞서 만들어 둔 변형문제 세트들을 선택하여 추가합니다.
5. **저장**을 클릭합니다.

{% hint style="info" %}
여러 세트를 한꺼번에 선택할 수 있습니다.
{% endhint %}

<figure><img src="../../.gitbook/assets/image (26).png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
#### 문제 통합하기

1. 생성된 책에서 **학습지** 버튼을 클릭합니다.
2. **모아서 만들기**를 선택합니다.

여러 수업(문제 세트)의 문제가 하나의 문제 풀로 통합됩니다.

<figure><img src="../../.gitbook/assets/image (27).png" alt=""><figcaption></figcaption></figure>

<br>
{% endstep %}

{% step %}
#### 랜덤으로 시험지 출제하기

1. 문제 추가 화면에서 **랜덤 섞기 선택** 버튼을 클릭합니다.
2. 원하는 문제 수량을 입력합니다.

여러 지문에서 골고루 섞인 시험지가 자동으로 구성됩니다.

{% hint style="info" %}
정답과 해설이 자동으로 포함되어, 별도로 답안지를 만들 필요가 없습니다.
{% endhint %}

## 활용 팁

* **다양한 시험지 생성**: 같은 문제은행에서 매번 다른 조합의 시험지를 무한히 만들 수 있습니다.
* **개별 맞춤 출제**: 학생마다 서로 다른 구성의 시험지를 제공할 수 있습니다.
* **반복 학습**: 동일 문제은행에서 매번 새로운 구성으로 연습 가능합니다.
* **실전 감각 향상**: 여러 지문이 섞여 출제되어 실제 학교 시험과 유사한 환경을 제공합니다.
