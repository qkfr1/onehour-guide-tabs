# 라이팅 과제를 내주고, 손글씨를 업로드하여 첨삭받도록 하고 싶어요

{% stepper %}
{% step %}
#### \[**수업관리]** 탭에서 라이팅 과제를 부여할 수업을 선택한 뒤, 해당 수업 우측에 위치한 **\[과제 내기]** 아이콘을 클릭해 주세요.

<figure><img src="../../.gitbook/assets/스크린샷 2026-04-08 오전 10.19.45.png" alt=""><figcaption></figcaption></figure>


{% endstep %}

{% step %}
#### 과제 선택 화면이 열리면 아래로 스크롤하여 **\[에세이 설정]** 영역을 확인해 주세요. 에세이 주제를 직접 입력하실 수 있으며, **\[주제 자동 생성]** 버튼을 클릭하시면 AI가 수업 내용에 기반한 에세이 주제를 자동으로 추천해 드립니다.

<figure><img src="../../.gitbook/assets/스크린샷 2026-04-08 오전 10.22.45.png" alt=""><figcaption></figcaption></figure>


{% endstep %}

{% step %}
#### 에세이 주제 입력이 완료되면, 화면 우측 하단의 **\[다음]** 버튼을 클릭하여 다음 단계로 진행해 주세요.

<figure><img src="../../.gitbook/assets/스크린샷 2026-04-08 오전 10.23.09.png" alt=""><figcaption></figcaption></figure>


{% endstep %}

{% step %}
#### 캘린더에서 과제를 시작할 날짜를 클릭하여 선택해 주세요. 선택한 날짜에 에세이 과제가 배정된 것을 확인하신 뒤, 우측 하단의 **\[다음]** 버튼을 클릭해 주세요.

<figure><img src="../../.gitbook/assets/스크린샷 2026-04-08 오전 10.23.28.png" alt=""><figcaption></figcaption></figure>


{% endstep %}

{% step %}
#### 과제를 부여할 **클래스**와 **학생**을 선택해 주시고, 우측 하단의 **\[과제 내기 완료]** 버튼을 클릭하시면 학생에게 과제가 부여됩니다.

<figure><img src="../../.gitbook/assets/스크린샷 2026-04-08 오전 10.24.08.png" alt=""><figcaption></figcaption></figure>


{% endstep %}

{% step %}
### 부여한 과제는 원아워 학생 계정으로 로그인하여 수행이 가능합니다. 로그인 페이지에서 아이디/비밀번호를 입력하여 로그인을 진행해주세요.



<figure><img src="../../.gitbook/assets/스크린샷 2026-04-08 오전 10.25.02.png" alt=""><figcaption></figcaption></figure>


{% endstep %}

{% step %}
#### 로그인 후 **\[과제보기]** 탭에서 해당 날짜에 부여된 과제 목록을 확인할 수 있습니다. 에세이 과제 항목 우측의 **\[과제 시작]** 버튼을 클릭하여 라이팅 과제를 시작합니다.

<figure><img src="../../.gitbook/assets/스크린샷 2026-04-08 오전 10.26.07 (1).png" alt=""><figcaption></figcaption></figure>


{% endstep %}

{% step %}
#### 에세이 작성 화면이 열리면 상단에 에세이 주제가 표시됩니다. 학생이 종이에 직접 손글씨로 작성한 에세이를 사진으로 촬영한 뒤, **\[손글씨 업로드]** 버튼을 클릭하여 해당 이미지 파일을 업로드해 주세요.

<figure><img src="../../.gitbook/assets/스크린샷 2026-04-08 오전 10.26.32.png" alt=""><figcaption></figcaption></figure>

**<예시 손글씨 파일>**

<figure><img src="../../.gitbook/assets/20251113_130437 (1).jpg" alt=""><figcaption></figcaption></figure>


{% endstep %}

{% step %}
#### 손글씨 이미지를 업로드하면, AI가 자동으로 손글씨를 인식하여 텍스트로 변환합니다. 인식 결과는 직접 수정하실 수 있으며, 내용 확인이 완료되면 우측 상단의 **\[1회차 제출하기]** 버튼을 클릭하여 제출해 주세요

<figure><img src="../../.gitbook/assets/스크린샷 2026-04-08 오전 10.27.19.png" alt=""><figcaption></figcaption></figure>


{% endstep %}

{% step %}
AI 첨삭이 완료되면 **총평**과 **항목별 점수**(어휘 사용, 구성, 문법, 유창성, 내용)를 한눈에 확인하실 수 있습니다. 총평에는 잘한 점과 점수를 향상시키기 위한 구체적인 개선 방향이 함께 제공됩니다. 피드백을 반영하여 다시 작성하고 싶은 경우 **\[다시 영작하기]** 버튼을 클릭하면 재작성이 가능하며, 최종 결과물에 만족하시면 **\[최종 제출하고 나가기]** 버튼을 클릭하여 과제를 완료합니다.

<figure><img src="../../.gitbook/assets/스크린샷 2026-04-08 오전 10.28.15.png" alt=""><figcaption></figcaption></figure>


{% endstep %}

{% step %}
학생이 과제를 최종 제출하면, 과제보기 화면에서 해당 과제가 **초록색(완료)** 상태로 표시됩니다. 선생님은 **\[리포트]** 버튼을 클릭하시면 해당 학생의 에세이 과제 수행 결과를 상세하게 확인하실 수 있습니다. 리포트에는 회차별 총점, 어휘 사용, 구성, 문법, 유창성, 내용 등 항목별 세부 점수와 AI 총평이 포함되어 있어, 학생의 라이팅 실력을 체계적으로 파악하고 지도하실 수 있습니다.
