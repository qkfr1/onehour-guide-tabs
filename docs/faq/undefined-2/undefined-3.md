# 학습지를 따로 파일로 저장하는 것이 가능한가요?

원아워 학습지는 PDF 파일로 저장하는 것이 가능합니다.

학습지 제작 후 \[프린트 하기] 버튼을 클릭하시면 아래와 같이 인쇄 팝업이 나오는데요,

아래 이미지와 같이 \[대상-PDF로 저장] 지정 후 \[저장] 버튼 클릭해 주시면 학습지를 PDF 파일로 저장하는 것이 가능합니다.
