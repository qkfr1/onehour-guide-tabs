# 커리큘럼 생성부터 과제내기까지 한번에 따라하기

## 커리큘럼이란?

커리큘럼은 **"여러 수업에서 만든 과제들을 하나의 학습 과정으로 묶어 관리하는 기능"**&#xC785;니다.&#x20;

반이 새로 개강할 때마다 같은 과제를 다시 설정하는 번거로움 없이, 미리 만들어 둔 커리큘럼을 통해 바로 과제를 부여할 수 있습니다.

***

## 한번에 따라하기

{% stepper %}
{% step %}
### 커리큘럼 만들기

{% embed url="https://app.storylane.io/share/wjham8wtrom4" %}



1. 원아워에 로그인한 후 상단의 **과제 관리** 탭을 클릭합니다.&#x20;
2. 커리큘럼에 포함할 **과제들을 선택**합니다.&#x20;
   1. 특정 월 전체 과제처럼 여러 기간에 걸친 과제를 함께 선택할 수 있습니다.
3. 과제 선택이 끝나면 **커리큘럼으로 저장** 버튼을 클릭합니다.
4. 커리큘럼 이름, 추천 대상, 커리큘럼 설명, 권장 학년을 입력한 뒤 **저장** 버튼을 클릭합니다.
5. 저장 완료 팝업에서 **확인** 버튼을 클릭하면 커리큘럼 상세 페이지로 이동합니다.&#x20;
   1. 1일차, 2일차 등 일자별로 어떤 학습 툴이 사용되는지 확인할 수 있습니다.
{% endstep %}

{% step %}
### 원하는 커리큘럼 찾기 ( 공개 커리큘럼 다운 및 검색하기)

{% embed url="https://app.storylane.io/share/ew36p9opvozk" %}



1. 공유받은 커리큘럼을 다운로드 받아 사용하시려면 공개 커리큘럼을 선택하세요.
2. 커리큘럼이 많아 찾기 어려울 경우, \
   난이도 필터나 검색 입력란에 커리큘럼 이름을 입력하면 커리큘럼이 즉시 필터링되어 표시됩니다.
3. 원하는 커리큘럼을 클릭하여 다운받기 하세요.
4. 다운받은 커리큘럼은 공유받은 커리큘럼에 저장됩니다. 언제든 학습에서 활용하실 수 있습니다.&#x20;
{% endstep %}

{% step %}
### 커리큘럼으로 과제 내기

{% embed url="https://app.storylane.io/share/rvvfxsnqfhge" %}



1. 커리큘럼 상세 페이지에서 **과제 내기** 버튼을 클릭합니다.
2. 달력에서 커리큘럼을 **시작할 날짜를 클릭**합니다. 선택한 날짜부터 커리큘럼 일정이 자동으로 배치됩니다.
3. **요일 설정** 후 **다음** 버튼을 클릭합니다.
4. 과제를 부여할 **반을 선택**한 후 **과제 부여하기** 버튼을 클릭합니다.
5. 과제 부여 완료 후 **캘린더에서 확인하기**를 클릭해 과제가 정상적으로 등록되었는지 확인합니다.

{% hint style="info" %}
**💡 요일 설정으로 수업 일정에 딱 맞게 조정하세요**

주 3회 수업이라면 수업이 없는 요일을 제외하면 자동으로 해당 요일에만 과제가 배치됩니다.&#x20;

한 번만 설정하면 전체 커리큘럼 일정이 자동으로 조정됩니다.
{% endhint %}


{% endstep %}
