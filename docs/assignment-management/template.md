# 과제 복사하기/ 템플릿

> 과제 템플릿 기능을 활용하면 자주 사용하는 과제 구성을 저장하거나,&#x20;
>
> 다른 학생에게 동일한 과제를 빠르게 복사하여 부여할 수 있습니다.

### 과제 복사하기

기존에 부여했던 과제를 복사하여 다른 일정 또는 학생에게 동일한 과제를 다시 부여할 수 있습니다.

#### 복사 방법

1. \[대시보드] > \[부여한 과제 현황] 탭에서 원하는 과제를 체크합니다.
2. \[복사] 버튼을 클릭합니다.
3. 과제 시작일과 요일을 선택합니다.
4. 클래스를 선택한 후, 학생을 선택합니다.
5. \[복사 완료]를 누르면 선택한 학생에게 동일한 과제가 부여됩니다.

복사 후에도 \[템플릿으로 저장] 옵션이 함께 제공되어 향후 활용을 위해 템플릿으로 저장할 수 있습니다.

***

### 템플릿으로 과제 생성하기

#### 템플릿 생성하기

1. \[대시보드] > \[부여한 과제 현황] 탭으로 이동합니다.
2. 복사할 과제의 체크박스를 선택합니다.
3. 화면 상단의 \[템플릿으로 저장] 버튼을 클릭합니다.
4. 템플릿 제목을 입력하고 \[저장]을 누릅니다.

***

#### 템플릿으로 과제내기

1. \[대시보드] > \[부여한 과제 현황] 탭으로 이동합니다.
2. \[과제 템플릿] 버튼을 클릭한 뒤, 원하는 템플릿을 선택합니다.
3. 시작일 및 수행 요일을 설정하고, 필요 시 휴일을 지정합니다.
4. 클래스를 선택한 후, 해당 클래스에서 과제를 부여할 학생을 선택합니다.
5. \[과제 부여하기] 버튼을 클릭하면 부여가 완료됩니다.

***

{% hint style="success" %}
과제 복사 및 템플릿 부여 시, 과제 내용과 구조는 유지되며, 일정과 대상 학생만 변경됩니다.&#x20;
{% endhint %}
