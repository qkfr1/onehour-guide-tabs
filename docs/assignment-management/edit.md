# 과제 수정하기

> 부여한 과제는 \[대시보드] → \[부여한 과제 현황]에서 수정할 수 있습니다.
>
> 과제는 **학생이 수행을 시작하기 전**까지만 일부 항목을 수정할 수 있습니다.

### 수정 방법

1. \[대시보드] → \[부여한 과제 현황]에서 수정할 과제의 제목을 클릭합니다.
2. 오른쪽 하단의 \[수정하기] 버튼을 클릭합니다.
3. 과제 상세 페이지에서 필요한 항목을 수정합니다.
4. 오른쪽 하단의 \[과제수 정] 버튼을 클릭하면 수정이 반영됩니다.

***

### 수정 가능한 항목

* 과제 제목
* 학습 항목 (단어, 문장, 말하기, 문제 등 ON/OFF 및 문항 구성)
* 수행 일정 (시작일, 반복 설정, 쉬는 날 지정 등)

***

### 수정할 수 없는 항목

* 과제 대상(클래스 또는 학생)
* 영작 과제

{% hint style="warning" %}
과제가 여러 명에게 부여된 경우,\
**한 명이라도 과제를 수행했다면 해당 과제는 수정할 수 없습니다.**\
수정이 필요한 경우 과제를 삭제하고 새로 생성해 주세요.
{% endhint %}

{% hint style="info" %}
과제 수정은 한 번에 하나씩만 가능합니다. \
많은 과제의 수정이 필요하시다면 해당 과제를 삭제하고 새로 부여하는 것이 편리할 수 있습니다.&#x20;
{% endhint %}
