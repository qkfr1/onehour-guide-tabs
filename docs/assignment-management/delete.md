# 과제 삭제하기

> 부여한 과제는 \[대시보드] → \[부여한 과제 현황]에서 삭제할 수 있습니다.
>
> 삭제한 과제는 복구할 수 없으며, 리포트에서도 관련 기록이 제거됩니다.

### 삭제 방법

1. \[대시보드] → \[부여한 과제 현황]에서 삭제할 과제의 제목을 클릭합니다.
2. 오른쪽 하단의 \[삭제하기] 버튼을 클릭합니다.
3. 삭제 화면에서 \[확인]을 클릭하면 삭제가 완료됩니다.

***

### 과제 다중 삭제 방법

여러 개의 과제를 한 번에 삭제하려면 다음 절차를 따릅니다.

1. \[대시보드] → \[부여한 과제 현황]탭으로 이동합니다.
2. 삭제할 과제명 앞의 체크박스를 선택합니다.
3. 화면 오른쪽 상단의 \[삭제] 버튼을 클릭합니다.
4. 확인 팝업에서 \[동의합니다] → \[삭제]를 누르면 삭제가 완료됩니다.

***

{% hint style="danger" %}
과제를 삭제하면 학생 계정에서도 과제가 사라지며,\
리포트에서도 해당 내용이 제거됩니다.\
**삭제한 과제는 복구할 수 없습니다.**
{% endhint %}
