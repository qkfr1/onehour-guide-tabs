# 워크시트 만들기

> 워크시트는 원아워 수업 콘텐츠를 기반으로 만든 **오프라인 학습용 인쇄 자료**입니다.
>
> 학생이 원아워를  로그인하여 수행하는 **과제**와는 다르며,
>
> **프린트하거나 PDF로 저장하여 출력해 사용하는 용도**입니다.

***

### 수업 워크시트 생성 방법

1. \[수업] 탭에서 워크시트를 만들고 싶은 수업을 찾는다.
2. 수업 오른쪽의 프린트 아이콘(🖨)을 클릭한다.
3. 워크시트 편집 화면에서 표지, 페이지 옵션, 학습 항목 등을 설정한다.
4. \[인쇄 화면 조정하기]로 구성 확인 후 \[프린트 하기] 를 클릭하여 인쇄 또는 PDF로 저장합니다.

***

### 책 워크시트 생성 방법

1. \[책] 탭에서 워크시트를 만들고 싶은 책을 찾는다.
2. 책을 클릭, 상세 페이지에서 \[🖨️인쇄] 버튼을 클릭합니다.
3. 워크시트 편집 화면에서 표지, 페이지 옵션, 학습 항목 등을 설정한다.
4. \[인쇄 화면 조정하기]로 구성 확인 후 \[프린트 하기]를 클릭하여 인쇄 또는 PDF로 저장합니다.

***

{% tabs %}
{% tab title="표지 생성하기" %}
워크시트 첫 페이지에 출력되는 표지를 설정할 수 있습니다.

* 정보를 입력하고 \[추가] 버튼을 누르면 표지가 생성됩니다.
* **삭제**를 클릭하면 표지를 제거할 수 있습니다.
* 커버에 표시되는 로고는 **마이페이지 > 우리학교/학원 정보**에서 변경할 수 있습니다. (바로가기)
{% endtab %}

{% tab title="페이지 옵션 설정하기" %}
* **답안지**: 추가한 항목들을 바탕으로 정답지를 함께 생성합니다.
* **이름칸**: 모든 페이지 상단에 이름을 작성할 수 있는 칸이 추가됩니다.
* **보기방식**: 화면을 세로로 분할하여 두 칸 또는 여러 칸으로 문제를 배치합니다.
* **글자크기**: 워크시트 내 전체 글씨 크기를 조절할 수 있습니다.
* **페이지 분리**: 문제 유형(단어, 문장, 문제 등)별로 자동 분할되어 출력됩니다.

{% hint style="warning" %}
문항 셔플을 적용한 경우, **답안지는 생성되지 않습니다.**
{% endhint %}
{% endtab %}

{% tab title="단어 / 문장 / 문제" %}
* 학습할 단어, 문장, 문제 유형을 선택한 뒤 \[**화면에 추가하기]**&#xB97C; 클릭해야 인쇄 목록에 반영됩니다.
* 문제는 해당 수업에 문제 세트가 등록된 경우에만 출제할 수 있습니다.
{% endtab %}
{% endtabs %}
