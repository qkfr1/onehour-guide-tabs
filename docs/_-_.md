# 구글시트\_자동연동\_설정안내

차트가 구글시트를 직접 읽어서 그리도록 연결하는 방법입니다.\
한 번만 설정하면, 이후로는 **시트만 고쳐도 차트가 자동 갱신**됩니다.

## 1단계 — 마스터를 구글시트로 올리기

{% stepper %}
{% step %}
## 마스터를 구글시트로 올리기

1. 구글 드라이브에서 새 구글시트 생성
2. `파일 → 가져오기 → 업로드`로 `원아워_콘텐츠_마스터.xlsx` 업로드
3. "스프레드시트 바꾸기" 또는 "새 시트 삽입" 선택
4. `콘텐츠_마스터` 시트가 들어왔는지 확인 (컬럼: ID·브랜드·콘텐츠명·스킬·시작학년·끝학년·CEFR시작·CEFR끝·추정확신도·추정근거·노출·마켓URL·비고)
{% endstep %}

{% step %}
## 시트를 "웹에 게시"하기

1. 구글시트 상단 메뉴 `파일 → 공유 → 웹에 게시`
2. 게시 대화상자에서:
   * **링크** 탭 선택
   * 게시할 시트: **`콘텐츠_마스터`** 선택 (전체 문서 아님)
   * 형식: **`쉼표로 구분된 값(.csv)`** 선택
3.  **게시** 버튼 클릭 → 나오는 CSV 주소를 복사

    (주소 형태: `https://docs.google.com/spreadsheets/d/e/...../pub?gid=0&single=true&output=csv`)
{% endstep %}

{% step %}
## 차트에 주소 연결

1. `원아워_콘텐츠_레벨차트_자동연동.html` 파일을 텍스트 편집기로 열기
2.  상단의 이 줄을 찾기:

    ```javascript
    const SHEET_CSV_URL = "PASTE_YOUR_PUBLISHED_CSV_URL_HERE";
    ```
3.  따옴표 안 내용을 2단계에서 복사한 CSV 주소로 교체:

    ```javascript
    const SHEET_CSV_URL = "https://docs.google.com/spreadsheets/d/e/...../pub?gid=0&single=true&output=csv";
    ```
4. 저장

이제 이 HTML을 브라우저로 열면 시트 데이터가 자동으로 표시됩니다.
{% endstep %}
{% endstepper %}

***

## 운영 방법

* **콘텐츠 추가**: 시트 맨 아래 새 행 추가 → 차트 새로고침하면 반영
* **레벨 수정**: 시작학년·끝학년·CEFR 셀 수정 → 즉시 반영
* **숨기기**: `노출` 컬럼을 `N`으로 → 차트에서 사라짐
* **클릭 링크**: `마켓URL` 컬럼에 주소 입력 → 막대 클릭 시 이동(↗ 표시)

{% hint style="info" %}
게시된 시트는 변경 후 반영까지 보통 몇 분 정도 걸릴 수 있습니다(구글 캐시).
{% endhint %}

## 깃북 페이지에 embed 또는 iframe 추가하기

1.  깃북 가이드 페이지 편집에서:

    ```html

    ```

## 주의 — 학년/CEFR 순서

차트 내부에 학년 순서(5세~~성인)와 CEFR 순서(Pre-A1~~C1)가 고정되어 있습니다.\
시트의 `시작학년`·`CEFR시작` 등에는 **이 목록과 똑같은 표기**를 써야 합니다(드롭다운 사용 권장):

* 학년: 5세, 6세, 7세, 초1~~초6, 예비중, 중1~~중3, 고1\~고3, 수능, 성인
* CEFR: Pre-A1, A1, A2, B1, B2, C1
