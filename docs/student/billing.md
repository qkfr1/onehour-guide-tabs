# 학생 결제

PC 환경에서 원아워 구독권을 결제하는 방법을 안내드립니다.\
※ 결제는 **모바일 및 태블릿 환경에서는 지원되지 않습니다.&#x20;**<mark style="background-color:yellow;">**반드시 PC에서 진행해주세요.**</mark>



### 1. 로그인 후 결제 페이지로 이동하기

* 선생님께 전달받은 **아이디와 비밀번호로 로그인**합니다.
* 로그인 후 자동으로 나타나는\
  \[구독권 구매/카드 등록하기] 팝업의 버튼을 클릭하거나,\
  화면 상단의 \[구독 / 결제] 버튼을 눌러 결제 페이지로 이동합니다.

### 2. 카드 등록하기

1. \[카드 등록] 버튼을 클릭합니다.
2. **성인(학부모님 또는 본인)의 카드 정보를 입력**합니다.
   * 카드번호
   * 유효기간
   * 생년월일 또는 사업자등록번호 등
3. 약관에 동의한 뒤 \[등록하기] 버튼을 클릭합니다.

{% hint style="warning" %}
**\[등록하기] 버튼이 비활성화되어 있다면?**\
입력해야 할 정보가 누락된 상태입니다.\
모든 입력란이 정확히 채워졌는지 확인해주세요.
{% endhint %}

{% hint style="warning" %}
**‘입력한 정보를 확인해주세요’ 오류가 뜬다면?**\
카드번호나 유효기간 등 입력된 값이 올바르지 않거나,\
카드 상태가 온라인 자동 결제가 불가능한 경우입니다.\
해당 카드사에 자동결제 가능 여부를 확인해주세요.
{% endhint %}

### 3. 구독권 구매하기

* 카드 등록을 완료하면 \[구독권 구매] 버튼이 나타납니다.
* 해당 버튼을 눌러 구독권을 결제하면\
  **이번 달 구독권 구매와 정기결제 등록이 동시에 완료**됩니다.

### 4. 결제 이후

* 정기결제는 **매월 1일 자동으로 진행**됩니다.\
  다음 달부터는 별도의 결제 없이 계속 이용하실 수 있습니다.

{% hint style="success" %}
당월 1일이 아닌 날짜에 결제하신 경우\
구독권 금액은 **일할 계산되어 자동 반영**됩니다.

결제 페이지에서 **\[카드 변경], \[구독 해지]**&#xB3C4; 직접 진행하실 수 있습니다.
{% endhint %}
