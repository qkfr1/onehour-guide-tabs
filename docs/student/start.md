# 시작하기

### 1. 기기별 기본 준비

#### 💻 PC

크롬 브라우저에서 ‘원아워’를 검색해 사이트에 접속해주세요.

{% hint style="info" %}
**크롬 이외의 다른 브라우저에서는 동작하지 않습니다.**

크롬 브라우저가 설치되어 있지 않다면 크롬 다운로드 후 이용해주세요.
{% endhint %}

#### 📱 태블릿 또는 스마트폰

플레이스토어 또는 앱스토어에서 \[원아워]를 검색해 앱을 설치해주세요.

***

### 2. 로그인

선생님께 전달받은 **아이디와 비밀번호**를 입력해 로그인해주세요.

{% hint style="warning" %}
**아이디 또는 비밀번호가 잘못 입력되었습니다** 라는 문구가 나올 경우

학생은 ID/비밀번호를 직접 찾을 수 없습니다.

→ **반드시 선생님께 문의해주세요.**
{% endhint %}

***

### 3. 과제하기

#### 📝 일반 과제

1. 과제가 부여된 날짜를 선택하고
2. 과제명을 눌러 \[과제 시작] 버튼을 눌러주세요.

* <mark style="color:$success;">**초록색 날짜**</mark>: 모든 과제 완료
* <mark style="color:yellow;">**주황색 날짜**</mark>: 완료되지 않은 과제가 있음
* <mark style="color:red;">**빨간생 날짜**</mark>: 과제를 시작하지 않음

오답이 있는 과제는 \[오답 풀기]를 눌러 모두 완료해주세요.

***

#### 🎙 더빙 / 프리토킹 과제

과제를 하기 위해서는 **기기의 마이크와 카메라 사용 권한이 허용**되어야 합니다.

{% hint style="success" %}
**권한 필요 오류가 뜬다면?**

아래 안내에 따라 마이크/카메라 권한을 허용해주세요.
{% endhint %}

**📌 PC**

오류 팝업에 표시되는 **영상 안내**를 따라 마이크/카메라 권한을 허용해주세요.![](<../.gitbook/assets/image (17).png>)

**📌 안드로이드**

아래 이미지처럼 **\[설정] > 원아워 앱 > 카메라 및 마이크 권한 허용**을 해주세요.

![](<../.gitbook/assets/image (18).png>)![](<../.gitbook/assets/image (24).png>)\


**📌 iOS**&#x20;

아래 이미지처럼 **\[설정] > 원아워 앱 > 카메라 및 마이크 권한 허용**을 해주세요.

![](<../.gitbook/assets/image (23).png>)
