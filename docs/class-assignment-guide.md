# class assignment guide

## 1. 자주 겪는 문제

> "학생 화면에서는 과제가 있는데, 과제관리에서 클래스를 선택하면 안 보여요"

이 문제는 기본반 상태에서 학생에게 직접 부여한 과제(학생개별과제)를 클래스 과제 관리 화면에서 조회하고 있기 때문입니다.\
학생개별과제는 클래스에 귀속되지 않아 클래스 선택 시에는 조회되지 않습니다.

_이미지: 학생 화면(과제 있음) vs 선생님 과제관리 클래스 선택 시(과제 안 보임) 비교_

***

## 2. 클래스 과제 vs 학생개별과제

과제를 부여할 때 클래스 선택 여부에 따라 과제 유형이 달라집니다.

<table><thead><tr><th width="161.9140625"></th><th width="274.28125">클래스 과제</th><th>학생개별과제</th></tr></thead><tbody><tr><td>생성 방법</td><td>클래스 선택 → 학생 선택 → 과제 부여</td><td>기본반 상태 → 학생 선택 → 과제 부여</td></tr><tr><td>귀속</td><td>해당 클래스에 귀속</td><td>클래스와 무관, 학생에게만 귀속</td></tr></tbody></table>

_이미지: 과제 부여 화면에서 클래스 선택 드롭다운 - 클래스 선택 상태 vs 기본반 상태 비교_

***

## 3. 과제관리 조회 기준

과제관리에서 어떤 과제가 보이는지는 선택 항목에 따라 달라집니다.

<table><thead><tr><th width="176.03125">클래스 선택</th><th width="181.0390625">학생 선택</th><th>보여지는 과제</th></tr></thead><tbody><tr><td>기본</td><td>전체학생</td><td>전체 과제</td></tr><tr><td>기본</td><td>학생 선택</td><td>해당 학생의 전체 과제</td></tr><tr><td>특정 클래스 선택</td><td>전체학생</td><td>해당 클래스 과제 전체</td></tr><tr><td>특정 클래스 선택</td><td>학생 선택</td><td>해당 클래스 과제 중 선택한 학생 과제</td></tr></tbody></table>

**핵심**

* 클래스 과제 → 클래스 선택 시, 기본 선택 시 모두 조회 가능
* 학생개별과제 → 기본 선택 시에만 조회 가능

_이미지: 과제관리 화면 - 1단계(기본/클래스) + 2단계(전체학생/학생개별) 선택 UI 캡쳐_

***

## 4. 올바른 과제 부여 방법

클래스별로 과제를 관리하신다면, 과제 부여 시 클래스를 먼저 선택해 주세요.

1. 과제 부여 화면에서 클래스 선택
2. 학생 선택 (전체 또는 일부)
3. 과제 부여

이렇게 하시면 과제관리에서 클래스 선택 시 모든 과제가 정상적으로 표시됩니다.

_이미지: 올바른 플로우 시연 - 클래스 선택 → 학생 선택 → 과제 부여 단계별 캡쳐 또는 GIF_

***

## 5. 학생개별과제 확인하기

이미 기본반에서 부여한 학생개별과제는 과제관리 → 기본반 + 학생 선택 시 확인할 수 있습니다.&#x20;

_이미지: 과제관리에서 기본 선택 → 과제 목록 보이는 화면_
