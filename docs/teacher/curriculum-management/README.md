# 커리큘럼 관리

## 커리큘럼이란?

커리큘럼은 여러 수업에서 출제된 과제를 하나의 학습 과정으로 묶어 관리하는 기능입니다. \
한 번 만들어두면 반복적으로 과제를 부여할 수 있어, 매번 과제를 일일이 출제하는 번거로움을 줄여줍니다.

### 커리큘럼 활용 예시

* 4주 완성 문법 코스
* 중간고사 대비 집중 과정
* 레벨별 리딩 프로그램
* 방학 특강 커리큘럼

### 커리큘럼 목록 화면

커리큘럼 목록은 두 개의 탭으로 구분됩니다.

| 탭          | 설명                   |
| ---------- | -------------------- |
| 내가 만든 커리큘럼 | 직접 생성한 커리큘럼 목록       |
| 공유 받은 커리큘럼 | 다른 선생님이 공유해준 커리큘럼 목록 |
