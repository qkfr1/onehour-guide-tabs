# 커리큘럼 링크로 공유하기

커리큘럼 외부 링크 공유 기능을 통해 선생님들이 직접 만든 커리큘럼을 다른 선생님들에게 **URL 링크만으로** 쉽고 빠르게 공유할 수 있습니다.

별도의 ID 지정 없이 링크 하나로 커리큘럼을 전달하고, 받는 분도 간편하게 내 커리큘럼에 추가할 수 있습니다.

## 상황별 사용법

### 커리큘럼 공유하기 (원작자)

내가 만든 커리큘럼을 다른 선생님들에게 공유하는 방법입니다.

#### 공유 링크 생성하기

{% stepper %}
{% step %}
### 커리큘럼 선택

**'내가 만든 커리큘럼'** 목록에서 공유할 커리큘럼을 선택합니다.
{% endstep %}

{% step %}
### 공유하기 클릭

커리큘럼 상세 페이지에서 **'공유하기'** 버튼을 클릭합니다.
{% endstep %}

{% step %}
### 링크 공유 활성화

공유 화면에서 **'링크 공유'** 토글을 클릭합니다.
{% endstep %}

{% step %}
### 링크 복사 및 전달

생성된 링크를 복사하여 카카오톡, 이메일 등으로 전달합니다.
{% endstep %}
{% endstepper %}

#### 공유 링크 관리하기

* **링크 비활성화:** 토글을 끄면 더 이상 해당 링크로 접근할 수 없습니다. 다시 활성화해도 링크 주소는 동일합니다.
* **공유 받은 사용자 확인:** 공유 모달에서 링크를 통해 커리큘럼을 추가한 사용자 목록을 확인할 수 있습니다.
* **공유 해제:** 목록에서 특정 사용자를 삭제하면 해당 사용자의 커리큘럼에서 공유 받은 커리큘럼이 제거됩니다.

### 공유 받은 커리큘럼 추가하기 (수신자)

#### 로그인 상태에서 링크 클릭 시

{% stepper %}
{% step %}
### 미리보기 이동

공유 링크를 클릭하면 커리큘럼 미리보기 페이지로 이동합니다.
{% endstep %}

{% step %}

{% step %}
### 추가 완료 및 확인

추가 완료 팝업이 나타나며, \*\*'커리큘럼 확인하기'\*\*를 클릭하면 추가된 커리큘럼으로 이동합니다.
{% endstep %}

{% step %}
### 과제 내기 가능

추가된 커리큘럼에서 바로 학생에게 과제를 낼 수 있습니다.
{% endstep %}
{% endstepper %}

#### 비로그인 상태에서 링크 클릭 시

{% stepper %}
{% step %}
### 미리보기 확인

공유 링크를 클릭하면 커리큘럼 미리보기 페이지를 확인할 수 있습니다.
{% endstep %}

{% step %}
### 로그인/회원가입 안내

**'내 커리큘럼에 추가'** 버튼 클릭 시 로그인/회원가입 페이지로 이동합니다.
{% endstep %}

{% step %}
### 자동 복귀

로그인 또는 회원가입 완료 후 자동으로 공유 페이지로 돌아옵니다.
{% endstep %}

{% step %}
### 추가 완료

추가 완료 팝업과 함께 커리큘럼이 자동으로 추가됩니다.
{% endstep %}
{% endstepper %}

{% hint style="info" %}
**참고:** 신규 회원가입 시 추천인 란에 공유자의 ID가 자동으로 입력됩니다.
{% endhint %}

## 자주 묻는 질문 (FAQ)

<details>

<summary><strong>Q1. 공유 받은 커리큘럼을 다른 사람에게 다시 공유할 수 있나요?</strong></summary>

아니요, 공유 받은 커리큘럼은 재공유가 불가능합니다. 공유하기 버튼이 표시되지 않습니다. 원작자만 해당 커리큘럼을 공유할 수 있습니다.

</details>

<details>

<summary><strong>Q2. 공유 링크를 비활성화하면 어떻게 되나요?</strong></summary>

링크를 비활성화하면 더 이상 해당 링크로 접근할 수 없습니다. 링크를 클릭한 사용자에게는 "링크가 비활성화되어 공유 받을 수 없습니다"라는 안내가 표시됩니다. 다시 활성화하면 동일한 링크 주소로 다시 공유할 수 있습니다.

</details>

<details>

<summary><strong>Q3. 이미 공유 받은 사용자를 목록에서 삭제하면 어떻게 되나요?</strong></summary>

해당 사용자의 '공유 받은 커리큘럼' 목록에서 커리큘럼이 자동으로 삭제됩니다. 사용자가 커리큘럼을 열람 중이었다면, 다음 액션(새로고침, 과제내기 등) 시 "공유가 종료된 커리큘럼입니다"라는 안내와 함께 목록 화면으로 이동됩니다.

</details>

<details>

<summary><strong>Q4. 삭제된 후 다시 같은 링크로 접근하면 어떻게 되나요?</strong></summary>

링크가 여전히 활성화 상태라면, 새로운 사용자처럼 다시 커리큘럼을 추가할 수 있습니다. 추가하면 공유자의 목록에도 다시 표시됩니다.

</details>

<details>

<summary><strong>Q5. 학생 계정으로 공유 링크에 접속하면 어떻게 되나요?</strong></summary>

학생 계정은 커리큘럼 추가 권한이 없으므로, 링크 클릭 시 별도 안내 없이 학생의 기본 화면(과제 보기 페이지)으로 자동 이동됩니다.

</details>

<details>

<summary><strong>Q6. 무료체험이 만료된 상태에서도 공유 받을 수 있나요?</strong></summary>

네, 선생님 계정이라면 구독 여부와 관계없이 커리큘럼을 공유 받을 수 있습니다. 다만, 과제를 내려면 구독이 필요하며, 과제내기 시 구독 유도 팝업이 표시됩니다.

</details>

<details>

<summary><strong>Q7. 내가 만든 공유 링크를 직접 클릭하면 어떻게 되나요?</strong></summary>

본인이 만든 링크를 클릭하면 '내가 만든 커리큘럼'의 해당 커리큘럼 상세 페이지로 이동합니다. 이를 통해 링크가 정상 작동하는지 확인할 수 있습니다.

</details>

<details>

<summary><strong>Q8. 공유 받은 커리큘럼을 삭제하고 싶어요.</strong></summary>

'공유 받은 커리큘럼' 목록에서 해당 커리큘럼의 삭제 버튼을 클릭하면 내 목록에서 제거할 수 있습니다.

</details>

<details>

<summary><strong>Q9. 공유 링크에 유효기간이 있나요?</strong></summary>

별도의 유효기간은 없습니다. 원작자가 링크를 비활성화하거나 커리큘럼을 삭제하지 않는 한 계속 사용할 수 있습니다.

</details>
