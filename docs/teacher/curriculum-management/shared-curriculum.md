# 공유받은 커리큘럼 사용하기

다른 선생님이 공유해준 커리큘럼을 활용하는 방법입니다.

### 공유받은 커리큘럼 확인

* 커리큘럼 목록의 \[공유 받은 커리큘럼] 탭에서 확인할 수 있습니다.
* 새로 공유받은 커리큘럼은 목록 가장 하단에 추가됩니다.

### 사용 가능한 기능

### 공유가 취소된 경우

원작자가 공유를 취소하면 더 이상 커리큘럼을 사용할 수 없습니다.

* 목록에서도 커리큘럼이 보여지지 않습니다.

> 참고: 공유가 취소되어도 이미 부여한 과제는 그대로 유지됩니다. \
> 다만, 해당 커리큘럼으로 새로운 과제를 부여할 수는 없습니다.
