# 공개 커리큘럼

### 공개 커리큘럼이란?

원아워의 모든 선생님이 직접 만든 커리큘럼을 갤러리에 공개하고,\
다른 선생님의 커리큘럼을 검색해 내 보관함으로 가져올 수 있는 기능입니다.

{% hint style="info" %}
공유 커리큘럼과 공개 커리큘럼의 차이는?&#x20;

공유 커리큘럼은 원작자 선생님께서 선생님을 지정하여 공유하는 형태이고

공개 커리큘럼은 전체 공개한 상태의 커리큘럼은 사용하실 선생님께서 자유롭게 다운받아 활용하는 형태에요.
{% endhint %}

***

### 공개 커리큘럼 갤러리 이용하기

**진입 경로:** 커리큘럼 > 사이드바 > 공개 커리큘럼

#### 검색 및 필터

* **검색:** 커리큘럼 제목, 설명, 태그로 검색 가능
* **정렬:** 인기순(기본) / 최신순
* **학년 필터:** 유치원 / 초등 / 중등 / 고등 / 성인 (복수 선택 가능)

#### 커리큘럼 가져오기

**방법 1  공유 받기** 보관함에 저장만 하고 싶을 때 사용해요.

1. 커리큘럼 > 사이드바 > 공개 커리큘럼 메뉴로 진입한 뒤 원하는 커리큘럼 찾아 클릭
2. 커리큘럼 상세 페이지 우측 상단 `[공유 받기]` 클릭
3. 확인 팝업에서 `[확인]` 클릭
4. 내 보관함(공유 받은 커리큘럼)에 자동 저장

**방법 2  바로 과제 내기** 저장과 동시에 바로 과제를 낼 수 있어요.

1. 커리큘럼 > 사이드바 > 공개 커리큘럼 메뉴로 진입한 뒤 원하는 커리큘럼 찾아 클릭
2. 커리큘럼 상세 페이지 우측 상단 `[과제 내기]` 클릭
3. 안내 팝업 확인 후 자동으로 보관함 저장 + 과제 설정 화면으로 이동

<figure><img src="../../.gitbook/assets/스크린샷 2026-03-06 오전 11.17.08 (1).png" alt=""><figcaption></figcaption></figure>

{% hint style="info" %}
이미 가져온 커리큘럼은 썸네일에 **받음** 뱃지가 표시되고, 바로 과제 내기가 가능합니다.
{% endhint %}

***

### 내 커리큘럼 공개하기

**방법 :** 내가 만든 커리큘럼 > 상세 페이지 > 공유하기 > "공개 커리큘럼 등록" 스위치 ON

{% hint style="info" %}
* 공개 시 모든 선생님이 검색하고 공유받을 수 있어요.
* 원본 수정 권한은 원작자(나)만 가집니다.
* 저작권을 위반하는 콘텐츠가 포함된 경우 사전 안내 없이 공개가 취소될 수 있습니다.
{% endhint %}

<figure><img src="../../.gitbook/assets/스크린샷 2026-03-06 오전 11.03.41.png" alt=""><figcaption></figcaption></figure>

#### 비공개로 전환하기

**방법 :** 내가 만든 커리큘럼 > 상세 페이지 > 공유하기 > 스위치 OFF > `[비공개로 전환]` 클릭

{% hint style="info" %}
* 이후에는 갤러리 검색에 노출되지 않습니다.
* 이미 공유받은 선생님은 계속 사용 가능합니다.
{% endhint %}

***

### 주의사항

* 공개 중이거나, 다른 선생님이 공유받은 커리큘럼은 **삭제가 불가능**합니다. \
  삭제하려면 먼저 공개를 해제하거나, 공유를 취소해 주세요.
