# 클래스 책장 관리

클래스에 책장을 추가하면 학생들이 **책 탭**에서 책장에 포함된 책을 한눈에 확인할 수 있습니다.

> 개별 책 추가·삭제는 **클래스 책 관리** 탭에서 진행해 주세요.

## 책장 추가하기

{% stepper %}
{% step %}
상단 메뉴에서 **클래스**를 클릭합니다.
{% endstep %}

{% step %}
왼쪽 패널에서 책장을 추가할 **클래스**를 선택합니다.
{% endstep %}

{% step %}
**클래스 책장 관리** 탭을 클릭합니다.
{% endstep %}

{% step %}
우측 상단의 **책장 추가** 버튼을 클릭합니다.
{% endstep %}

{% step %}
추가할 책장을 선택하고 저장합니다.
{% endstep %}
{% endstepper %}

{% hint style="info" %}
책장 추가 시 체크박스로 여러 책장을 한 번에 선택할 수 있습니다.
{% endhint %}

***

## 책장 제거하기

{% stepper %}
{% step %}
**클래스 책장 관리** 탭에서 제거할 책장의 체크박스를 선택합니다.
{% endstep %}

{% step %}
우측 상단의 **선택 제거** 버튼을 클릭합니다.
{% endstep %}

{% step %}
확인 메시지에서 **제거**를 클릭합니다.
{% endstep %}
{% endstepper %}

{% hint style="info" %}
책장을 클래스에서 제거해도 책장 자체는 삭제되지 않습니다. 해당 클래스에서만 보이지 않게 됩니다.
{% endhint %}
