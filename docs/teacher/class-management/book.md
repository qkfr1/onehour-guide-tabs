# 클래스 책 관리

클래스에 개별 책을 추가하고 관리하는 페이지입니다.

> 책장에 포함된 도서는 **클래스 책장 관리** 탭에서 수정해 주세요.\
> 학생들은 '책' 탭에서 개별 책과 책장에 포함된 책을 모두 확인할 수 있습니다.

## 책 추가하기

{% stepper %}
{% step %}
상단 메뉴에서 **클래스**를 클릭합니다.
{% endstep %}

{% step %}
왼쪽 패널에서 책을 추가할 **클래스**를 선택합니다.
{% endstep %}

{% step %}
**클래스 책 관리** 탭을 클릭합니다.
{% endstep %}

{% step %}
우측 상단의 **책 추가** 버튼을 클릭합니다.
{% endstep %}

{% step %}
추가할 책을 선택하고 저장합니다.
{% endstep %}
{% endstepper %}

{% hint style="info" %}
책 추가 시 체크박스로 여러 권을 한 번에 선택할 수 있습니다.
{% endhint %}

***

## 책 제거하기

{% stepper %}
{% step %}
**클래스 책 관리** 탭에서 제거할 책의 체크박스를 선택합니다.
{% endstep %}

{% step %}
우측 상단의 **선택 제거** 버튼을 클릭합니다.
{% endstep %}

{% step %}
확인 메시지에서 **제거**를 클릭합니다.
{% endstep %}
{% endstepper %}

{% hint style="info" %}
책을 클래스에서 제거해도 책 자체는 삭제되지 않습니다. 해당 클래스에서만 보이지 않게 됩니다.
{% endhint %}
