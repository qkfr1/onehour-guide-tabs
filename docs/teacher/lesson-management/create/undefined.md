# 기출 쌍둥이 변형 문제

기출문제 PDF에서 문제 유형을 추출하고, \
선생님이 입력한 새 지문과 결합하여 동일 유형의 변형문제 세트를 자동으로 생성하는 기능입니다.

## 기능 설명

기출문제 PDF를 업로드하면 AI가 문제 유형(빈칸, 순서, 어법 등)을 자동으로 분석합니다. \
여기에 선생님이 교과서 본문을 입력하면, 추출된 유형 그대로 새 지문에 맞는 변형문제가 만들어집니다.

* 유형 최대 50개 × 지문 최대 10개 = 최대 500개 문제 생성
* 객관식만 지원
* 생성된 문제에는 정답과 해설이 자동으로 포함됩니다

## 추천 대상

* 학교/학원별 기출 스타일에 맞춘 문제가 필요한 경우
* 내신 대비 변형 문제를 직접 만들어야 하는 선생님

### 지문 입력

교과서 본문을 복사하여 붙여넣습니다.

* 지문당 300자 이상, 5,000자 이하
* 최대 10개 지문까지 입력 가능
* **+ 지문 추가** 버튼으로 여러 지문을 등록할 수 있습니다

### 옵션 설정

지문 변형 여부와 변형 수준을 선택합니다.

| 옵션     | 설명                   |
| ------ | -------------------- |
| 지문 그대로 | 입력한 지문을 변형 없이 그대로 사용 |
| 지문 변형  | 입력한 지문도 패러프레이징하여 변형  |

지문 변형을 선택하면 5단계 중 원하는 수준을 고를 수 있습니다.

| 변형 범위          | 설명                         |
| -------------- | -------------------------- |
| 부분 변형 · 단어     | 약 50% 문장에서 단어만 교체          |
| 부분 변형 · 단어+    | 약 80% 문장에서 상위 어휘로 교체       |
| 부분 변형 · 문장 구조  | 약 80% 문장에서 단어와 구조 함께 변형    |
| 전체 변형 · 문장 구조+ | 전체를 논문/토플 수준으로 패러프레이징      |
| 전체 변형 · 최대     | 논문/토플 수준 + 부연 설명 추가로 길이 확장 |

## 주의사항

#### 업로드 제한

* **PDF 파일**: 최대 30MB, 30장 이내
* **지문 개수**: 최대 10개까지 추가 가능
* **지문 글자 수**: 지문당 300자 이상, 5,000자 이하

#### 미지원 유형

다음 유형은 쌍둥이 문제로 생성되지 않습니다.

* 듣기 문제
* 주관식
* 서술형
* 도표
