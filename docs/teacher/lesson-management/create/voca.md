# 단어장

단어 목록만 있으면 AI가 뜻과 예문을 자동으로 채우고, 여러 개의 수업으로 나누어 단어장을 만들어 드립니다.&#x20;

복사·붙여넣기 또는 엑셀 업로드로 몇 분 안에 단어 수업을 완성할 수 있습니다.

## 단어 입력

단어를 입력하는 방법은 두 가지입니다. **둘 중 한 가지 방식으로** 사용할 수 있습니다.

### 텍스트 붙여넣기

<figure><img src="../../../.gitbook/assets/스크린샷 2026-06-11 오후 2.08.12.png" alt=""><figcaption></figcaption></figure>

입력창에 단어 목록을 그대로 붙여넣습니다.

* 단어만 입력 (예: `apple`)
* 단어와 뜻을 함께 입력

### 엑셀 파일 업로드

<figure><img src="../../../.gitbook/assets/스크린샷 2026-06-11 오후 2.10.53.png" alt="" width="563"><figcaption></figcaption></figure>

* 영어 단어만 인식하며, 한글·숫자·기호가 섞인 항목은 자동으로 제외됩니다.
* 첫 번째 행의 안내 문구는 자동으로 건너뜁니다.

{% hint style="warning" %}
엑셀 파일은 정해진 템플릿 형식만 지원합니다.&#x20;

엑셀 탭 오른쪽의 **예시 템플릿 다운로드**를 눌러 양식을 받은 뒤, 단어를 입력해 업로드해 주세요.
{% endhint %}

## 단어장 이름&#x20;

<figure><img src="../../../.gitbook/assets/스크린샷 2026-06-11 오후 2.14.49.png" alt="" width="563"><figcaption></figcaption></figure>

단어장 이름을 입력합니다. 이 이름을 기준으로 각 수업의 이름이 만들어집니다.

* `TOEIC 필수 단어`를 입력하면 → `TOEIC 필수 단어 1과` · `TOEIC 필수 단어 2과` ... 형태로 만들어집니다.
* 이름을 비워 두면 `단어장 1과` · `단어장 2과` 형태로 자동 생성됩니다.

## 정렬과 분할

정렬 방식과 수업당 단어 수를 정한 뒤 단어장을 생성합니다.

<figure><img src="../../../.gitbook/assets/스크린샷 2026-06-11 오후 2.16.32.png" alt=""><figcaption></figcaption></figure>

### 단어 정렬 방식

* 입력 순서 그대로 (기본) : 입력한 순서를 그대로 유지합니다.
* ABC 순서 : 알파벳 순으로 정렬합니다.

### 수업당 단어 수

슬라이더 또는 숫자 입력으로 한 수업에 담을 단어 수를 정합니다.

* 최소 10개 \~ 최대 500개
* 추천: 10\~30개

수업당 단어 수를 조정하면 **총 수업 수와 각 수업의 단어 수가 실시간으로 바뀝니다.**&#x20;
{% endstep %}

## 생성이 시작되면

**단어장 생성을 시작했어요** 라는 안내가 표시됩니다. 단어장은 여러 개의 수업이 담긴 **책 형태**로 만들어집니다.

* 생성은 백그라운드에서 진행되며, 잠시 후 수업 목록에서 확인할 수 있습니다.
* 안내창의 **수업 보러가기**를 누르면 만들어진 책으로 이동합니다.

{% hint style="info" %}
단어 수가 많아 수업이 여러 개일 경우, 모두 만들어지기까지 시간이 조금 걸릴 수 있습니다.
{% endhint %}
