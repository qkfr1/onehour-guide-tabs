# 단어장 가이드

단어 목록만 있으면 AI가 뜻과 예문을 자동으로 채우고, 여러 개의 수업으로 나누어 단어장을 만들어 드립니다.&#x20;

복사·붙여넣기 또는 엑셀 업로드로 몇 분 안에 단어 수업을 완성할 수 있습니다.

## Step 1 · 단어 입력

단어를 입력하는 방법은 두 가지입니다. **둘 중 한 가지 방식으로** 사용할 수 있습니다.

### 텍스트 붙여넣기

입력창에 단어 목록을 그대로 붙여넣습니다. 형식은 자유롭게 입력하셔도 됩니다.

* 단어만 입력 (예: `apple`)
* 단어와 뜻을 함께 입력

쉼표, 탭, 공백 등 어떤 구분자로 입력해도 단어와 뜻을 자동으로 구분합니다.

### 엑셀 파일 업로드

{% hint style="warning" %}
엑셀 파일은 정해진 템플릿 형식만 지원합니다.&#x20;

엑셀 탭 오른쪽의 **예시 템플릿 다운로드**를 눌러 양식을 받은 뒤, 단어를 입력해 업로드해 주세요.
{% endhint %}

* 영어 단어만 인식하며, 한글·숫자·기호가 섞인 항목은 자동으로 제외됩니다.
* 첫 번째 행의 안내 문구는 자동으로 건너뜁니다.

{% hint style="info" %}
입력 방식을 바꾸면 먼저 입력한 내용은 삭제됩니다. 텍스트에서 엑셀로 전환하면 붙여넣은 텍스트가, 엑셀에서 텍스트로 전환하면 첨부한 파일이 삭제된다는 안내가 표시됩니다.
{% endhint %}

### AI 자동 생성 옵션

이미 입력된 뜻이나 예문은 그대로 사용하며, **비어 있는 항목만** AI가 채웁니다.
{% endstep %}

{% step %}
## Step 2 · 단어장 이름

단어장 이름을 입력합니다. 이 이름을 기준으로 각 수업의 이름이 만들어집니다.

* `TOEIC 필수 단어`를 입력하면 → `TOEIC 필수 단어 1과` · `TOEIC 필수 단어 2과` ... 형태로 만들어집니다.
* 이름을 비워 두면 `단어장 1과` · `단어장 2과` 형태로 자동 생성됩니다.

{% hint style="info" %}
단어장 이름은 선택 항목입니다. 비워 두어도 다음 단계로 넘어갈 수 있으며, 만든 뒤에도 학원 관리 페이지에서 언제든 수정할 수 있습니다.
{% endhint %}
{% endstep %}

{% step %}
## Step 3 · 정렬과 분할

정렬 방식과 수업당 단어 수를 정한 뒤 단어장을 생성합니다.

### 단어 정렬 방식

| 방식             | 설명                 |
| -------------- | ------------------ |
| 입력 순서 그대로 (기본) | 입력한 순서를 그대로 유지합니다. |
| ABC 순서         | 알파벳 순으로 정렬합니다.     |

### 수업당 단어 수

슬라이더 또는 숫자 입력으로 한 수업에 담을 단어 수를 정합니다.

* 최소 10개 \~ 최대 500개
* 추천: 10\~30개

수업당 단어 수를 조정하면 **총 수업 수와 각 수업의 단어 수가 실시간으로 바뀝니다.** 아래 미리보기에서 어떻게 나뉘는지 바로 확인할 수 있습니다.

{% hint style="info" %}
한 수업에는 최대 500개까지 담을 수 있습니다. 단어가 더 많으면 자동으로 여러 수업으로 나뉘며, 마지막 수업은 남은 단어 수만큼 배정됩니다.
{% endhint %}

### 최종 확인 후 생성

화면 아래 요약 카드에서 단어장 이름, 총 단어 수, 정렬 방식, 수업당 단어 수, 총 수업 수, AI 옵션을 한 번에 확인할 수 있습니다. 확인이 끝나면 오른쪽 아래 **단어장 생성** 버튼을 눌러 주세요.
{% endstep %}
{% endstepper %}

## 생성이 시작되면

**단어장 생성을 시작했어요** 라는 안내가 표시됩니다. 단어장은 여러 개의 수업이 담긴 **책 형태**로 만들어집니다.

* 생성은 백그라운드에서 진행되며, 잠시 후 수업 목록에서 확인할 수 있습니다.
* 안내창의 **수업 보러가기**를 누르면 만들어진 책으로 이동합니다.

{% hint style="info" %}
단어 수가 많아 수업이 여러 개일 경우, 모두 만들어지기까지 시간이 조금 걸릴 수 있습니다.
{% endhint %}
