# 원아워로 중고등 내신대비 200% 활용하기 / 오만한 영어 이태연 원장 (26.04.17)

\*본 가이드는 4/17(금)라이브로 진행된 세미나를 바탕으로 구성되었습니다.

### **I. 이런 분들에게 추천해요**

* 내신대비 커리큘럼을 짜야 하는데 막막한 분
* 내신 기간마다 문제 제작, 인쇄, 채점, 오답 관리에 시간을 쏟느라 지쳐가는 분
* 교과서 범위는 넓어지고 외부지문·서술형 비중은 확대되는데, 자료 준비가 벅찬 분
* 강사 개인기나 밤샘 작업이 아닌, 지속 가능한 내신대비 루틴을 만들고 싶은 분
* 학부모 상담 시 아이의 '노력'을 데이터로 보여주고 싶은 분

### **III. 세미나 플로우 직접 따라해보기**

#### **Part 1. 입력: 원하는 지문, 1초 만에 세팅하기**

없는 자료는? '복붙' 하세요. 외부 지문, 부교재... 텍스트만 있다면 즉시 수업이 됩니다. 타이핑도 귀찮다면? '사진'을 찍으세요.

1. **텍스트로 지문 입력하기:** 시험 범위 본문이나 부교재 지문이 정리되어 있다면, 복사/붙여넣기만으로 즉시 수업 자료가 생성됩니다.

* [_자세한 가이드는 해당 페이지를 확인해 주세요!_](https://1hour.gitbook.io/guide/teacher/undefined-1/1./ii.)

2. **사진으로 지문 입력하기:** 꾸깃꾸깃한 프린트물도 사진 한 장이면 디지털 수업이 됩니다. 촬영 → AI 인식 → 원어민 음성 생성 → 문제 자동 생성까지 한 번에 이어져요.

* [_자세한 가이드는 해당 페이지를 확인해 주세요!_](https://1hour.gitbook.io/guide/teacher/undefined-1/1./i.)

#### **Part 2. 생성: 클릭 한 번에 쏟아지는 200문제**

어휘 + 문법 + 독해 + 기출문제 = 1분 만에 200문제 준비 완료. 클릭만 잘하면 됩니다.

* 지문 입력 후 원하는 문제 유형을 체크하면, 변형 문제와 정답지, 해설지가 한 번에 생성됩니다.
* _**Part 1의 가이드에서 지문 입력 후 이어지는 단계를 따라가시면 됩니다.**_

#### **Part 3. 훈련: 문제 풀기 전, 빈틈없는 암기 루틴**

무작정 풀리지 않습니다. 입과 손으로 먼저 체화합니다.

1. **단어/문장 학습 + 교재 풀이:** 플래시 카드로 의미와 발음을 노출하고, 스펠링 학습으로 철자 감각을 만들어 주세요. AI 더빙으로 학생이 직접 소리 내어 읽고 발음 피드백을 받을 수 있어요. 매일 같은 루틴으로 안정적인 반복 학습을 세팅해 주세요.
   * [_과제 유형 알아보기_](https://1hour.gitbook.io/guide/teacher/assignment-management/assignment-type#undefined-9)
   * [_과제 생성하기_](https://1hour.gitbook.io/guide/teacher/assignment-management/create)

<figure><img src="../../.gitbook/assets/스크린샷 2026-04-22 오후 4.14.19.png" alt=""><figcaption></figcaption></figure>



2. **서술형 영작:** 영작의 꽃! 조건부 서술형 영작을 부여하면, AI가 즉각 채점하여 감점 요인과 해설을 제공합니다.

* [_자세한 가이드는 해당 페이지를 확인해 주세요!_](https://1hour.gitbook.io/guide/teacher/undefined-2)

<figure><img src="../../.gitbook/assets/image (37).png" alt=""><figcaption></figcaption></figure>





3. **레슨별 타겟 문법이 필요하다면?** 디테일의 끝, '문법 프롬프트'. 우리 학교 타겟 문법, '한 줄' 요청으로 핀셋 공략할 수 있습니다.

* 예를 들어 "중학교 1학년 대상, be동사 평서문, 부정문, 의문문을 현재/과거형 섞어서 30문제 만들어줘"라고 입력하면 맞춤 문법 문제가 자동 생성됩니다.
* 개념 확인부터 실전 적용까지 레벨별 문항(어순배열/빈칸채우기)이 함께 만들어져요.
* _(가이드 준비 중)_



#### **Part 4. 실전: 연습은 끝났다, 시험 모드 ON**

훈련 모드에서는 틀리면 다시 풀게 하고 힌트를 주지만, '시험 모드'를 켜면 이 모든 것이 차단되고 타이머가 돌아갑니다.

1. **시험 모드 활용하기:** 과제 부여 시 '시험 모드' 옵션을 켜 주세요. 재도전 불가, 힌트 없음, 시간 제한이 적용됩니다.
   * [_자세한 가이드는 해당 페이지를 확인해 주세요!_](https://1hour.gitbook.io/guide/teacher/assignment-management/create)
2. **문제은행 모아서 인쇄하기:** 변형문제를 모아 오프라인 시험지로 인쇄할 수 있어요.
   * _(가이드 준비 중)_
3. **동형 모의고사:** 미리 보는 학교 시험지. 여러 지문을 합쳐 실제 학교 시험지 형태(PDF/인쇄용)로 완벽하게 구현할 수 있습니다.
   * [_자세한 가이드는 해당 페이지를 확인해 주세요!_](https://1hour.gitbook.io/guide/teacher/undefined-1/1./iii.)

#### **마무리**

1. **이태연 원장님이 제안하는 내신 5주 완성 커리큘럼**
   * 1주차: 1과 본문 / 단어 암기
   * 2주차: 1과 실전 문제풀이
   * 3주차: 2과 본문/단어 암기 + 학교 기출
   * 4주차: 2과 실전 문제풀이
   * 5주차: Final 시험 모드 + 학교 기출
2. **학부모가 감동하는 '증거', 리포트:** 성적표만 보내지 마세요. 아이의 '노력'을 보내세요.
