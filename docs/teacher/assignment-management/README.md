# 과제 관리

> 과제는 학생이 원아워에서 온라인으로 수행할 수 있도록 출제하는 학습 단위입니다.
>
> 원아워에서는 수업 단위로 과제를 출제하는 **수업 과제**와,
>
> 여러 수업을 묶은 책 단위로 출제하는 **책 과제** 두 가지 방식이 제공됩니다.

### 각 과제 유형별 특징

#### 수업 과제&#x20;

* 하나의 수업을 기반으로 과제를 부여합니다.
* 수업에 포함된 단어, 문장, 문제 세트중 선택하여 부여 가능합니다.

**추천 사용 상황**

* 개별 수업 콘텐츠에 맞춰 학습시키고 싶을 때

#### 책 과제

* 여러 수업이 포함된 '책' 단위를 기준으로 과제를 부여합니다.
* 책에 포함된 수업을 다중 선택하여 한번에 여러 개의 과제를 부여할 수 있습니다.
* 책 상세 페이지에서 생성할 수 있습니다.

**추천 사용 상황**

* 커리큘럼 단위로 일괄 출제하고 싶을 때
* 하나의 주제나 주차별 학습 단위를 자동 구성하고 싶을 때
