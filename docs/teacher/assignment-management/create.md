과제 선택

{% tabs %}
{% tab title="옵션 설정" %}
과제 수행 시 적용될학습 옵션을 설정할 수 있습니다.

<table><thead><tr><th width="158.00006103515625">옵션명</th><th>설명</th></tr></thead><tbody><tr><td><strong>무음</strong></td><td>음원 없이 과제를 수행하게 합니다.</td></tr><tr><td><strong>영상/PDF 숨김</strong></td><td>영상이나 PDF가 보이지 않도록 설정합니다.</td></tr><tr><td><strong>문장 번역 숨김</strong></td><td>문장 번역(해석)을 비활성화합니다.</td></tr><tr><td><strong>영상 시청</strong></td><td>수업 영상 또는 음성을 끝까지시청하고 과제를 수행합니다.</td></tr><tr><td><strong>시험 모드</strong></td><td>오답 시 재도전 없이 한 번만 문제를 풀 수 있는 모드입니다.</td></tr></tbody></table>
{% endtab %}

{% tab title="단어" %}
출제하고 싶은 단어를 선택합니다. 일부 혹은 전체 선택 가능

* **유형**: 단어 고르기, 해석 고르기, 단어 써보기
* **옵션**: 플래시카드 기능 ON/OFF
* **활용 팁**: 단어 암기 중심 학습에 적합합니다.
{% endtab %}

{% tab title="문장" %}
* **유형**: 어순 배열 (하, 중, 상), 받아쓰기 (하, 중, 상), Grammar AI
* **난이도** &#x20;
  * 하 : 영문의 40%가 마스킹
  * 중 : 영문의 60\~70%가 마스킹
  * 상 : 영문 전체가 마스킹
* **옵션**: 학습 시 동일 문장을 반복하여 풀 수 있도록 설정
* **활용 팁**: 문장 구조와 문법 감각을 익히는 데 적합합니다.
{% endtab %}

{% tab title="문제" %}
* **유형**: 문제
* **옵션**: 문제 셔플, 보기 셔플
* **활용 팁**: 단원 이해도 점검용으로 활용됩니다.
{% endtab %}

{% tab title="말하기" %}
* **유형**: 더빙, 통시통역
* **난이도**&#x20;

<table><thead><tr><th width="93.81817626953125">난이도</th><th>더빙</th><th>프리토킹</th></tr></thead><tbody><tr><td>하</td><td>영문, 해석 전체 제시</td><td>영문, 해석 전체 제시</td></tr><tr><td>중</td><td>영문의 일부 마스킹</td><td>해석만 제시(동시통역)</td></tr><tr><td>상</td><td>해석만 제시</td><td>전체 마스킹(쉐도잉)</td></tr></tbody></table>

* **옵션**: 1배속 고정 또는 속도 조절
* **활용 팁**: 소리영어, 듣기 및 말하기 훈련에 활용됩니다.
{% endtab %}

{% tab title="영작 AI" %}
* **유형**: 단어 영작, 문장 영작
* **옵션**: 자동 채점 지원 (일부 항목)
* **활용 팁**: 문장 구성 및 표현력 향상에 유용합니다.
{% endtab %}
{% endtabs %}
{% endstep %}

{% step %}
날짜, 휴일 설정

<figure><img src="../../.gitbook/assets/스크린샷 2025-08-27 오후 2.52.30.png" alt=""><figcaption></figcaption></figure>

{% tabs %}
{% tab title="시작 날짜 선택" %}
과제를 시작할 날짜를 캘린더에서 클릭합니다.

* 선택한 날짜부터 캘린더에 과제가 자동 배정됩니다.
* 출제 가능 날짜 : 당일부터 6개월 이내

{% hint style="warning" %}
출제 당일보다 과거 날짜 혹은 6개월 이상 미래 날짜에는 과제 출제 불가
{% endhint %}
{% endtab %}

{% tab title="과제 분할 / 반복 설정" %}
수업 하나는 기본적으로 과제 1개로 부여되고 분량은 옵션으로 조절합니다.

* **수업 분할**: 하나의 수업을 몇 개 과제로 나눌지 설정합니다.
* **하루 분량**: 학생이 하루에 풀어야 할 과제 수를 설정합니다.

{% hint style="info" %}
수업 분할과 하루 분량을 함께 사용할 수 없습니다.&#x20;
{% endhint %}
{% endtab %}

{% tab title="수행 요일" %}
과제를 출제할 요일을 선택합니다.

* 선택한 요일만 캘린더에 과제가 자동 배정됩니다.
* 주 2\~3일 학습 등 유연한 계획에 적합합니다.
{% endtab %}

{% tab title="휴일 설정" %}
휴을 개별적으로 설정합니다.

* 토글을 켜면 캘린더에서 제외할 날짜를 직접 선택할 수 있습니다.
* 쉬는 날로 설정된 날짜에는 과제가 배정되지 않습니다.
{% endtab %}
{% endtabs %}
{% endstep %}

{% step %}
클래스 학생 선택

<figure><img src="../../.gitbook/assets/스크린샷 2025-08-27 오후 2.52.45.png" alt=""><figcaption></figcaption></figure>

{% tabs %}
{% tab title="클래스 선택" %}
과제를 부여할 클래스를 선택합니다.

* 클래스를 선택하면 해당 클래스에 속한 학생 목록이 학생 선택에 표시됩니다.
* 돋보기 아이콘을 클릭하면 클래스 목록을 검색할 수 있습니다.
{% endtab %}

{% tab title="학생 선택" %}
과제를 부여할 학생을 선택합니다.

* 클래스 선택 후, 전체 선택 또는 일부 학생만 개별 선택할 수 있습니다.
* 검색창에서 학생명을 검색하여 개별 지정할 수도 있습니다.
{% endtab %}
{% endtabs %}
{% endstep %}
