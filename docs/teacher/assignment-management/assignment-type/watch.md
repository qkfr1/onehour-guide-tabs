# 콘텐츠 시청 과제

### 콘텐츠 시청 과제란?&#x20;

수업에 포함된 영상 또는 음원을 학생이 끝까지 시청하도록 과제로 부여하는 기능입니다.

#### 학생 화면에서 체험하기

{% embed url="https://app.storylane.io/share/qdxjuqsngl0y" %}

#### 학생이 열심히 콘텐츠를 시청했는지 어떻게 확인하나요?&#x20;

1. 미시작 상태에서는 **과제 시작**, 진행 중에는 **이어서 하기** 버튼이 표시됩니다.
2. **영상 앞으로 건너뛰기가 차단**됩니다. 클릭 시 "영상을 처음부터 끝까지 시청해야 해요" 안내가 표시됩니다.
3. 다른 탭으로 이동하면 영상이 자동으로 일시정지됩니다. 탭으로 돌아오면 직접 재생 버튼을 눌러야 합니다.
4. **중간에 나가면** 과제가 완료처리 되지 않고 **수행 중 상태로 저장**되며 이어서 볼 수 있습니다.

### **기간 및 옵션 설정**

<figure><img src="../../../.gitbook/assets/스크린샷 2026-03-27 오후 1.53.09.png" alt=""><figcaption></figcaption></figure>

* 다른 과제(단어, 문장 등)와 함께 분할 과제로 설정할 경우, **콘텐츠 시청은 한번만** 옵션을 선택할 수 있습니다.
  * **ON**: 분할된 첫날에만 콘텐츠 시청 과제가 생성됩니다.
  * **OFF**: 분할된 날마다 콘텐츠 시청 과제가 생성됩니다.
* 콘텐츠 시청 과제만 단독으로는 분할 설정이 불가합니다.

### 어떤 수업으로 콘텐츠 시청 과제를 낼 수 있나요?

영상 또는 음원이 포함된 수업에만 콘텐츠 시청 과제를 부여할 수 있습니다.

| 과제 부여 가능 O    | 과제 부여 불가 x     |
| ------------- | -------------- |
| 유튜브·비메오·영상 수업 | PDF만 있는 수업     |
| PDF + 영상 수업   | 이미지·텍스트 수업     |
| PDF + 음원 수업   | 유튜브 비공개·삭제된 영상 |
| 음원 수업         |                |

{% hint style="info" %}
과제를 낼 수 없는 수업을 선택하고 과제 내기 시 콘텐츠 학습 섹션 자체가 표시되지 않습니다.
{% endhint %}

### 자주 묻는 질문

**Q. 영상이 걸거나 짧아도 동일하게 동작하나요?**&#x20;

네, 영상 길이와 관계없이 동일한 정책이 적용됩니다.

**Q. 과제가 완료된 후 학생이 영상을 다시 볼 수 있나요?**&#x20;

완료된 과제로는 재진입이 불가합니다. 영상을 다시 보려면 \[학습 기능] - \[수업 시작]을 이용해 시청해주세요.
