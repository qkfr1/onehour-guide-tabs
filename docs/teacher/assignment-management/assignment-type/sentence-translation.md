# 해석 과제

통문장 해석은 영어 문장 전체를 한국어로 번역하고 **AI가 자동 채점**하는 새로운 과제 유형입니다.

학생이 입력한 해석을 의미 전달·문법 정확성·어휘 사용·자연스러움 4가지 기준으로 분석하여 즉시 피드백을 제공합니다.

{% hint style="info" %}
통문장 해석은 **쓰기** 카테고리에 포함됩니다. 대시보드와 리포트에서 영작(에세이)과 함께 쓰기 영역으로 집계됩니다.
{% endhint %}

## 통문장 해석 과제 내기

1. 상단 메뉴에서 **수업 관리**를 클릭합니다.
2. 과제를 내고자 하는 수업을 선택합니다.
3. **과제 내기** 버튼을 클릭합니다.
4. 학습활동(Task) 추가 목록에서 **통문장 해석**을 선택합니다.
5. 문장 범위를 설정하고 아래 옵션을 구성합니다.

## 통문장 해석 과제 수행

1. 홈 화면의 해당 날짜의 과제 목록을 확인합니다.
2. **통문장 해석** 과제의 **과제 시작** 버튼을 눌러 학습을 시작합니다.
3. 피드백 확인 후 **다시 해석하기**를 누르면 다시 풀 수 있고, **다음 문장**을 누르면 다음 문제로 넘어갑니다.

{% hint style="info" %}
**이어서 하기 가능**

과제 도중 나간 경우, **이어서 하기**를 눌러마지막으로 제출한 문장 다음부터 이어서 진행할 수 있습니다.
{% endhint %}
