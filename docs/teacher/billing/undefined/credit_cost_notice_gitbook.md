# credit\_cost\_notice\_gitbook

6월 18일부터 적용되는 수업 생성 AI 크레딧과 관련해, 원장님들께서 미리 사용량을 가늠해보실 수 있도록 문제 유형별 예상 크레딧을 안내드립니다.

아래 내용은 실제 운영을 돕기 위한 예상 안내이며, 정확한 차감량은 콘텐츠 종류와 분량에 따라 달라질 수 있습니다.

{% hint style="info" %}
**크레딧(CP) 기준**\
1 CP = 20원 (부가세 포함) 기준으로 환산됩니다.
{% endhint %}

***

## 문제 유형별 예상 크레딧

여기서 **1세트**는 문제 유형 한 가지를 한 번 생성하는 단위로, 보통 10\~20문제가 함께 생성됩니다.

### 가볍게 사용하는 유형 (1세트 10\~20 CP)

| 문제 유형           | 1세트 예상 CP |
| --------------- | --------- |
| 내용이해 T/F        | 20 CP     |
| 내용이해 객관식        | 20 CP     |
| 키즈 내용이해 / T/F   | 20 CP     |
| 영어 해설 (객관식·T/F) | 10 CP     |

### 보통 사용량의 유형 (1세트 50\~80 CP)

| 문제 유형           | 1세트 예상 CP |
| --------------- | --------- |
| 단어 (쓰임·영영풀이)    | 50 CP     |
| 문법 (본문분석·연습·기출) | 60 CP     |
| 파이널 내신·수능       | 80 CP     |
| 내용이해 기출형        | 80 CP     |

### 비용이 큰 유형 — 별도 확인 권장

| 문제 유형  | 예상 CP        |
| ------ | ------------ |
| 동형모의고사 | 문제 1개당 25 CP |

{% hint style="warning" %}
**동형모의고사는 사용량 확인이 필요합니다**\
동형모의고사는 다른 유형과 달리 문제 1개씩 생성되는 방식입니다.\
20문제 기준 1회 생성 시 약 500 CP가 사용됩니다.
{% endhint %}

***

## 기본 크레딧으로 어느 정도 사용 가능한가요?

기본 크레딧은 결제 금액의 일정 비율로 매월 자동 충전되며, 학생 수가 많아질수록 함께 늘어납니다.

### 학원 규모별 기본 크레딧 (참고)

| 학생 수 | 월 기본 크레딧   |
| ---- | ---------- |
| 10명  | 약 250 CP   |
| 30명  | 약 750 CP   |
| 50명  | 약 1,250 CP |
| 100명 | 약 2,500 CP |

### 예시 — 학생 30명 학원 (월 기본 크레딧 약 750 CP)

| 만드는 콘텐츠         | 1개당 CP | 월 생성 가능 |
| --------------- | ------ | ------- |
| 내용이해 객관식 (20문제) | 20 CP  | 약 38개   |
| 파이널 내신 (20문제)   | 80 CP  | 약 9개    |
| 동형모의고사 (20문제)   | 500 CP | 약 1\~2개 |

## 참고해 주세요

* 위 수치는 4월 사용 데이터를 기준으로 한 예상치입니다.
* 같은 문제 유형이라도 지문 길이와 분량에 따라 차감량이 달라집니다.
