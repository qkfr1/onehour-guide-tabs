# 결제 수단(카드) 관리

### **카드 등록**

서비스 무료 체험 종료 전 또는 유료 전환 시 최초 1회 등록이 필요합니다.

1. 관리자 페이지 접속 후 오른쪽 상단 \[이름-기관명] 클릭
2. \[구독 결제 관리] > \[카드 등록] 버튼 클릭
3. \[개인 카드] 또는 \[법인 카드] 중 카드 유형을 선택
4. 카드 정보 입력 후 \[등록하기] 버튼 클릭

### **카드 변경 및 삭제**

사용 중인 카드가 만료되거나 다른 카드로 변경을 원하실 때 이용합니다.

1. \[구독 결제 관리] > \[카드 변경] 또는 \[카드 삭제] 버튼 클릭
2. 기존 카드를 삭제하거나 새로운 카드 정보를 입력하여 갱신
