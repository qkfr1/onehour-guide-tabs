## 어떤 플랜을 선택하면 되나요?

원아워를 어떻게 사용하고 계신지에 따라 아래와 같이 선택하시면 됩니다.

| 이렇게 사용하신다면               | 추천 플랜       |
| ------------------------ | ----------- |
| 기본 과제, 시험, 자동 채점, 리포트 중심 | Basic       |
| 말하기 수업과 발음 평가를 자주 활용     | Speaking AI |
| 영작, 해석, 내신 서술형 대비가 중요    | Writing AI  |
| 말하기 AI와 쓰기 AI를 모두 활용     | AI Pro      |

#### 말하기와 쓰기 수업을 함께 운영하시는 학원에는 AI Pro 플랜을 가장 추천드립니다.

## 수업 생성 AI 크레딧이란?

수업 생성 AI 크레딧(CP)은 문제 생성, 수업 자료 생성 등 AI로 수업 콘텐츠를 만들 때 사용하는 크레딧입니다.

각 요금제에는 월 결제 금액의 일정 비율에 해당하는 크레딧이 매월 기본 제공되며, **기본 제공량을 초과해 사용하시는 경우에만 추가 비용이 발생**합니다.

{% hint style="info" %}
**크레딧(CP) 기준** 1 CP = 20원 (부가세 포함) 기준으로 환산됩니다.
{% endhint %}

| 플랜          | 기본 크레딧        |
| ----------- | ------------- |
| Basic       | 결제 금액의 10% 상당 |
| Speaking AI | 결제 금액의 12% 상당 |
| Writing AI  | 결제 금액의 12% 상당 |
| AI Pro      | 결제 금액의 15% 상당 |

수업 생성 AI 크레딧의 [자세한 사용량 안내는 별도 문서를 참고](cms-credit.md)해 주세요.

## 비즈니스 파트너 플랜

원아워를 단순한 비용이 아니라 학원의 새로운 수익 모델로 활용하고 싶은 원장님께는 비즈니스 파트너 플랜을 별도로 안내드립니다.

비즈니스 파트너 플랜은 학부모가 결제하고, 학원은 수익을 정산받는 구조입니다.&#x20;

원아워를 활용한 온라인 과제, AI 평가, 학습 관리 서비스를 학원의 유료 상품으로 운영하실 수 있습니다.

비즈니스 파트너 플랜의 경우 기본 크레딧이 높은 비율로 제공됩니다.

| 플랜          | 기본 크레딧        |
| ----------- | ------------- |
| Basic       | 결제 금액의 15% 상당 |
| Speaking AI | 결제 금액의 17% 상당 |
| Writing AI  | 결제 금액의 17% 상당 |
| AI Pro      | 결제 금액의 20% 상당 |

비즈니스 파트너 플랜에 관심 있으신 원장님께서는 [별도 문서를 참고](../student.md)해주세요.
