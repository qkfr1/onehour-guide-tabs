# 크래딧 사용량

원아워에서 AI를 활용해 수업과 문제를 만드실 때 사용되는 크레딧(CP) 안내입니다.&#x20;

원장님께서 미리 사용량을 가늠해보실 수 있도록 입력 유형별·문제 유형별 예상 크레딧을 정리했습니다.

{% hint style="info" %}
**적용 일정**\
2026년 8월 1일부터 수업 생성과 문제 생성 전 과정에 크레딧이 일괄 적용됩니다.

**크레딧(CP) 기준**\
1 CP = 20원 (부가세 포함) 기준으로 환산됩니다.
{% endhint %}

***

## 크레딧은 언제 사용되나요?

크레딧은 AI가 작동하는 두 단계에서 사용됩니다.

**1단계 — 수업 생성**\
영상·음원·PDF·이미지 등을 업로드하여 수업 자료를 만드는 과정에서 AI가 작동합니다.

**2단계 — 문제 생성**\
만들어진 수업 자료를 바탕으로 문제를 생성할 때 AI가 작동합니다.

***

## 실전 예시

원장님이 자주 만드시는 형태로 환산해보면 다음과 같습니다.

**예시 1. 유튜브 영상으로 수업 만들고 문제 생성**

5분짜리 유튜브 영상 + 내용이해 객관식 10문제

* 수업 생성 약 30 CP + 문제 생성 20 CP = **약 50 CP**

**예시 2. PDF로 수업 만들고 내신 대비 문제 생성**

문장 100개 PDF + 파이널 내신 20문제

* 수업 생성 약 40 CP + 문제 생성 80 CP = **약 120 CP**

**예시 3. 음원과 PDF로 수업 만들고 문법 문제 생성**

5분 음원 + PDF 50문장 + 문법 20문제

* 수업 생성 약 212 CP + 문제 생성 60 CP = **약 272 CP**

## 수업 생성 시 예상 크레딧 (입력 유형별)

수업을 만드실 때 어떤 자료를 업로드하시는지에 따라 사용되는 크레딧이 달라집니다. \
아래는 자주 사용되는 입력 유형별 평균 크레딧입니다.

### 영상으로 수업 만들기

영상의 길이에 비례해 크레딧이 사용됩니다.

| 영상 길이   | 평균 사용 CP |
| ------- | -------- |
| 2분 미만   | 약 14 CP  |
| 2\~5분   | 약 30 CP  |
| 5\~10분  | 약 55 CP  |
| 10\~20분 | 약 100 CP |

_유튜브 URL과 영상 파일 업로드 모두 비슷한 수준으로 사용됩니다._

### 음원으로 수업 만들기

음원 길이에 비례해 크레딧이 사용됩니다.

| 음원 길이   | 평균 사용 CP |
| ------- | -------- |
| 2분 미만   | 약 11 CP  |
| 2\~5분   | 약 32 CP  |
| 5\~10분  | 약 69 CP  |
| 10\~20분 | 약 107 CP |

### PDF로 수업 만들기

PDF에 포함된 문장 수에 따라 크레딧이 사용됩니다.

| PDF 문장 수   | 평균 사용 CP |
| ---------- | -------- |
| 1\~10문장    | 약 16 CP  |
| 11\~30문장   | 약 21 CP  |
| 31\~50문장   | 약 19 CP  |
| 51\~100문장  | 약 40 CP  |
| 101\~200문장 | 약 61 CP  |
| 200문장 이상   | 약 81 CP  |

### 이미지(사진)로 수업 만들기

이미지에서 추출된 문장 수에 따라 크레딧이 사용됩니다.

| 이미지 문장 수  | 평균 사용 CP |
| --------- | -------- |
| 1\~10문장   | 약 8 CP   |
| 11\~30문장  | 약 12 CP  |
| 31\~50문장  | 약 30 CP  |
| 51\~100문장 | 약 34 CP  |
| 100문장 이상  | 약 50 CP  |

### 음원 + PDF 함께 사용

음원과 PDF를 함께 업로드하시는 경우는 두 자료가 모두 처리되어 크레딧 사용량이 큰 편입니다.

| 음원 길이   | 평균 사용 CP |
| ------- | -------- |
| 2분 미만   | 약 42 CP  |
| 2\~5분   | 약 88 CP  |
| 5\~10분  | 약 212 CP |
| 10\~20분 | 약 181 CP |

### 본문으로 직접 생성 (텍스트 입력)

원장님이 직접 입력하신 본문을 기반으로 수업을 만드시는 경우입니다.

| 본문 문장 수   | 텍스트 기반 CP |
| --------- | --------- |
| 1\~10문장   | 약 7 CP    |
| 11\~30문장  | 약 10 CP   |
| 31\~50문장  | 약 17 CP   |
| 51\~100문장 | 약 22 CP   |
| 100문장 이상  | 약 45 CP   |

### 강의 / 내신 맞춤 강의

기본 문제 20개가 자동 포함되는 카드입니다. 입력 길이와 관계없이 평균 7\~9 CP 수준으로 사용됩니다.

{% hint style="warning" %}
**평균값 안내**\
위 수치는 평균 사용량입니다. 자료의 분량, 내용 복잡도에 따라 사용량은 최대 2\~3배까지 달라질 수 있습니다.
{% endhint %}

## 문제 생성 시 예상 크레딧 (문제 유형별)

여기서 **1세트**는 문제 유형 한 가지를 한 번 생성하는 단위로, 보통 10\~20문제가 함께 생성됩니다.

### 일반 문제 유형

| 문제 유형           | 1세트 문제 수 | 1세트 예상 CP |
| --------------- | -------- | --------- |
| 내용이해 T/F        | 10문제     | 20 CP     |
| 내용이해 객관식        | 10문제     | 20 CP     |
| 키즈 내용이해 / T/F   | 10문제     | 20 CP     |
| 영어 해설 (객관식·T/F) | 10문제     | 10 CP     |
| 단어 (쓰임·영영풀이)    | 10문제     | 50 CP     |
| 문법 (본문분석·연습·기출) | 20문제     | 60 CP     |
| 파이널 내신·수능       | 20문제     | 80 CP     |
| 내용이해 기출형        | 20문제     | 80 CP     |

### 동형 모의고사

동형 모고는 문제를 직접 추가하는 방식으로, 기본 문제 수가 지정되어 있지 않습니다. 아래는 문제 20개 기준 예시입니다.

| 문제 유형  | 문제 1개당 CP | 1세트 예상 CP (20문제 기준) |
| ------ | --------- | ------------------- |
| 동형모의고사 | 25 CP     | 500 CP              |

## 기본 크레딧으로 어느 정도 사용 가능한가요?

기본 크레딧은 결제 금액의 일정 비율로 매월 자동 충전되며, 학생 수가 많아질수록 함께 늘어납니다.

### 학원 규모별 기본 크레딧&#x20;

예시 )**30명의 학생**과 **Speaking ai 플랜**을 사용하는 학원의 경우,&#x20;

**5분짜리 유튜브 영상 + 내용이해 객관식 10문제 수업**을 한 달에 약 **28개** 만드실 수 있습니다.

| 학생 수 | Basic (5,900원·10%) | Speaking·Writing (7,900원·12%) | AI Pro (9,900원·15%) |
| ---- | ------------------ | ----------------------------- | ------------------- |
| 10명  | 295 CP             | 470 CP                        | 740 CP              |
| 30명  | 885 CP             | 1,420 CP                      | 2,227 CP            |
| 50명  | 1,475 CP           | 2,370 CP                      | 3,712 CP            |
| 100명 | 2,950 CP           | 4,740 CP                      | 7,425 CP            |

## 크레딧 수치는 어떻게 정해지나요?

원아워의 AI 기능은 OpenAI 등 외부 AI 서비스를 활용해 제공됩니다. 외부 AI 사용료는 처리되는 토큰(텍스트·음성·이미지의 단위)에 따라 달러로 결제되며, 원아워는 이 비용을 기준으로 크레딧을 산정하고 있습니다.

이러한 구조 때문에 크레딧 수치는 환율과 외부 AI 사용료에 따라 변동될 수 있습니다.

예를 들어 어제 ‘내용이해 T/F 10문제’ 생성에 20 CP가 사용되었더라도, 환율이 오른 다음 날에는 동일한 작업에 24 CP가 사용될 수 있습니다.

본 문서의 수치는 평균 예상치이며, 실제 차감되는 크레딧은 작업 시점의 환율과 사용료에 따라 달라질 수 있다는 점을 참고해 주시면 감사하겠습니다.

* 동형 모의고사처럼 문제 수가 많은 유형은 사용량이 빠르게 늘어날 수 있으니 미리 확인해 주시면 좋습니다.
