# 수업 생성 AI 크레딧

6월 18일부터 적용되는 수업 생성 AI 크레딧과 관련해, 원장님들께서 미리 사용량을 가늠해보실 수 있도록 **문제 유형별 예상 크레딧을 안내**드립니다.

{% hint style="info" %}
## **크레딧(CP) 기준**

#### 1 CP = 20원 (부가세 포함) 기준으로 환산됩니다.
{% endhint %}

***

## 문제 유형별 예상 크레딧

여기서 **1세트**는 문제 유형 한 가지를 한 번 생성하는 단위로, 보통 10\~20문제가 함께 생성됩니다.

### 일반 문제 유형

| 문제 유형           | 유형별 문제 생성 수 | 1세트 예상 CP |
| --------------- | ----------- | --------- |
| 내용이해 T/F        | 10문제        | 20 CP     |
| 내용이해 객관식        | 10문제        | 20 CP     |
| 키즈 내용이해 / T/F   | 10문제        | 20 CP     |
| 영어 해설 (객관식·T/F) | 10문제        | 10 CP     |
| 단어 (쓰임·영영풀이)    | 10문제        | 50 CP     |
| 문법 (본문분석·연습·기출) | 20문제        | 60 CP     |
| 파이널 내신·수능       | 20문제        | 80 CP     |
| 내용이해 기출형        | 20문제        | 80 CP     |

## 기본 크레딧으로 어느 정도 사용 가능한가요?

기본 크레딧은 결제 금액의 일정 비율로 매월 자동 충전되며, 학생 수가 많아질수록 함께 늘어납니다.

### 학원 규모별 기본 크레딧 (참고)

| 학생 수 | 월 기본 크레딧   |
| ---- | ---------- |
| 10명  | 약 250 CP   |
| 30명  | 약 750 CP   |
| 50명  | 약 1,250 CP |
| 100명 | 약 2,500 CP |
