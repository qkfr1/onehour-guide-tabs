# 온라인 학원 만들기

#### I. 이런 분들에게 추천해요

* 유튜브 강의를 활용해 학생들에게 체계적인 온라인 과제를 운영하고 싶은 분
* 영상 시청, 퀴즈, 영작, 발음평가까지 한 번에 관리할 수 있는 온라인 학원 시스템이 필요한 분
* 학생별 과제 결과와 점수를 한눈에 확인하고, 점수와 대시보드로 효율적으로 관리하고 싶은 분

#### III. 이미지 보며 따라하기

1. **우상단 "+ 수업 생성" 버튼 클릭 후 "유튜브" 버튼을 선택해 주세요.**

2. **활용하고자 하시는 유튜브 강의 링크를 넣어주세요.**

<figure><img src="../.gitbook/assets/스크린샷 2026-04-29 오후 8.13.39 (1).png" alt=""><figcaption></figcaption></figure>



3. **강의를 통해 학생들에게 학습 시키고 싶은 변형 문제를 선택해 주세요.**

* 우선, 문제를 선택하지 않고 수업을 만드는 것도 가능하며, "수업 생성" 바로 클릭하시면 됩니다.

<figure><img src="../.gitbook/assets/스크린샷 2026-04-29 오후 8.14.08 (1).png" alt=""><figcaption></figcaption></figure>



4. **"작업목록" 진행 중 상태가 완료되면 문제 생성이 끝나요.**

* 굳이 '문제 생성 시간'을 기다리지 않아도 됩니다. 동시에 다른 문제를 바로 생성하실 수 있어요!

<figure><img src="../.gitbook/assets/스크린샷 2026-04-29 오후 8.15.01 (1).png" alt=""><figcaption></figcaption></figure>



5. **문제 생성이 완료되었다면 "수업관리"에서 확인하실 수 있어요.**
   * 이후 아래 2가지 용도에 따라 활용하실 수 있습니다.
     * \[과제 내기] 온라인 학습용, 학생에게 과제 부여 [\*자세한 가이드는 해당 페이지를 확인해 주세요!](https://1hour.gitbook.io/guide/teacher/assignment-management)
     * \[학습지 만들기] 오프라인 수업용, 프린트 학습지를 만드는 용도
