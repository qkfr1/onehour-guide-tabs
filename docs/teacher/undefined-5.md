# 온라인 학원 만들기

#### I. 이런 분들에게 추천해요

* 유튜브 강의를 활용해 학생들에게 체계적인 온라인 과제를 운영하고 싶은 분
* 영상 시청, 퀴즈, 영작, 발음평가까지 한 번에 관리할 수 있는 온라인 학원 시스템이 필요한 분
* 학생별 과제 결과와 점수를 한눈에 확인하고, 점수와 대시보드로 효율적으로 관리하고 싶은 분



#### II. 이런 결과를 얻으실 수 있어요

* 유튜브 강의를 바로 온라인 과제로 전환하여 수업 준비 시간을 줄일 수 있습니다.
* 영상 시청, 퀴즈, 영작, 발음평가까지 한 번에 운영할 수 있어 과제 구성이 훨씬 체계적입니다.
* 틀린 문제를 자동으로 다시 제시해 학생의 반복 학습과 복습 효과를 높일 수 있습니다.
* 학생별 과제 결과와 점수를 대시보드에서 한눈에 확인할 수 있어 관리가 훨씬 효율적입니다.
* 결과적으로 과제 생성부터 운영, 확인, 피드백까지의 과정을 더 쉽고 빠르게 진행할 수 있습니다.

{% tab title="단어" %}
출제하고 싶은 단어를 선택합니다. 일부 혹은 전체 선택 가능

* **유형**: 단어 고르기, 해석 고르기, 단어 써보기
* **옵션**: 플래시카드 기능 ON/OFF
* **활용 팁**: 단어 암기 중심 학습에 적합합니다.
{% endtab %}

{% tab title="문장" %}
* **유형**: 어순 배열 (하, 중, 상), 받아쓰기 (하, 중, 상), Grammar AI
* **난이도** &#x20;
  * 하 : 영문의 40%가 마스킹
  * 중 : 영문의 60\~70%가 마스킹
  * 상 : 영문 전체가 마스킹
* **옵션**: 학습 시 동일 문장을 반복하여 풀 수 있도록 설정
* **활용 팁**: 문장 구조와 문법 감각을 익히는 데 적합합니다.
{% endtab %}

{% tab title="문제" %}
* **유형**: 문제
* **옵션**: 문제 셔플, 보기 셔플
* **활용 팁**: 단원 이해도 점검용으로 활용됩니다.
{% endtab %}

{% tab title="말하기" %}
* **유형**: 더빙, 통시통역
* **난이도**&#x20;

* **활용 팁**: 문장 구성 및 표현력 향상에 유용합니다.

{% tab title="시작 날짜 선택" %}
과제를 시작할 날짜를 캘린더에서 클릭합니다.

* 선택한 날짜부터 캘린더에 과제가 자동 배정됩니다.
* 출제 가능 날짜 : 당일부터 6개월 이내

{% hint style="warning" %}
출제 당일보다 과거 날짜 혹은 6개월 이상 미래 날짜에는 과제 출제 불가
{% endhint %}
{% endtab %}

{% tab title="과제 분할 / 반복 설정" %}
수업 하나는 기본적으로 과제 1개로 부여되고 분량은 옵션으로 조절합니다.

* **수업 분할**: 하나의 수업을 몇 개 과제로 나눌지 설정합니다.
* **하루 분량**: 학생이 하루에 풀어야 할 과제 수를 설정합니다.

{% hint style="info" %}
수업 분할과 하루 분량을 함께 사용할 수 없습니다.&#x20;
{% endhint %}
{% endtab %}

{% tab title="수행 요일" %}
과제를 출제할 요일을 선택합니다.

* 선택한 요일만 캘린더에 과제가 자동 배정됩니다.
* 주 2\~3일 학습 등 유연한 계획에 적합합니다.
{% endtab %}

{% tab title="휴일 설정" %}
휴을 개별적으로 설정합니다.

* 토글을 켜면 캘린더에서 제외할 날짜를 직접 선택할 수 있습니다.
* 쉬는 날로 설정된 날짜에는 과제가 배정되지 않습니다.
{% endtab %}
