### 학생 삭제하기

#### • 학생 한 명 삭제하기

1. 상단의 **\[클래스]**&#xBA54;뉴를 클릭합니다.&#x20;
2. 왼쪽 탭에서 \[전체반(학원명)]을 클릭합니다.
3. 삭제하고자 하는 학생의 아이디를 클릭합니다.&#x20;
4. 하단의 **\[삭제]**&#xBC84;튼을 클릭합니다.&#x20;

#### •학생 한 번에 삭제하기

1. 상단의 **\[클래스]**&#xBA54;뉴를 클릭합니다.&#x20;
2. 왼쪽 탭에서 \[전체반(학원명)]을 클릭합니다.
3. 목록에서 여러 명의 학생을 체크박스로 선택합니다.
4. 상단 **\[삭제]** 버튼을 눌러 한 번에 삭제할 수 있습니다.

{% hint style="danger" %}
**삭제된 학생 정보는 복구할 수 없습니다.** 삭제할 학생이 맞는지 다시 한 번 확인해주세요.     &#x20;
{% endhint %}

### 학생 정보 수정하기

수정 방법

1. 상단 메뉴에서 **\[클래스]** 를 클릭합니다.
2. 왼쪽 탭에서 클래스를 선택한 후 수정할 학생을 찾습니다.
3. 학생 이름을 클릭해 정보를 수정합니다.

수정 가능한 정보는 다음과 같습니다

* 이름
* 비밀번호
* 소속 클래스 (내보내기만 가능)

{% hint style="info" %}
아이디는 수정할 수 없습니다.\
학생 아이디를 변경하려면 기존 학생을 삭제한 뒤, 새 아이디로 다시 추가해 주세요.
{% endhint %}

***

### 클래스에 학생 추가하기

1. 상단 메뉴에서 **\[클래스]** 를 클릭합니다.
2. 왼쪽 탭에서 학생을 추가할 클래스를 선택합니다.
3. 우측 상단 **\[학생 불러오기]** 버튼을 클릭한 후 **\[학생 계정 추가]** 를 선택합니다.
4. 학생의 아이디, 이름을 입력하고 저장합니다.

***

### 다른 클래스로 이동 / 복사하기

1. 상단 메뉴에서 **\[클래스]** 를 클릭합니다.
2. 왼쪽 탭에서 학생이 기존에 소속되어 있는 클래스를 선택합니다.
3. 이동 또는 복사할 학생을 체크박스로 선택합니다.
4. 우측 상단 **\[다른 클래스로 이동]** 또는 **\[다른 클래스에 복사]** 버튼을 클릭합니다.
5. 대상 클래스를 선택한 뒤 **\[확인]** 을 눌러 완료합니다.

{% hint style="info" %}
**이동**: 기존 클래스에서 제거되고 새 클래스로 이동

**복사**: 기존 클래스는 유지되고 새 클래스에도 추가됨
{% endhint %}
