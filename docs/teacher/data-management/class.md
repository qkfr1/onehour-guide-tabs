# 클래스에서 과제 관리하기

클래스 메뉴에서 학생들의 모든 과제 데이터를 한 눈에 관리할 수 있습니다.

## 학생별 바로가기 버튼

각 학생 행에서 5개의 바로가기 버튼을 통해 상세 정보에 빠르게 접근할 수 있습니다.<br>

{% step %}
#### 완료 버튼

학생이 완료한 과제 현황을 캘린더에서 확인할 수 있습니다.

**이동 화면**: 완료된 과제 현황 캘린더

**주요 기능**:

* 날짜별 완료한 과제 개수 표시
* \[학습정보보기] 버튼으로 해당 날짜에 완료한 과제 상세 내역 확인
{% endstep %}

{% step %}
#### 리포트 버튼

학생의 과제 수행 목록을 확인하고 개별 리포트에 접근할 수 있습니다.

**이동 화면**: 학생 전체 리포트&#x20;

**확인 가능한 정보**:

* 최종 수행 시각
* 과제명
* 과제 날짜
* 유형 (단어/문장)
* 테스트 유형 (단어 고르기, 어순배열 등)
* 진행 상태 (완료/수행 중)
* 점수
* 개별 리포트 보기

{% hint style="info" %}
목록에서 리포트 버튼을 눌러 각 과제의 상세 리포트를 확인할 수 있습니다.
{% endhint %}
{% endstep %}

{% step %}
#### 월간 버튼

학생의 월간 학습 현황 리포트를 확인할 수 있습니다.

**이동 화면**: 월간 리포트

**확인 가능한 정보**:

* **학습량**: 이번 달 단어, 문장, 더빙, 동시통역, 에세이, 해석 학습량
* **학습 변화**: 최근 3개월간의 학습량 추이 그래프
* **학습 완료 현황**: 과제/연습에서 끝까지 정답을 맞춘 비율
* **한번에 맞춘 정답 수**: 최근 3개월 간 첫 시도에 맞춘 정답 수

{% hint style="info" %}
월간 리포트는 공유하기 또는 인쇄하기 기능을 통해 학부모에게 전달할 수 있습니다.
{% endhint %}
{% endstep %}

{% step %}
#### 로그인 버튼

학생 계정으로 직접 접속하여 학생이 보는 화면을 확인할 수 있습니다.

**이동 화면**: 학생 계정 화면 (새 탭)

**활용 방법**:

* 학생의 학습 환경을 직접 확인
* 학생/학부모 문의 시 동일한 화면에서 안내 가능
* 선생님 계정은 그대로 유지되어 별도 로그아웃 불필요

{% hint style="warning" %}
\[확인] 버튼을 누르면 새 탭에서 학생 페이지가 열립니다.
{% endhint %}
{% endstep %}

{% step %}
#### 과제 완료율 버튼

학생에게 부여된 과제 현황을 캘린더 형태로 확인할 수 있습니다.

**이동 화면**: 부여한 과제 현황 캘린더

**주요 기능**:

* 월간/주간 단위로 부여된 과제 목록 확인
* 과제별 단어/문장 학습 진행 상태 확인
* 과제 선택하여 **복사** 또는 **삭제** 가능

{% hint style="warning" %}
캘린더에서 과제를 선택하면 해당 과제를 복사하거나 삭제할 수 있습니다.
{% endhint %}

**과제 완료율 정렬**

과제 완료율 열의 화살표를 클릭하면 학생 목록을 완료율 기준으로 정렬할 수 있습니다.

* **오름차순 정렬 (↑)**: 0%부터 낮은 학생부터 순서대로 정리
* **내림차순 정렬 (↓)**: 100%부터 높은 학생부터 순서대로 정리

{% endhint %}
