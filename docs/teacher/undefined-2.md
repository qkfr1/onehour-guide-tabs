# 📍 영작

#### I. 이런 분들에게 추천해요

* 단어를 바탕으로 영작 과제를 부여하고 싶은 분
* 내신 서술형 스타일로 영작을 지도하고 싶은 분
* 해석을 기반으로 영작 연습을 시키고 싶은 분
* 에세이 형태의 영작 학습을 진행하고 싶은 분



#### II. 이미지 보며 따라하기

1. **좌상단 "수업관리" 버튼 클릭 후 생성된 수업 중 영작 생성을 원하시는 수업을 선택해 주세요.**

* 생성된 수업을 갖고 있지 않는 경우는 [해당 페이지](https://1hour.gitbook.io/guide/teacher/lesson-management/create)를 먼저 확인해 주세요!

3. **아래 4가지 주제를 참고하여, 원하는 영작 내용을 클릭해 주세요.**

* 단어 영작: 단어로 영작하기
* 문장 영작 '서술형': 내신 서술형 스타일로 영작하기
* 문장 영작 '해석', 해석 통문장: 해석 보고 영작하기
* 에세이 설정: 에세이 영작하기
