# 학생 리포트 카카오 알림 발송하기

학부모(또는 학생)에게 학습 리포트를 카카오톡으로 자동 발송하는 기능이에요.&#x20;

한 번 설정하면 정해진 요일·시간에 알아서 발송되니, 매번 직접 안내하는 부담을 줄일 수 있어요.

> **이런 메시지가 발송돼요**&#x20;
>
> 학원명, 학생 이름, 과제 완료율, 학습량, 학습 시간 등이 포함되며, \
> 마지막에 상세 리포트 링크 버튼이 함께 전송됩니다.
>

>
>

## 시작하기 전에 준비해주세요

원활한 설정을 위해 아래 항목을 먼저 준비하면 좋아요.

* 학부모 또는 학생 본인의 **휴대폰 번호 목록** (010으로 시작하는 11자리)
* 학생이 많은 경우, **엑셀 파일로 한 번에 등록**하는 방법을 추천드려요

***

### &#x20;기본 정보 설정

상단의 알림명에 구분하기 쉬운 이름을 입력하세요. (최대 30자)

* 예: `1반 주간 리포트`, `초등부 월요일 알림`
* 비워두면 `알림_YYYYMMDD` 형식으로 자동 입력됩니다

#### 2. 수신자 선택

`학부모` 또는 `학생 본인` 중 누구에게 발송할지 선택하세요.

* **학부모**: 학부모 휴대폰 번호로 발송
* **학생 본인**: 학생 휴대폰 번호로 발송

#### 3. 발송 일정 설정

언제 발송할지 정해주세요.

* **요일**: 1개 이상 선택 (복수 선택 가능, 예: 월·수·금)
* **시간**: 오전 7시 \~ 오후 9시 사이 1시간 단위

**알아두세요**

* 통신사 사정에 따라 설정한 시간보다 몇 분 늦게 발송될 수 있어요
* 학생당 하루 1건만 발송됩니다
{% endhint %}

#### 4. 메시지 구성

알림톡에 어떤 학습 데이터를 포함할지 선택하세요.

**필수 항목 (해제 불가)**

**선택 항목**

* 학습 시간 (어제 / 이번 달 누적 / 하루 평균)
* 총 학습 현황 (전체 누적)
*   학습량 6종: 단어, 문장, 문제, 말하기, 영작, 해석

### 학생 선택 및 번호 등록

알림을 받을 학생 또는 학부모의 휴대폰 번호를 등록해주세요.

<figure><img src="../../.gitbook/assets/스크린샷 2026-04-30 오전 10.29.50.png" alt="" width="375"><figcaption></figcaption></figure>

#### 1. 학생 찾기

우측 상단에서 클래스를 선택하거나 `전체 클래스`로 모든 학생을 한 번에 볼 수 있어요. \
이름·학생 ID·번호로 검색도 가능합니다.

#### 2. 번호 등록하기

**A. 직접 입력** (학생 수가 적을 때)

각 학생 행의 번호 칸을 클릭해 직접 입력하세요. \
엔터를 누르거나 다른 곳을 클릭하면 즉시 저장됩니다.

**B. 엑셀 일괄 업로드** (학생이 많을 때 추천)

**엑셀 업로드** 버튼으로 한 번에 여러 학생의 번호를 등록할 수 있어요.

#### 3. 학생 선택

번호가 등록된 학생 옆 체크박스를 클릭해 발송 대상으로 선택하세요. \
마스터 체크박스로 전체 선택/해제도 가능해요.

{% hint style="warning" %}
**번호가 등록되지 않은 학생은 선택할 수 없어요.** 먼저 번호를 입력한 뒤 선택해주세요.
{% endhint %}

## 그 다음엔 어떻게 되나요?

* 설정한 요일·시간이 되면 알림톡이 자동 발송됩니다
* 발송 결과는 `예약된 알림` 페이지 하단의 **발송 이력**에서 확인 가능해요
* 예약 내용은 언제든 **수정·일시정지·삭제**할 수 있어요

***

## 학생 리포트 알림톡 요금 안내

#### 요금: 학원 학생 수 × 월 1,000원 (기본 플랜과 별도 청구)

예시

* 학생 30명 → 월 30,000원
* 학생 50명 → 월 50,000원
* 학생 100명 → 월 100,000원

결제 방식

* 최초 결제는 결제일 \~ 월말까지 일할 계산되며, 이후 매월 1일에 자동 결제됩니다
* 예: 4월 15일 구독 시작 → 4월분은 16일치만 결제, 5월 1일부터 한 달분 자동 결제

학생 수 변동

* 매월 1일 기준 학원에 등록된 전체 학생 수로 산정됩니다
* 월 중 학생이 변동되어도 당월 요금은 그대로 유지됩니다
* 다음 달 1일에 새 학생 수 기준으로 다시 청구됩니다

해지 시

* 해지하셔도 당월 말일까지는 정상 이용 가능합니다
* 중도 해지에 따른 환불은 없습니다
* 다음 달 1일부터 신규 발송이 중단됩니다

유의사항

* 등록된 카드로만 결제됩니다 (원아워 포인트 사용 불가)
* 원아워 기본 플랜(아카데미/BP) 구독자만 이용 가능합니다

## 자주 묻는 질문

<details>

<summary>Q. 하루에 두 번 알림을 보낼 수 있나요?</summary>

학생당 하루 1건만 발송 가능합니다. 이미 예약된 요일에 추가로 예약을 시도 시 예약이 완료되지 않습니다.&#x20;

</details>

<details>

<summary>Q. 발송 시간이 정확한가요?</summary>

설정한 시간에 발송이 시작되지만, 통신사 사정에 따라 몇 분 정도 늦게 도착할 수 있어요.

</details>

<details>

<summary>Q. 학생이 퇴원하면 어떻게 되나요?</summary>

퇴원 학생은 자동으로 발송 대상에서 제외됩니다. \
만약 발송 대상이 0명이 되면 해당 알림은 **자동 일시정지**되며, 학생을 추가한 뒤 수동으로 재개할 수 있어요.

</details>

<details>

<summary>Q. 자녀 두 명이 같은 학원에 다니면요?</summary>

각각 별도로 발송됩니다. 같은 휴대폰 번호여도 자녀별로 따로 알림톡이 도착해요.

</details>

<details>

<summary>Q. 발송 일정을 변경하고 싶어요.</summary>

`예약된 알림` 페이지에서 해당 알림의 **수정** 버튼을 눌러 언제든 변경할 수 있어요. \
일시정지 상태에서도 수정 가능합니다.

</details>

<details>

<summary>Q. 학부모 연락처가 바뀌면 어떻게 하나요?</summary>

중앙 컬럼의 학생 행에서 번호를 직접 클릭해 수정하면 즉시 반영돼요.

</details>
