# 클래스 관리

> **클래스는 학생을 그룹으로 관리하는 기본 단위**입니다.
>
> 필요에 따라 별도의 클래스를 생성해 학생을 그룹으로 나누고,\
> 각 그룹에 맞춰 수업이나 과제를 관리할 수 있습니다.

### 클래스 구분이 필요한 이유

다음과 같은 경우에도 편리하게 학습을 관리할 수 있습니다.

* 학생을 수준이나 반별로 구분하고 싶을 때
* 수업이나 과제를 그룹별로 따로 운영하고 싶을 때
* 클래스별 리포트를 따로 보고 싶을 때

{% hint style="warning" %}
&#x20;**기본반이란?**

* 회원가입 시 자동으로 생성되는 기본 클래스입니다.
* 학생의 동일한 이름으로 표시되며, 등록된 모든 학생은 이 클래스에 포함됩니다.
* 기본반은 삭제는 불가능하며, 이름만 수정할 수 있습니다.
{% endhint %}

### 클래스 이름 변경하기

1. 상단 메뉴에서 **클래스**를 클릭합니다.
2. 좌측 목록에서 이름을 변경할 클래스를 선택합니다.
3. 상단 클래스 이름 옆의 연필 아이콘(✏️)을 클릭하여 새 이름을 입력하고 `Enter` 키를 누르면 저장됩니다.

***

### 클래스 삭제하기

1. 상단 메뉴에서 **클래스**를 클릭합니다.
2. 삭제할 클래스를 선택한 상태에서, 우측 상단 **\[학생 불러오기]** 버튼 옆의 세 점 메뉴 `⋮`를 클릭합니다.
3. **\[클래스 삭제]** 를 선택하고 확인하면 해당 클래스가 삭제됩니다.

***

### 클래스에 학생 추가하기

1. 상단 메뉴에서 **클래스**를 클릭합니다.
2. 클래스를 선택한 상태에서, 우측 상단 **\[학생 불러오기]** 버튼을 클릭합니다.&#x20;
3. 추가하고 싶은 학생을 선택, \[추가] 버튼을 클릭합니다.&#x20;

***

### 클래스에서 학생 내보내기

1. 상단 메뉴에서 **클래스**를 클릭합니다.
2. 클래스를 선택하고 클래스에서 내보내고 싶은 학생을 선택합니다.
3. 우측 상단의 \[내보내기] 버튼을 클릭합니다.&#x20;
