# vii. 프롬프트 입력 → 타겟 문법 학습

#### **I. 이런 분들에게 추천해요**

* 레슨별 타겟 문법이 포함된 문장을 직접 만들기 번거로운 분
* 학교 시험 출제 범위에 딱 맞는 문법 문장을 빠르게 생성하고 싶은 분

#### **III. 이미지 보며 따라하기**

1. **우상단 "+ 수업 생성" 버튼 클릭 후 "본문 변형 문제" 선택해 주세요.**

<figure><img src="../../../../.gitbook/assets/&#x27;.jpeg" alt=""><figcaption></figcaption></figure>



2. **원하는 타겟 문법에 대한 프롬프트를 입력 후, \[다음] 버튼을 클릭해 주세요.**

> <mark style="color:blue;">**<예시 프롬프트>**</mark>\
> \ <mark style="color:blue;">**한국의 중1을 대상으로 조동사 will과 can을 활용한 문장 25개를 스토리로 만들어줘**</mark>\
> \ <mark style="color:blue;">**간접의문문은 의문문이 다른 문장의 일부로 쓰이는 형태이다. 「의문 사+주어+동사\~』 의 어순으로 쓰며, 의문문의 형태와 달리 do/does/did를 사용하지 않는다. 간접의문문은 알다(know), 궁금하 다(wonder), 말하다(tell), 묻다(ask) 등과 같은 동사의 목적어로 주로 쓰인다. 간접의문문을 쓸 때는 주어와 동사의 순서, 동사의 형태에 주 의해야 한다. guess think believe suppose 가 동사로 쓰일때 의문사의 위치가 바뀌는 문장까지 합쳐서 25개 만들어줘**</mark>

<figure><img src="../../../../.gitbook/assets/스크린샷 2026-04-22 오후 8.22.09.png" alt=""><figcaption></figcaption></figure>



3. **원하는 변형 문제 유형을 선택한 후, \[수업 생성] 버튼을 클릭해 주세요**

<figure><img src="../../../../.gitbook/assets/스크린샷 2026-04-22 오후 8.22.44.png" alt=""><figcaption></figcaption></figure>
