# 학생 계정 바로가기

과제가 학생에게 어떻게 보이는지 확인하려면?

버튼 하나로 선생님 로그인은 그대로 유지하면서, 새 탭에서 학생 화면을 바로 확인할 수 있습니다.

***

**사용 방법**

{% embed url="https://app.storylane.io/share/ec5yagg3xgx1" %}

**방법 1: 과제관리에서 접속**

1. 상단 메뉴에서 **과제관리** 클릭
2. 확인하고 싶은 **과제** 클릭
3. 학생별 현황에서 원하는 학생의 **\[학생으로 접속]** 버튼 클릭
4. 새 탭에서 해당 학생의 과제 화면이 열립니다

**방법 2: 클래스 관리에서 접속**

1. 상단 메뉴에서 **클래스** 클릭
2. 학생 클래스 관리 탭에서 학생 목록 확인
3. 원하는 학생의 **\[로그인 아이콘 🔓]** 클릭
4. 새 탭에서 해당 학생의 화면이 열립니다

***

**이런 점이 좋아요**

| 기존                  | 개선 후                   |
| ------------------- | ---------------------- |
| 학생 화면 확인하려면 로그아웃 필요 | 로그아웃 없이 새 탭에서 바로 확인    |
| 확인 후 다시 선생님 로그인 필요  | 기존 탭에서 선생님 계정 유지       |
| 화면 비교가 어려움          | 탭 전환으로 선생님/학생 화면 비교 가능 |

***

**⚠️ 주의사항**

학생 계정으로 실제 로그인된 상태이기 때문에, 이 화면에서 과제를 수행하면 **학생 계정에 데이터가 기록**됩니다.

* 과제 내용 **확인용으로만 사용**해주세요

***

**자주 묻는 질문**

**Q. 학생 화면에서 로그아웃하면 선생님 계정도 로그아웃되나요?**&#x20;

A. 아니요, 각각 별도의 탭에서 작동하므로 선생님 계정에는 영향이 없습니다.

**Q. 모든 학생 계정에 접속할 수 있나요?**&#x20;

A. 네, 선생님이 관리하는 클래스의 학생이라면 모두 접속 가능합니다.

**Q. 모바일에서도 사용 가능한가요?**&#x20;

A. 현재는 PC/웹 환경에서 지원됩니다.
