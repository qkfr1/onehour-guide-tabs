# task queue guide

수업이나 과제를 생성할 때, 실시간으로 진행 상황을 확인할 수 있는 기능입니다.

## 작업목록 확인하기

수업 또는 과제 생성을 요청하면, 화면 하단에 **진행 중 작업목록** 창이 나타납니다.

이 창에서는 현재 생성 중인 작업의 진행 상황과 완료 여부를 한눈에 확인할 수 있습니다.

## 작업 상태 안내

작업목록의 각 항목은 다음 상태 중 하나로 표시됩니다.

| 상태      | 설명                                                 |
| ------- | -------------------------------------------------- |
| **대기중** | 작업이 생성되었으나 아직 시작되지 않은 상태입니다.                       |
| **진행중** | 작업이 진행 중입니다. 과제의 경우 `과제 생성중 3/5개`처럼 진행률이 함께 표시됩니다. |
| **완료**  | 모든 작업이 성공적으로 완료되었습니다.                              |
| **실패**  | 작업 생성에 실패했습니다. 오류 메시지를 확인해 주세요.                    |

## 작업 필터링

작업목록 상단의 필터 버튼을 사용해 원하는 상태의 작업만 볼 수 있습니다.

* **전체**: 모든 작업 표시
* **진행중**: 현재 진행 중인 작업만 표시
* **완료**: 완료된 작업만 표시
* **대기중**: 대기 중인 작업만 표시

## 작업 항목 관리

### 개별 항목 닫기

각 작업 항목 우측의 **X** 버튼을 눌러 목록에서 제거할 수 있습니다.

### 모두 읽음 처리

완료된 작업이 여러 개일 때, 하단의 **모두 읽음** 버튼을 눌러 한 번에 정리할 수 있습니다.

## 오류 발생 시

작업 실패 시 항목에 오류 메시지가 표시됩니다. 메시지를 확인한 후 다시 시도해 주세요.

지속적으로 오류가 발생하는 경우, 고객센터로 문의해 주세요.
