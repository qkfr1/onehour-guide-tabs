# 일반 결제(기본)

원아워는 월 구독 방식의 유료 서비스를 제공합니다.

서비스 이용 요금은 학생 수에 따라 산정되며, 결제 기준 및 유의사항은 아래 내용을 참고해주세요.

### 요금 구조

* **과금 방식**: 학생당 5,000원
* **최소 결제 금액**: 50,000원
* **결제 수단**: 개인 카드 또는 법인 카드
* **정산 기준**
  * 중도 유료 전환 시: 전환일 기준으로 남은 기간 일할 계산
  * 유료 사용 중 학생 추가 시: 추가일 기준으로 남은 기간 일할 계산 → 다음 정기 결제에 반영
  * 유료 사용 중 학생 삭제 시: 삭제일 기준으로 남은 기간 일할 계산 → 다음 정기 결제에 반영

***

### 결제 흐름

1. 무료 체험 종료 시 자동으로 결제 팝업 노출
2. 카드가 등록되지 않았다면 카드 등록 화면으로 이동
3. 카드 등록 후 \[구독권 구매] 버튼 클릭 시 결제 완료

***

### 카드 등록 방법

1. 선생님 계정으로 로그인- 오른쪽 상단의 \[이름/기관명] 버튼을 클릭합니다.&#x20;
2. \[슈퍼 관리자] - \[카드 등록] 버튼을 클릭합니다.&#x20;
3. \[개인 카드] 또는 \[법인 카드] 중 카드 유형을 선택합니다.&#x20;
4. #### 카드 정보 입력하고 \[등록하기] 버튼 클릭합니다.&#x20;

***

### 카드 변경 방법

1. 선생님 계정으로 로그인- 오른쪽 상단의 \[이름/기관명] 버튼을 클릭합니다.&#x20;
2. \[슈퍼 관리자] - \[카드 변경]버튼을 클릭합니다.&#x20;
3. \[개인 카드] 또는 \[법인 카드] 중 카드 유형을 선택합니다.&#x20;
4. #### 카드 정보 입력하고 \[등록하기] 버튼 클릭합니다.&#x20;

***

### 일반 결제 환불 정책

* **구독 해지 시, 환불 정책은 아래 기준에 따라 적용됩니다.**
* **환불 요청 시, 구독권 사용 기간 조정이 반드시 필요합니다.**

| 시기        | 환불 요청 시기                     | 1:1 안내 내용 |
| --------- | ---------------------------- | --------- |
| 이전 달 결제 분 | -                            | 환불 불가     |
| 이번 달 결제 분 | 영업일 기준 2일 이내 사용/미사용          | 전액 환불     |
|           | 정기결제 후 7일 이내 미사용             | 전액 환불     |
|           | 정기결제 후 3일 이후 사용 혹은 7일 이후 미사용 | 일할계산 후 환불 |
