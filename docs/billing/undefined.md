# 변경되는 과금 정책

### 결제 및 환불 안내

원아워는 시트 기반 월 구독 모델로 운영되며, 학생 1명당 1시트를 기준으로 요금이 과금됩니다.\
구독권은 매월 1일에 선불로 결제되며, 결제 이후에는 환불이 제한됩니다.

**💡 시트 기반 과금이란?**

* ‘1시트 = 학생 1명’으로 간주하여 요금이 산정됩니다.
* 시트는 결제 월 동안 잠금되며, 중도 삭제 또는 변경 시 환불되지 않습니다.
* 삭제된 시트는 해당 월 내 재사용이 가능합니다. (다른 학생 추가하여 사용 가능)

***

### 일반 결제

#### **과금 기준**

| 과금 단위    | 학생 1명당 5,000원          |
| -------- | ---------------------- |
| 최소 결제 금액 | 50,000원 (10명까지 기본 제공)  |
| 결제 주기    | 매월 1일, 선불 결제           |
| 결제 수단    | 개인 또는 법인 카드 (슈퍼관리자 등록) |

> 학생 추가/삭제 시, **변경된 시트 수는 다음 결제 주기에 반영**됩니다. 현재 결제 월의 요금은 변경되지 않으며, 일할 계산은 제공되지 않습니다.

#### **환불 정책**

| 결제 후 미이용 (로그인, 수업 시작 등 없음) | 결제 후 7일 이내 전액 환불 가능            |
| -------------------------- | ------------------------------ |
| 서비스 이용 시작                  | 결제 후 2일 이내 전액 환불 가능, 이 외 환불 불가 |
| 결제 후 7일 경과                 | 환불 불가                          |
| 중도 퇴원/계정 삭제                | 환불 불가 (결제 월까지 사용 가능)           |

* 9월 3일에 학생 1명을 추가한 경우 → 추가 즉시 당월 남은 날짜만큼 일할 계산되어 즉시 결제됩니다.
* 9월 10일에 학생 1명을 삭제한 경우 → 9월 요금은 그대로 유지되며, 10월부터 제외됩니다.

***

### 학생 결제

#### **개요**

* 학생이 직접 카드로 구독료를 결제합니다.
* 선생님이 설정한 학생 요금에서 원아워 이용료(5,000원) 및 정산 수수료 및 세금이 공제된 후 **익월 10일에 정산**됩니다.

| 최소 결제 금액 | 10,000원                           |
| -------- | --------------------------------- |
| 결제 주체    | 학생 (법정대리인 명의 카드 필요)               |
| 보증금      | 매월 5만 원 선결제 → 정산 시 전액 환급 또는 일부 차감 |

#### **환불 정책**

> 학생 결제 플랜 역시 시트 기반 월 구독 모델로 운영되며, 아래 기준에 따라 환불이 제한됩니다.

| 결제 후 미이용 (로그인, 수업 시작 등 없음) | 결제 후 7일 이내 전액 환불 가능            |
| -------------------------- | ------------------------------ |
| 서비스 이용 시작                  | 결제 후 2일 이내 전액 환불 가능, 이 외 환불 불가 |
| 결제 후 7일 경과                 | 환불 불가                          |

주의 일할 환불은 제공되지 않습니다.

구독권 해지는 결제 이후 진행할 수 없으며, 결제된 월은 해당 계정이 잠긴 상태로 유지됩니다.

***

#### 자주 묻는 질문

**Q. 학생을 삭제했는데 이번 달 요금이 줄지 않아요. 왜 그런가요?**

> 원아워는 시트 기반 구독 모델로 운영되며, 학생을 삭제해도 해당 월에는 이미 시트가 잠금되었기 때문에 **삭제된 월의 요금은 환불되지 않습니다.** 다만, 삭제된 시트는 다른 학생으로 교체하여 사용할 수 있습니다.

***

**Q. 결제했는데 이용을 안 했어요. 환불되나요?**

> 결제 후 2일 이내일 경우 또는 7일 이내에 수업 시청/과제 제출 등이 없었다면 **전액 환불이 가능합니다.** 단, 7일 이내에 활동 이력이 있는 경우에는 **환불이 불가**합니다.

**Q. 중간에 구독 해지하고 싶은데 어떻게 하나요?**

> 원아워는 월 단위 선불 구독이기 때문에 결제 월 중간에는 구독 해지나 부분 환불이 불가합니다. 다음 달부터 결제를 중단하고 싶으시다면, 구독권 해지를 통해 구독을 종료해주세요.
