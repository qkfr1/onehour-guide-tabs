# 커리큘럼 공유하기

### 공유하는 방법

1. 내가 만든 커리큘럼 상세 페이지로 이동합니다.
2. \[공유하기] 버튼을 클릭합니다.
3. 공유받을 선생님의 ID를 입력합니다.
4. \[추가] 버튼을 클릭하면 공유 목록에 추가됩니다.
5. 여러 선생님에게 공유하려면 위 과정을 반복합니다.

### 공유 취소하기

1. 커리큘럼 상세 페이지에서 \[공유하기] 버튼을 클릭합니다.
2. 공유 목록에서 취소할 대상의 \[X] 버튼을 클릭합니다.
3. 해당 학원의 모든 선생님에게 공유가 해제됩니다.

> 주의: 공유를 취소하면 공유받은 선생님의 목록에서 해당 커리큘럼이 비활성화됩니다.
