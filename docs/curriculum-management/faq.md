# FAQ

#### 자주 묻는 질문

**Q. 공유받은 커리큘럼으로 낸 과제를 새 커리큘럼으로 저장할 수 있나요?**

아니요, 공유받은 커리큘럼으로 부여한 과제는 새로운 커리큘럼으로 저장할 수 없습니다. 직접 만든 커리큘럼의 과제만 저장 가능합니다.

**Q. 커리큘럼에 포함된 수업을 삭제하면 어떻게 되나요?**

해당 수업과 관련된 과제가 커리큘럼에서 자동으로 삭제됩니다. 커리큘럼 자체는 유지되며, 다른 수업의 과제는 영향받지 않습니다.

**Q. 공유 취소 후에도 이미 낸 과제는 유지되나요?**

네, 이미 부여된 과제는 그대로 유지됩니다. 학생들의 학습 현황도 정상적으로 확인할 수 있습니다.

**Q. 한 커리큘럼을 여러 학원에 공유할 수 있나요?**

네, 각 학원에 소속된 선생님의 ID를 각각 추가하면 여러 학원에 공유할 수 있습니다.
