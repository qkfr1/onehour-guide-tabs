# 커리큘럼 수정하기

내가 만든 커리큘럼의 정보를 수정할 수 있습니다.

### 수정 가능한 항목

* 커리큘럼 이름
* 권장 학습자
* 소개
* 권장 학년
* 썸네일 이미지 (이미지 영역 클릭하여 변경)

### 목록 순서 변경하기

커리큘럼 목록의 순서를 원하는 대로 정렬할 수 있습니다.

* 커리큘럼 항목을 드래그 앤 드롭하여 순서를 변경합니다.
* '내가 만든 커리큘럼'과 '공유 받은 커리큘럼'은 각각 독립적인 순서를 가집니다.
* 검색 중에는 순서 변경이 불가능합니다.

### 검색하기

* 상단 검색창에 커리큘럼 이름을 입력하면 실시간으로 필터링됩니다.
* 영문 검색 시 대소문자 구분 없이 검색됩니다.

> 참고: 공유받은 커리큘럼은 수정할 수 없습니다.
