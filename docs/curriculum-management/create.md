# 커리큘럼 생성하기

## 커리큘럼 생성하기

이미 부여한 과제들을 모아서 커리큘럼으로 저장할 수 있습니다.

### 생성 방법

1. 대시보드의 부여한 과제 현황에서 저장하고 싶은 과제들을 선택합니다. (최대 1일 10개의 과제 / 180일)
2. \[커리큘럼으로 저장] 버튼을 클릭합니다.
3. 커리큘럼 이름과 정보를 입력한 뒤 저장합니다.

### 입력 항목

> 참고: 공유받은 커리큘럼으로 부여한 과제는 새로운 커리큘럼으로 저장할 수 없습니다.
