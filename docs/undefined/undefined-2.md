# 커리큘럼 삭제하기

커리큘럼 삭제는 상황에 따라 다르게 동작합니다.

### 삭제 정책

<table><thead><tr><th width="331.50390625">상황</th><th width="405.9140625">삭제 가능</th></tr></thead><tbody><tr><td>내가 만든 커리큘럼 (공유 안 함)</td><td>✅ 가능</td></tr><tr><td>내가 만든 커리큘럼 (공유 중)</td><td>❌ 불가 , 공유 취소 후 삭제 가능</td></tr><tr><td>공유 받은 커리큘럼</td><td>✅ 가능</td></tr></tbody></table>

### 삭제 방법

1. 삭제할 커리큘럼 상세 페이지로 이동합니다.
2. \[삭제하기] 버튼을 클릭합니다.
3. 확인 모달에서 \[확인]을 클릭합니다.

> 참고: 커리큘럼을 삭제해도 이미 부여된 과제는 삭제되지 않습니다.
