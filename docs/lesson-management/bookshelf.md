# 책장 관리

> 책장은 여러 권의 책을 묶어 클래스나 학생에게 일괄 부여하거나 커리큘럼 단위로 정리할 수 있는 단위입니다.\
> 책과 마찬가지로 책장에 포함된 책은 학생에게 자습용으로 노출됩니다.

### 책장 생성하기

1. **수업 관리 > 책장** 탭으로 이동합니다.
2. 우측 상단의 **책장 생성** 버튼을 클릭하면, ‘새 책장’ 섹션에 기본 책장이 생성됩니다.
3. 새 책장의 제목을 클릭해 원하는 이름으로 수정한 뒤 **엔터**를 눌러 저장합니다.
4. `+`이미지를 클릭하여  추가할 책을 선택한 뒤 저장하면 책장이 완성됩니다.

***

### 책장에 책 추가하기

1. 책장 제목 오른쪽 끝의 **`⋮`버튼**을 클릭합니다.
2. 책 추가 버튼을 클릭합니다.
3. 추가할 책을 선택한 뒤 저장합니다.

***

### 책장에서 책 순서 바꾸기

책장에 추가된 책은 **드래그 앤 드롭**으로 순서를 변경할 수 있습니다.

1. 책장 내에서 위치를 바꾸고 싶은 책을 클릭한 상태로 유지합니다.
2. 원하는 위치로 드래그하여 이동시키면 즉시 저장됩니다.

{% hint style="info" %}
&#x20;책장에서 책 순서 변경 시, 학생에게 노출되는 순서도 함께 변경됩니다.
{% endhint %}

***

### 책장에서 책 제거하기

책 위에 마우스를 올리면  **`⋮` 버튼**이 나타납니다.\
2.**`⋮` 버튼**을 클릭하고 **제거**를 선택하면 책장 속 책이 삭제됩니다.

{% hint style="info" %}
책장에서 책을 제거해도 책 자체는 삭제되지 않으며, 책 탭에서 계속 확인할 수 있습니다.
{% endhint %}

***

### 책장 삭제하기

1. 책장명 오른쪽 끝의 **`⋮` 버튼**을 클릭합니다.
2. **삭제**를 선택하고 확인하면 해당 책장이 삭제됩니다.

***

### 클래스에 책장 열람 권한 부여하기

선생님께서 원하시는 책이 담긴 책장을 학생이 언제든 확인하고 자율적으로 학습할 수 있도록 하는 열람 권한을 부여하는 방법입니다.

1. 왼쪽의 **책장** 메뉴를 클릭합니다.
2. 권한을 부여하고자 하는 책장명 오른쪽 끝의 **⋮ 버튼**을 클릭합니다.
3. **\[클래스 설정] 버튼 클릭합니다.**
4. **클래스 선택 및 저장합니다.**
   * 책장을 열람할 수 있도록 허용할 클래스(반)를 선택하고 \[확인] 버튼을 눌러 완료합니다.
