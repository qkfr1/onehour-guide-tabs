# 책 관리

> 책은 여러 개로 분할된 수업을 **순차적으로 학습하거나 체계적으로 관리할 수 있도록 도와줍니다.**
>
> 수업을 주제나 커리큘럼에 맞춰 묶어두면,&#x20;
>
> 학생이 자습용으로 활용하거나 선생님이 전체 흐름을 한눈에 파악하기 쉬워집니다.&#x20;

### 책 생성하기

1. 상단 메뉴에서 **수업 관리**를 클릭해 왼쪽 탭 중 **책 탭**을 선택합니다.
2. 우측 상단의 **`+ 책 생성`** 버튼을 클릭합니다.
3. 생성된 **새 책의 제목을 클릭**, 원하는 제목을 입력한 뒤 **엔터**를 눌러 저장합니다.
4. `+ 수업추가` 버튼을 클릭, 추가할 수업을 고르고 **저장**을 눌러 마무리합니다.

{% hint style="info" %}
수업 추가 시, 중복으로 선택해 한 번에 여러 개 추가할 수 있습니다.
{% endhint %}

***

### 책에서 수업 제거하기

1. **내가  만든 책** 탭에서 수정 원하는 **책**을 클릭해 수업 상세페이지로 이동합니다.&#x20;
2. 삭제를 원하는 수업 제목 오른쪽 끝의 **⋮ 버튼**을 클합니다.
3. 목록에서 제거할 수업 오른쪽의 **`수업 제거` 버튼**을 클릭해 삭제합니다.

{% hint style="info" %}
수업제거 시 해 책에서 빠지지만, 수업 자체는 삭제되지 않습니다.
{% endhint %}

***

### 수업 정렬하기 (이미지넣기)

* **자동 정렬 (수업명 기준)**\
  책 상세 페이지에서 상단의 **‘수업명’ 항목을 클릭**하면 오름차순/내림차순으로 정렬됩니다.
* **수동 정렬 (드래그)**\
  책 상세 페이지에서 수업을 **클릭한 상태로 끌어 원하는 위치에 드롭**하면 순서를 자유롭게 바꿀 수 있습니다.

***

### 책 표지, 제목 수정하기(이미지넣기)

1. **책 탭에서 수정할 책을 클릭**하면 책 상세 페이지로 이동합니다.
2. 상단에서 **책 표지를 클릭하면 이미지 수정**,\
   **책 제목 영역을 클릭하면 제목 수정**이 가능합니다.
3. 수정한 내용은 자동 저장되며, 목록에도 바로 반영됩니다.

{% hint style="success" %}
표지를 설정하면 책을 시각적으로 더 쉽게 구분할 수 있습니다.
{% endhint %}

***

### 책 삭제하기

1. 책 목록에서 **삭제할 책에 마우스를 올리면 나타나는 ⋮ 버튼**을 클릭합니다.
2. 메뉴에서 **삭제**를 선택하고 확인하면 책이 목록에서 제거됩니다.

{% hint style="info" %}
책을 삭제해도 개별 수업은 삭제되지 않습니다.

책이 포함된 책장이나 클래스에 부여된 경우 함께 제거됩니다.
{% endhint %}
