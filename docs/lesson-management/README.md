# 수업 관리

> **수업**은 책, 영상, 음원, PDF, 이미지 등 다양한 콘텐츠를 기반으로 생성되는&#x20;
>
> **원아워 학습의 기본 단위**입니다.
>
> 콘텐츠에서 학습할 문장과 단어를 발췌해 수업을 구성하고,&#x20;
>
> 이를 바탕으로 수업을 하거나 과제로 출제할 수 있습니다.

{% tab title="📜수업(전체 수업)" %}
### 1. 전체 수업

모든 수업을 한눈에 확인할 수 있는 탭입니다.

* 내가 만든 수업과 공유 받은 수업이 모두 표시됩니다.
* 수업명, 콘텐츠 구성, 과제 출제 여부 등을 확인할 수 있어요.

{% hint style="success" %}
&#x20;**TIP**\
수업명을 클릭하면 해당 수업의 상세 화면으로 바로 이동할 수 있어요.
{% endhint %}
{% endtab %}

{% tab title="내가 만든 수업" %}
### 2. 내가 만든 수업

선생님이 직접 만든 수업만 필터링해서 보여줍니다.

* '수업 생성'에서 생성한 항목이 여기에 저장됩니다.
* 수업을 학생자습용으로 활용하려면 책이나 책장에 포함해야 합니다.

{% hint style="warning" %}
**주의사항**\
수업 단독으로는 학생이 수업보기에서 접근할 수 없습니다.  > 책 부여 방법 바로가기
{% endhint %}
{% endtab %}

{% tab title="공유 받은 수업" %}
### 3. 공유 받은 수업

원마켓 다운로드컨텐츠는 여기에 표시됩니다.

* 수업은 ‘책 단위’로 다운로드되며, 수업명은 ‘유닛명’ 기준으로 표시됩니다.
* 공유받은 수업은 편집이 불가능하며, 과제 출제 및 출력만 가능합니다
{% endtab %}

{% tab title="수업 태그" %}
### 4. 수업 태그

수업에 포함된 **콘텐츠의 형식**과 **학습 가능한 스터디 체인 종류**를 한눈에 보여주는 표시입니다.

{% hint style="info" %}
ex) `영상` `단어` `문장` `더빙`이 태깅되어 있다면

→ **영상 기반 수업이며**,

→ **단어, 문장, 더빙 체인 학습이 가능한 수업**이라는 의미입니다.
{% endhint %}
{% endtab %}

{% tab title="옵션" %}
### 5. 수업옵션

단어,문장, 문제

* 수업에 포함된 단어,문장, 문제의 개수를 표시하고 있습니다.&#x20;

수업 시작

* 칠판형 수업 시작 페이지로 바로 진입할 수 있습니다.

과제 생성

* 수업 과제를 생성할 수 있습니다.

워크시트

* 수업을 워크시트(시험지) 형식으로 인쇄하거나 PDF로 만들 수 있습니다.&#x20;

`⋮` 편집

* 수업을 삭제하거나 수정할 수 있습니다.
{% endtab %}

{% tab title="🔍검색창" %}
### 5. 검색창

수업명 또는 유닛명을 입력해 원하는 수업을 찾아볼 수 있습니다.
{% endtab %}
{% endtabs %}
