# 수업 생성하기

수업 생성 페이지에서는 다양한 자료를 활용해 원하는 형태의 수업을 빠르게 만들 수 있습니다.\
상단의 \[+ 수업 생성] 버튼을 눌러 아래 방법 중 하나를 선택하세요.

***

### \[중·고등] 내신·수행평가 100점 만들기

외부 지문이나 본문을 활용해 내신 대비 문제를 제작할 수 있습니다.

<figure><img src="../.gitbook/assets/스크린샷 2025-08-11 오후 5.06.43.png" alt=""><figcaption></figcaption></figure>

{% embed url="https://youtu.be/r7az6LxT_6M" %}

{% tabs %}
{% tab title="외부 지문 완벽대비" %}
외부 지문 이미지만 있으면 AI로 240개의 변형 문제가 포함된 원아워 수업으로 만들어 드려요.&#x20;

\[방법 안내]

1. 외부 지문을 찍은 이미지를 업로드합니다.&#x20;
2. 원하는 변형 문제 유형을 선택하고 \[수업 생성]을 클릭합니다.&#x20;
3. 완성된 수업을 \[수업관리] 페이지에서 확인합니다.
{% endtab %}

{% tab title="본문 텍스트" %}
직접 입력한 본문 또는 AI가 생성한 문장으로 문제를 제작해 드려요.<br>

\[방법 안내]

1. 본문 텍스트를 직접 입력하거나 AI 문장 생성을 선택합니다.
2. \[수업 생성]을 클릭 후, \[수업관리] 페이지에서 결과를 확인합니다.
{% endtab %}

{% tab title="나만의 문제은행" %}
보유 중인 기출 문제 이미지를 업로드해 개인 문제은행을 만들 수 있어요.<br>

\[방법 안내]

1. '문제, 지문, 선지까지' 모두 포함하여 찍은 기출 문제를 각각 촬영하여 30장 이내 업로드합니다.
2. 각 문제에 정답을 지정합니다.
3. \[수업 생성]을 클릭 후, \[수업관리] 페이지에서 확인합니다.
{% endtab %}
{% endtabs %}

### \[유·초등] 영어를 유창하게 말하고 쓰기

영상·음원을 활용해 듣기, 말하기, 쓰기 활동 중심의 영어 수업을 제작합니다.

<figure><img src="../.gitbook/assets/스크린샷 2025-08-11 오후 6.21.38.png" alt=""><figcaption></figcaption></figure>

{% tabs %}
{% tab title="유튜브" %}
영어 유튜브 영상 링크만으로 학습할 문장과 단어를 추출하고, 발화 싱크를 자동으로 맞춰드려요.<br>

\[방법 안내]

1. 15분 이내의 영어 유튜브 링크를 입력합니다.
2. 학생들에게 학습 시키고자 하는 변형 문제 유형을 선택해주세요.
3. \[수업 생성]을 클릭 후, \[수업관리]에서 확인합니다.
{% endtab %}

{% tab title="영상" %}
영어 영상으로 학습할 문장과 단어를 추출하고, 발화 싱크를 자동으로 맞춰드려요.<br>

\[방법 안내]

1. 200MB 이하, 15분 이내의 MP4 파일을 업로드합니다.
2. 학생들에게 학습 시키고자 하는 변형 문제 유형을 선택해주세요.
3. \[수업 생성]을 클릭 후, \[수업관리] 페이지에서 결과를 확인합니다.
{% endtab %}

{% tab title="음원" %}
영어 영상으로 학습할 문장과 단어를 추출하고, 발화 싱크를 자동으로 맞춰드려요.<br>

\[방법 안내]

1. 200MB 이하, 15분 이내의 MP4 파일을 업로드합니다.
2. 학생들에게 학습 시키고자 하는 변형 문제 유형을 선택해주세요.
3. \[수업 생성]을 클릭 후, \[수업관리] 페이지에서 결과를 확인합니다.

### 나만의 AI 수업

사진, PDF, 음원 등 다양한 자료를 조합해 맞춤형 수업을 제작할 수 있습니다.

<figure><img src="../.gitbook/assets/스크린샷 2025-08-11 오후 6.23.33.png" alt=""><figcaption></figcaption></figure>

{% tabs %}
{% tab title="사진" %}
사진 자료로 맞춤형 AI 수업을 빠르게 제작해 드려요.

\[방법 안내]

1. 영어 텍스트가 포함된 사진(10장 이내)을 업로드합니다.
2. 학생들에게 학습 시키고자 하는 변형 문제 유형을 선택해주세요.
3. \[수업 생성]을 클릭 후, \[수업관리] 페이지에서 결과를 확인합니다.
{% endtab %}

{% tab title="PDF" %}
PDF 자료를 기반으로 다양한 활동을 포함한 수업을 만들 수 있어요.

\[방법 안내]

1. 30MB 이하, 30쪽 이내의 PDF 파일을 업로드합니다.
2. 원하는 수업 형태와 활동 유형을 선택합니다.
3. \[수업 생성]을 클릭 후, \[수업관리]에서 결과를 확인합니다.
{% endtab %}

{% tab title="음원과 PDF" %}
음원과 PDF를 함께 활용해 종합형 수업을 제작할 수 있어요.&#x20;

학습할 문장과 단어를 추출하고, 발화 싱크, 페이지 싱크를 자동으로 맞춰드려요.<br>

\[방법 안내]

1. 음원 파일(25MB/15분 이하)과 PDF 파일(30MB/30쪽 이하)을 업로드합니다.
2. 학생들에게 학습 시키고자 하는 변형 문제 유형을 선택해주세요.
3. \[수업 생성]을 클릭 후, \[수업관리]에서 확인합니다.
{% endtab %}

{% tab title="커스텀 제작" %}
자료와 활동을 자유롭게 조합해 원하는 형태의 수업을 제작해 드려요.

\[방법 안내]

1. 사용할 자료가 파일의 한도보다 길거나 파일 없이 직접 타이핑하여 생성하고 싶다면 수동으로 입력해 줍니다.
2. \[수업 생성]을 클릭 후, \[수업관리] 페이지에서 결과를 확인합니다.
{% endtab %}
{% endtabs %}
