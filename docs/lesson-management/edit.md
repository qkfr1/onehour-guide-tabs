# 수업 수정하기

### 제목, 수업 썸네일 수정 방법

1. &#x20;상단 메뉴에서 **\[수업 관리]**&#xB97C; 클릭해주세요.
2. 생성하신 수업 목록 중, 문제를 추가할 수업의 오른쪽 끝에 있는 **\[⁝] 버튼**을 클릭해주세요.
3. **\[정보 변경]** 을 클릭하시면 수정 화면으로 진입합니다.

### 수업 수정 방법

1. &#x20;상단 메뉴에서 **\[수업 관리]**&#xB97C; 클릭해주세요.
2. 생성하신 수업 목록 중, 문제를 추가할 수업의 오른쪽 끝에 있는 **\[⁝] 버튼**을 클릭해주세요.
3. **\[수업 수정]** 을 클릭하시면 수정 화면으로 진입합니다.
4. 화면 최하단의 **초록색 \[수정하기]** 버튼을 눌러주세요.
5. 문장, 싱크, 단어, 문제, 책을 통해 수정이 가능합니다.&#x20;
