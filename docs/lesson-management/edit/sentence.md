# 문장 수정 방법

### 문장 에디터 접속하기

1. 수업관리에서 수정할 수업을 선택합니다.
2. **수업 수정** 버튼을 클릭합니다.
3. 상단 **문장** 탭을 선택합니다.

***

### 화면 구성

| 영역    | 설명                              |
| ----- | ------------------------------- |
| 번호    | 문장 순서 (자동 부여)                   |
| ∧ ∨   | 문장 순서 이동 버튼                     |
| 페이지   | PDF/이미지 수업의 페이지 번호 (영상 수업은 미표시) |
| 학습 문장 | 영어 문장 입력란                       |
| 번역 문장 | 한글 번역 입력란                       |
| X     | 문장 삭제 버튼                        |

***

### 기본 사용법

#### 문장 추가하기

{% tabs %}
{% tab title="하단에 추가" %}
1. 문장 목록 맨 아래 **\[+ 문장 추가]** 버튼을 클릭합니다.
2. 빈 문장이 추가되고 입력창에 포커스됩니다.
{% endtab %}

{% tab title="중간에 삽입" %}
1. 문장 사이에 마우스를 올립니다.
2. 나타나는 **+** 버튼을 클릭합니다.
3. 해당 위치에 빈 문장이 삽입됩니다.
{% endtab %}

{% tab title="키보드로 추가" %}
1. 문장 입력 중 맨 끝에서 **Enter** 키를 누릅니다.
2. 다음 줄에 새 문장이 추가됩니다.
{% endtab %}
{% endtabs %}

#### 문장 수정하기

1. 수정할 문장의 입력창을 클릭합니다.
2. 내용을 수정합니다.
3. 수정 사항이 있으면 저장 버튼이 주황색으로 변합니다.

#### 문장 삭제하기

{% tabs %}
{% tab title="개별 삭제" %}
삭제할 문장 오른쪽의 **X** 버튼을 클릭합니다.
{% endtab %}

{% tab title="전체 삭제" %}
1. 상단 툴바의 **전체 삭제** 버튼을 클릭합니다.
2. 확인 팝업에서 삭제를 승인합니다.
{% endtab %}
{% endtabs %}

#### 문장 순서 바꾸기

1. 이동할 문장의 **∧** (위로) 또는 **∨** (아래로) 버튼을 클릭합니다.
2. 한 칸씩 이동합니다.

#### 페이지 번호 입력하기

{% hint style="info" %}
PDF/이미지 수업에서만 페이지 번호 컬럼이 표시됩니다.
{% endhint %}

1. 페이지 컬럼의 입력창을 클릭합니다.
2. 해당 문장이 위치한 페이지 번호를 입력합니다.
   * 숫자만 입력 가능

***

### AI 기능 사용법

#### 단어로 문장 생성

1. **단어로 문장 생성** 버튼을 클릭합니다.
2. 난이도를 선택합니다 (하 / 중 / 상).
3. 저장된 단어를 기반으로 문장이 자동 생성됩니다.

#### 문장 번역

{% tabs %}
{% tab title="전체 번역" %}
1. **문장 번역** 버튼을 클릭합니다.
2. 모든 학습 문장이 한글로 번역됩니다.
{% endtab %}

{% tab title="범위 번역" %}
1. **문장 번역** 버튼 옆 드롭다운을 클릭합니다.
2. 시작 번호와 끝 번호를 입력합니다.
3. 해당 범위만 번역됩니다.
{% endtab %}
{% endtabs %}

{% hint style="warning" %}
이미 번역이 있는 문장도 덮어쓰기 됩니다.
{% endhint %}

#### 붙여넣기 (대량 추가)

1. **붙여넣기** 버튼을 클릭합니다.
2. 외부에서 복사한 텍스트를 붙여넣습니다.
3. AI가 자동으로 문장을 분리하고 영어/한글을 매칭합니다.

***

### 저장 및 복구

#### 저장하기

1. 수정을 완료한 후 **저장하기** 버튼을 클릭합니다.
2. 저장 전 확인사항:
   * 학습 문장이 비어있으면 저장 불가

#### 복구하기

실수로 잘못 수정했을 때 사용합니다.

1. **복구하기** 버튼을 클릭합니다.
2. 마지막으로 저장한 시점으로 되돌아갑니다.

{% hint style="danger" %}
저장하지 않은 모든 수정 사항이 사라집니다.
{% endhint %}

***

### 주의사항

#### 과제 출제 후 제한

{% hint style="warning" %}
이미 과제를 출제한 수업은 일부 기능이 제한됩니다.

* 문장 합치기 (Backspace) 불가
* 정렬하기 불가

학생들의 학습 데이터 보호를 위한 조치입니다.
{% endhint %}

#### 싱크 확인

음원/영상 수업은 문장 수정 후 **싱크 탭**에서 싱크가 맞는지 확인해주세요.

#### 지원 특수문자

아래 특수문자만 입력 가능합니다.

```
# . , " ' ! ? - : ; $ % ( ) [ ] { } _ ~ / |
```

***

### 자주 묻는 질문

<details>

<summary>페이지 번호 컬럼이 안 보여요.</summary>

영상 수업은 페이지 번호가 필요 없어 표시되지 않습니다.

</details>

<details>

<summary>저장 버튼이 비활성화되어 있어요.</summary>

빈 문장이나 빈 번역, 빈 페이지 번호가 있는지 확인해주세요.

</details>

<details>

<summary>문장 순서를 바꿨는데 과제에 반영이 안 돼요.</summary>

이미 출제된 과제는 기존 순서가 유지됩니다. 새로 출제하는 과제부터 반영됩니다.

</details>
