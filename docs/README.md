원아워는 영어 교육 현장을 위한 맞춤형 학습 플랫폼입니다.

AI 기반의 자동 수업 생성, 학습 리포트, 워크시트 출력 등 다양한 기능을 통해 수업 준비부터 관리까지 효율적으로 도와줍니다.

### 주요 기능

* **자동 수업 생성**\
  콘텐츠를 업로드하면 학습 가능한 문장과 단어를 추출해 수업을 자동으로 생성합니다.\

* **과제 부여 및 리포트**\
  생성한 수업을 기반으로 온라인 과제를 부여하고, 학생의 학습 결과를 리포트로 확인할 수 있습니다.\

* **워크시트 출력**\
  수업 내용을 기반으로 오프라인 학습지를 출력할 수 있어, 온·오프라인 병행 수업이 가능합니다.\

* **원마켓 콘텐츠 이용**\
  수준별/주제별로 구성된 커리큘럼 콘텐츠를 다운로드해 수업에 활용할 수 있습니다.
