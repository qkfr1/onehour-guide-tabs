# 커리큘럼 (1)

> 한 번 만들어 두면 클릭 한 번으로 과 제를 바로 부여할 수 있는 학습 과정 묶음입니다.

## 커리큘럼이란?

커리큘럼은 여러 수업에서 만든 과제들을 하나의 학습 과정으로 묶어 관리하는 기능입니다.&#x20;

반이 새로 개강할 때마다 같은 과제를 다시 설정하는 번거로움 없이, \
미리 만들어 둔 커리큘럼을 통해 바로 과제를 부여할 수 있습니다.

커리큘럼 탭은 크게 세 가지로 구성됩니다.

* **내가 만든 커리큘럼** — 선생님이 직접 생성한 커리큘럼 목록
* **공유받은 커리큘럼** — 다른 선생님으로부터 공유받은 커리큘럼 목록
* **공개된 커리큘럼** — 원아워 또는 선생님(이용자)가 공개한 커리큘럼 목록, 다운받아 사용 가능

***

### 1. 커리큘럼 만들기

{% embed url="https://app.storylane.io/share/wjham8wtrom4" %}

{% stepper %}
{% step %}
원아워에 로그인한 후 상단의 **과제 관리** 탭을 클릭합니다. 선생님이 부여한 과제 목록이 표시됩니다.
{% endstep %}

{% step %}
커리큘럼에 포함할 **과제들을 선택**합니다.&#x20;

특정 월 전체 과제처럼 여러 기간에 걸친 과제를 함께 선택할 수 있습니다.
{% endstep %}

{% step %}
과제 선택이 끝나면 **커리큘럼으로 저장** 버튼을 클릭합니다.
{% endstep %}

{% step %}
커리큘럼 이름, 추천 대상, 커리큘럼 설명, 권장 학년을 입력한 뒤 **저장** 버튼을 클릭합니다.
{% endstep %}

{% step %}
저장 완료 팝업에서 **확인** 버튼을 클릭하면 커리큘럼 상세 페이지로 이동합니다.&#x20;

1일차, 2일차 등 일자별로 어떤 학습 툴이 사용되는지 확인할 수 있습니다.
{% endstep %}
{% endstepper %}

***

### 2. 원하는 커리큘럼 찾기 (목록 정렬 및 검색하기)

{% stepper %}
{% step %}
### 커리큘럼 카드를 드래그해 순서를 변경합니다.

커리큘럼 목록에서 원하는 커리큘럼 카드를 클릭한 채로 원하는 위치로 드래그하면 순서가 변경됩니다.
{% endstep %}

{% step %}
### 검색 입력란으로 커리큘럼을 찾습니다.

커리큘럼이 많아 찾기 어려울 경우, 검색 입력란에 커리큘럼 이름의 일부를 입력하면 해당 커리큘럼이 즉시 필터링되어 표시됩니다.
{% endstep %}
{% endstepper %}

***

### 3. 커리큘럼으로 과제 내기

{% embed url="https://app.storylane.io/share/rvvfxsnqfhge" %}

{% stepper %}
{% step %}
커리큘럼 상세 페이지에서 **과제 내기** 버튼을 클릭합니다.
{% endstep %}

{% step %}
달력에서 커리큘럼을 **시작할 날짜를 클릭**합니다.&#x20;

선택한 날짜부터 커리큘럼 일정이 자동으로 배치됩니다.
{% endstep %}

{% step %}
**요일 설정** 후 **다음** 버튼을 클릭합니다.
{% endstep %}

{% step %}
과제를 부여할 **반을 선택**한 후 **과제 부여하기** 버튼을 클릭합니다.
{% endstep %}

{% step %}
과제 부여 완료 후 **캘린더에서 확인하기**를 클릭해 과제가 정상적으로 등록되었는지 확인합니다.
{% endstep %}
{% endstepper %}

> **💡 요일 설정으로 수업 일정에 딱 맞게 조정하세요**
>
> 주 3회 수업이라면 수업이 없는 요일을 제외하면 자동으로 해당 요일에만 과제가 배치됩니다.&#x20;
>
> 한 번만 설정하면 전체 커리큘럼 일정이 자동으로 조정됩니다.

## 커리큘럼 다른 선생님에게 공유하기

{% stepper %}
{% step %}
### 커리큘럼 상세 페이지에서 공유를 시작합니다.

커리큘럼 상세 페이지에서 **공유하기** 버튼을 클릭합니다.
{% endstep %}

{% step %}
### 공유할 선생님의 계정을 입력합니다.

공유할 선생님의 원아워 계정(이메일)을 입력하고 공유를 완료합니다.
{% endstep %}

{% step %}
### 공유 상태를 확인합니다.

공유가 완료되면 커리큘럼 정보 옆에 **공유 중** 라벨이 표시됩니다.
{% endstep %}

{% step %}
### 공유를 취소합니다.

공유를 취소하려면 다시 **공유하기** 버튼을 클릭해 해당 계정을 삭제합니다. 공유가 해제되면 라벨도 함께 사라집니다.
{% endstep %}
{% endstepper %}

> **⚠️ 공유 중인 커리큘럼은 삭제할 수 없습니다**
>
> 다른 선생님에게 공유 중인 커리큘럼은 삭제하기 버튼이 비활성화됩니다. 삭제가 필요하다면 먼저 공유를 취소한 뒤 삭제해 주세요.

***

## 커리큘럼 정보 수정 및 삭제하기

{% stepper %}
{% step %}
### 커리큘럼 정보를 수정합니다.

커리큘럼 상세 페이지에서 **정보 수정** 버튼을 클릭하면 커리큘럼 이름, 추천 대상, 설명, 권장 학년을 변경할 수 있습니다. 수정 후 저장 버튼을 클릭합니다.
{% endstep %}

{% step %}
### 커리큘럼을 삭제합니다.

커리큘럼을 삭제하려면 **삭제하기** 버튼을 클릭합니다. 삭제 전 안내 팝업이 표시되므로 내용을 확인한 뒤 삭제 또는 취소를 선택합니다.
{% endstep %}
{% endstepper %}

> **⚠️ 삭제된 커리큘럼은 복구할 수 없습니다**
>
> 삭제하기 전 팝업의 안내 문구를 반드시 확인하세요. 이미 과제가 부여된 반의 기존 과제에는 영향을 주지 않지만, 커리큘럼 자체는 복구되지 않습니다.
