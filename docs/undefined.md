# 커리큘럼

> 자주 사용하는 과제 묶음을 커리큘럼으로 저장하고, 동료 선생님과 공유하여 수업 준비 시간을 줄이세요.

## 커리큘럼이란?

커리큘럼은 선생님이 반복적으로 활용하는 과제들을 하나의 묶음으로 저장해두는 기능입니다.

한 번 만들어두면 매번 과제를 새로 구성할 필요 없이 커리큘럼을 불러와 바로 부여할 수 있어, 수업 준비가 훨씬 빨라집니다.

**이런 경우에 편리합니다**

* 매 학기 비슷한 순서로 과제를 내는 선생님
* 같은 학원의 다른 반 선생님과 수업 흐름을 맞추고 싶을 때
* 신규 선생님에게 기존 커리큘럼을 빠르게 전달하고 싶을 때

커리큘럼은 **최대 180일, 하루 10개** 과제로 구성할 수 있으며, 내가 직접 만든 커리큘럼과 다른 선생님께 공유받은 커리큘럼을 구분하여 관리합니다.

***

## 커리큘럼 만들기

{% stepper %}
{% step %}
### 대시보드의 **부여한 과제 현황**으로 이동합니다.
{% endstep %}

{% step %}
### 커리큘럼에 담고 싶은 과제들을 선택합니다.

하루 최대 10개, 최대 180일치 과제를 선택할 수 있습니다.
{% endstep %}

{% step %}
### **\[커리큘럼으로 저장]** 버튼을 클릭합니다.
{% endstep %}

{% step %}
### 커리큘럼 이름, 권장 학습자, 권장 학년, 소개, 썸네일 이미지 등 정보를 입력하고 저장합니다.
{% endstep %}
{% endstepper %}

{% hint style="warning" %}
**공유받은 커리큘럼으로 낸 과제는 저장 불가**

다른 선생님에게 공유받은 커리큘럼으로 부여한 과제는 새로운 커리큘럼으로 저장할 수 없습니다. 직접 만든 커리큘럼의 과제만 저장할 수 있습니다.
{% endhint %}

***

## 커리큘럼 정보 수정하기

{% stepper %}
{% step %}
### **\[내가 만든 커리큘럼]** 탭에서 수정할 커리큘럼을 선택해 상세 페이지로 이동합니다.
{% endstep %}

{% step %}
### 수정 가능한 항목을 변경합니다.

커리큘럼 이름, 권장 학습자, 권장 학년, 소개 텍스트는 해당 필드를 직접 수정하고, 썸네일 이미지는 이미지 영역을 클릭하면 새 이미지로 교체할 수 있습니다.
{% endstep %}

{% step %}
### 변경 사항을 저장합니다.
{% endstep %}
{% endstepper %}

{% hint style="info" %}
**공유받은 커리큘럼은 수정할 수 없습니다**

다른 선생님이 공유해준 커리큘럼의 내용은 수정이 불가합니다. 내용을 바꾸고 싶다면 원작자 선생님께 수정을 요청하세요.
{% endhint %}

***

## 커리큘럼 목록 순서 변경하기

{% stepper %}
{% step %}
### 커리큘럼 목록 페이지로 이동합니다.
{% endstep %}

{% step %}
### 순서를 바꾸고 싶은 커리큘럼 항목을 **드래그 앤 드롭**하여 원하는 위치로 이동합니다.
{% endstep %}

{% step %}
### **\[내가 만든 커리큘럼]** 탭과 **\[공유 받은 커리큘럼]** 탭의 순서는 각각 독립적으로 저장됩니다.
{% endstep %}
{% endstepper %}

{% hint style="info" %}
**검색 중에는 순서 변경이 되지 않습니다**

검색창에 텍스트가 입력된 상태에서는 드래그 앤 드롭이 비활성화됩니다. 순서를 변경하려면 검색창을 먼저 비워주세요.
{% endhint %}

***

## 커리큘럼 검색하기

{% stepper %}
{% step %}
### 커리큘럼 목록 상단의 **검색창**에 찾고 싶은 커리큘럼 이름을 입력합니다.
{% endstep %}

{% step %}
### 입력하는 즉시 실시간으로 목록이 필터링됩니다.

영문 이름은 대소문자 구분 없이 검색됩니다.
{% endstep %}
{% endstepper %}

***

## 커리큘럼 삭제하기

{% stepper %}
{% step %}
### 삭제할 커리큘럼의 상세 페이지로 이동합니다.
{% endstep %}

{% step %}
### **\[삭제하기]** 버튼을 클릭합니다.
{% endstep %}

{% step %}
### 확인 모달에서 \*\*\[확인]\*\*을 클릭하면 삭제가 완료됩니다.
{% endstep %}
{% endstepper %}

{% hint style="success" %}
**삭제해도 이미 부여된 과제는 유지됩니다**

커리큘럼을 삭제해도 이미 학생들에게 부여된 과제는 그대로 유지됩니다. 다만 해당 커리큘럼으로 새 과제를 부여하는 것은 더 이상 불가합니다.
{% endhint %}

***

## 다른 선생님에게 커리큘럼 공유하기

{% stepper %}
{% step %}
### 공유할 커리큘럼의 상세 페이지로 이동합니다.
{% endstep %}

{% step %}
### **\[공유하기]** 버튼을 클릭합니다.
{% endstep %}

{% step %}
### 공유받을 선생님의 **ID**를 입력한 뒤 **\[추가]** 버튼을 클릭합니다.

공유 목록에 해당 선생님이 추가된 것을 확인합니다.
{% endstep %}

{% step %}
### 여러 선생님에게 공유하려면 ID 입력과 \[추가] 버튼 클릭을 반복합니다.

각 학원 소속 선생님 ID를 각각 추가하면 여러 학원에도 공유할 수 있습니다.
{% endstep %}
{% endstepper %}

***

## 공유 취소하기

{% stepper %}
{% step %}
### 공유한 커리큘럼의 상세 페이지에서 **\[공유하기]** 버튼을 클릭합니다.
{% endstep %}

{% step %}
### 공유 목록에서 공유를 취소할 선생님 옆의 **\[X]** 버튼을 클릭합니다.
{% endstep %}

{% step %}
### 확인 후 해당 선생님의 목록에서 커리큘럼이 비활성화됩니다.
{% endstep %}
{% endstepper %}

{% hint style="warning" %}
**공유 취소 시 상대방 목록에서 즉시 비활성화됩니다**

공유를 취소하면 공유받은 선생님의 커리큘럼 목록에서 해당 커리큘럼이 즉시 보이지 않게 됩니다. 단, 공유 취소 전에 이미 부여된 과제는 학생들의 학습 현황과 함께 그대로 유지됩니다.
{% endhint %}

***

## 공유받은 커리큘럼 사용하기

{% stepper %}
{% step %}
### 커리큘럼 목록에서 **\[공유 받은 커리큘럼]** 탭을 클릭합니다.

새로 공유받은 커리큘럼은 목록 가장 하단에서 확인할 수 있습니다.
{% endstep %}

{% step %}
### 사용할 커리큘럼을 선택하여 과제를 부여합니다.

과제 부여, 목록 내 삭제, 목록 순서 변경은 자유롭게 할 수 있습니다.
{% endstep %}

{% step %}
### 공유받은 커리큘럼의 **내용 수정이나 다른 선생님에게 재공유**는 불가합니다.

필요한 경우 원작자 선생님에게 요청하세요.
{% endstep %}
{% endstepper %}

{% hint style="info" %}
**원작자가 공유를 취소하면 커리큘럼을 더 이상 사용할 수 없습니다**

공유가 취소된 커리큘럼은 목록에서 사라지며 새 과제를 부여할 수 없습니다. 하지만 이미 부여한 과제와 학생들의 학습 현황은 영향받지 않고 그대로 확인할 수 있습니다.
{% endhint %}
