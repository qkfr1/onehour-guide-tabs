# 원아워 워크플로우

원아워는 **수업 준비 → 수업 진행 → 과제 부여 → 학습 결과 확인 → 커리큘럼 생성**의 흐름이 하나의 플랫폼 안에서 자연스럽게 이어지도록 설계되었습니다.&#x20;

본 가이드는 처음 원아워를 도입하신 선생님께서 서비스의 큰 그림을 빠르게 파악하실 수 있도록 전체 흐름을 6단계로 정리한 문서입니다.

{% hint style="info" %}
각 단계의 세부 사용법은 단계별 상세 가이드에서 안내드리고 있으니,&#x20;

본 문서는 **"무엇이 가능한지"를 한눈에 확인하시는 용도**로 활용해 주세요.
{% endhint %}

## 워크플로우 한눈에 보기

| 단계            | 핵심 활동             | 주요 기능               |
| ------------- | ----------------- | ------------------- |
| **1. 콘텐츠 확보** | 수업에 사용할 자료 준비     | 원마켓 다운로드 / 수업 직접 생성 |
| **2. 수업 진행**  | 학생과 함께 학습         | 전자칠판 수업 / 학습지 출력    |
| **3. 과제 부여**  | 예습·복습용 온라인 학습 출제  | 7가지 학습 유형 과제        |
| **4. 학생 학습**  | 학생의 자기주도 학습 수행    | 더빙·영작·문제 풀이 등       |
| **5. 결과 확인**  | 학습 이력 및 성취도 점검    | 실시간 이력 / 리포트        |
| **6. 커리큘럼화**  | 우수 진도를 자산으로 저장·공유 | 커리큘럼 저장 / 선생님 간 공유  |

{% stepper %}
{% step %}
### 1. 수업 콘텐츠 확보하기

원아워에서 수업을 시작하시는 방법은 크게 두 가지입니다.

#### ① 원마켓에서 검증된 콘텐츠 다운로드하기

<figure><img src=".gitbook/assets/image (42).png" alt="" width="563"><figcaption></figcaption></figure>

원마켓은 출판사별·교재별로 정리된 교육 콘텐츠 마켓입니다.

* **출판사별 탐색** : 원하시는 출판사를 선택해 해당 출판사의 전체 교재를 살펴보실 수 있습니다.
* **키워드 검색** : `grammar`, `listening` 등 학습 목적에 맞는 키워드로 교재를 빠르게 찾아보실 수 있습니다.
* **시즌 콘텐츠** : 듣기 평가·모의고사는 **시험 종료 일주일 이내**에 업로드되어 즉시 활용 가능합니다.

{% hint style="info" %}
**다운로드 후 활용 팁**\
다운받으신 콘텐츠는 자동으로 **\[수업 관리]** 에 저장되어 언제든 다시 꺼내 쓰실 수 있습니다.&#x20;

연이어 다른 교재를 받으시려면 \[이어 받기], 바로 수업을 살펴보시려면 \[책 확인하러 가기]를 선택해 주세요.
{% endhint %}

#### ② 보유하신 자료로 수업 직접 생성하기

<figure><img src=".gitbook/assets/image (30).png" alt="" width="563"><figcaption></figcaption></figure>

이미 가지고 계신 자료가 있다면 \[수업 생성] 기능으로 간단히 온라인 학습으로 전환하실 수 있습니다.

지원되는 자료 형태는 다음과 같습니다.

* **유튜브 영상** : 링크 붙여넣기만으로 영어 영상 수업 생성
* **음원 파일**
* **PDF 문서**
* **이미지 (사진)**
* **모의고사 문서**

{% hint style="info" %}
**유튜브 수업 생성 예시**\
유튜브 링크를 붙여 넣으시고 원하시는 문제 유형을 선택한 뒤 \[수업 생성]을 누르시면, AI가 영상의 음성을 분석해 다음을 **자동으로 추출·생성**합니다.

* **단어** : 품사와 함께 뜻이 표시됩니다.
* **문장** : 영상 내 모든 영어 대사가 추출되며 해석도 자동 생성됩니다.
* **문제** : 문장 내용을 분석한 True/False 등의 문제가 자동 생성됩니다.
{% endhint %}
{% endstep %}

{% step %}
### 2. 수업 진행하기

준비된 수업은 \[학습 기능]을 통해 두 가지 방식으로 진행하실 수 있습니다.

#### ① 전자칠판으로 수업 진행

<figure><img src=".gitbook/assets/image (31).png" alt="" width="563"><figcaption></figcaption></figure>

\[수업 시작] 버튼을 누르시면 생성된 수업이 전자칠판에 표시되어, 학생들과 화면을 공유하며 함께 학습을 진행하실 수 있습니다.

#### ② 오프라인 학습지로 출력

<figure><img src=".gitbook/assets/image (33).png" alt="" width="563"><figcaption></figcaption></figure>

\[학습지 만들기]를 누르시면 온라인 수업을 즉시 오프라인 학습지로 전환하실 수 있습니다.

1. 왼쪽 패널에서 원하시는 학습 유형을 선택
2. \[추가하기] 클릭
3. 학습지 미리보기 확인 후 **인쇄** 또는 **PDF 저장**

{% hint style="info" %}
동일한 콘텐츠를 **온라인 수업 + 오프라인 학습지**로 동시에 활용하실 수 있어, 수업과 자습의 연계가 매끄럽습니다.
{% endhint %}
{% endstep %}

{% step %}
### 3. 과제 부여하기

수업 시간에 다룬 내용을 예습·복습으로 이어가도록 \[과제 내기]에서 다양한 유형의 온라인 과제를 출제하실 수 있습니다.<br>

#### 원아워에서 제공하는 7가지 학습 유형

<figure><img src=".gitbook/assets/image (35).png" alt=""><figcaption></figcaption></figure>

| 학습 유형      | 핵심 내용                 |
| ---------- | --------------------- |
| **콘텐츠 시청** | 수업 영상을 skip 없이 끝까지 시청 |
| **단어 학습**  | 스펠링·뜻 암기              |
| **단어 영작**  | 단어의 활용법 익히기           |
| **문장 학습**  | 문장 구조·문법 학습           |
| **말하기 학습** | AI 발음 평가가 포함된 더빙 학습   |
| **쓰기 학습**  | 문장 만들기 및 해석           |
| **문제 학습**  | 자동 채점·실시간 피드백         |
| **에세이 영작** | AI 실시간 피드백 기반 장문 작성   |

{% hint style="info" %}
학생의 학년·수준에 맞춰 과제 유형을 조합해 출제하시면 학습 효과를 극대화하실 수 있습니다.
{% endhint %}
{% endstep %}

### 5. 학습 결과 확인하기

<figure><img src=".gitbook/assets/image (36).png" alt="" width="563"><figcaption></figcaption></figure>

선생님 화면에서는 학생의 학습 진행 상황을 실시간으로 확인하실 수 있습니다.

* **실시간 수행 이력** : 학생이 과제를 수행하는 즉시 진행 현황이 반영됩니다.
* **상세 리포트** : 과제별 정답률, 학습 시간, AI 평가 결과 등을 리포트에서 확인하실 수 있습니다.

{% hint style="info" %}
학습 데이터를 바탕으로 학생별 맞춤 피드백을 제공하시거나, 학부모 상담 자료로 활용하실 수 있습니다.
{% endhint %}
{% endstep %}

{% step %}
### 6. 커리큘럼으로 저장하고 공유하기

원아워의 가장 강력한 기능 중 하나는, \
한 번 만들어 둔 과제 세트를 **재사용 가능한 자산**으로 저장할 수 있다는 점입니다.

#### 커리큘럼 저장 절차

<figure><img src=".gitbook/assets/image (40).png" alt="" width="563"><figcaption></figcaption></figure>

1. 저장하고자 하시는 기간의 과제를 선택 (예: \[4월 전체 선택])
2. \[커리큘럼으로 저장] 클릭
3. 참고 정보 입력 후 \[저장]

이렇게 저장하시면 **1개월 분량의 과제를 클릭 한 번으로 출제할 수 있는 커리큘럼**이 만들어집니다.

#### 저장한 커리큘럼 재출제 방법

<figure><img src=".gitbook/assets/image (41).png" alt=""><figcaption></figcaption></figure>

1. \[과제 내기] 클릭
2. 일정 설정
3. 대상 학생 선택 → 출제 완료

#### 다른 선생님과 공유하기

생성된 커리큘럼은 **다른 선생님께 공유**하실 수 있으며, 반대로 **다른 선생님께서 공유해 주신 커리큘럼도 즉시 사용**하실 수 있습니다.&#x20;

학원 내 우수 진도를 표준화하시거나, 동료 선생님의 노하우를 빠르게 흡수하실 때 유용합니다.
{% endstep %}
