## 플랜은 언제든 바꿀 수 있나요?

네, 플랜은 매월 변경하실 수 있습니다. 변경 방식에 따라 적용 시점이 다릅니다.

* **업그레이드(상위 플랜으로 변경)**: 신청 즉시 차액이 결제되며, 바로 해당 기능을 사용하실 수 있습니다.
* **다운그레이드(하위 플랜으로 변경)**: 신청 시점부터 다음 결제일까지는 현재 권한이 유지되며, 다음 결제일부터 새 플랜이 적용됩니다.

{% hint style="warning" %}
다운그레이드 시, 새 플랜에서 사용할 수 없는 AI 기능이 포함된 과제는 다음 결제일부터 비활성화됩니다.\
이미 **완료된 과제의 결과는 영구 보존**되어 학생·선생님 모두 계속 열람하실 수 있습니다.
{% endhint %}

## 수업 생성 AI 크레딧이란?

수업 생성 AI 크레딧(CP)은 문제 생성, 수업 자료 생성 등 AI로 수업 콘텐츠를 만들 때 사용하는 크레딧입니다.

각 요금제에는 월 결제 금액의 일정 비율에 해당하는 기본 크레딧이 매월 기본 제공되며, 지급 당월에 한해 이용하실 수 있습니다.\
**기본 제공량을 초과해 사용하시는 경우에만 추가 크레딧 구매가 필요**합니다. 추가 구매 크레딧은 1년 간 유효합니다.

{% hint style="info" %}
**크레딧(CP) 기준** 1 CP = 20원 (부가세 포함) 기준으로 환산됩니다.
{% endhint %}

| 플랜          | 기본 크레딧        |
| ----------- | ------------- |
| Basic       | 결제 금액의 10% 상당 |
| Speaking AI | 결제 금액의 12% 상당 |
| Writing AI  | 결제 금액의 12% 상당 |
| AI Pro      | 결제 금액의 15% 상당 |

## 학원별 볼륨 할인 (Enterprise)

학생 수가 많은 학원이나 다지점 프랜차이즈를 운영하시는 원장님께는 **볼륨 할인**을 별도로 안내드립니다.

월간 실제 사용된 학생 계정 수를 기준으로, 전체 계정에 동일한 할인율을 적용해 드립니다.

* **적용 대상**: 월간 실사용 계정 **100명 이상**
* **적용 방식**: 자동 적용이 아니며, 고객 요청 시 협의 후 적용됩니다.
* **다지점 합산 가능**: 프랜차이즈는 여러 지점의 계정 수를 합산하여 구간을 산정할 수 있습니다.
* **플랜 무관 동일 할인**: 모든 플랜에 같은 할인율이 적용됩니다.

| 월간 실사용 계정 수     | 할인율 | 7,900원 플랜 | 9,900원 플랜 |
| --------------- | --- | --------- | --------- |
| 100 \~ 299명     | 5%  | 7,505원    | 9,405원    |
| 300 \~ 999명     | 10% | 7,110원    | 8,910원    |
| 1,000 \~ 2,999명 | 15% | 6,715원    | 8,415원    |
| 3,000 \~ 9,999명 | 20% | 6,320원    | 7,920원    |
| 10,000명 이상      | 25% | 5,925원    | 7,425원    |

{% hint style="info" %}
* 할인율은 **초과분에만 적용하는 방식이 아니라, 전체 계정에 동일하게 적용**됩니다.
* 최소 보장 인원(MG) 방식이 아닌, **월간 실제 사용된 계정 수를 기준**으로 정산합니다.
* 알림 기능(학생당 +1,000원)은 별도 부과됩니다.
{% endhint %}

볼륨 할인에 관심 있으신 원장님께서는 채널톡 또는 담당 매니저에게 문의해 주세요.

## 연간 결제

월 결제 외에 **연간 결제**도 선택하실 수 있습니다. 1년치를 한 번에 결제하시면 **10% 할인**이 적용됩니다.

| 플랜          | 월 이용료  | 연간 결제 시 (학생 1인당, 10% 할인) |
| ----------- | ------ | ------------------------ |
| Basic       | 5,900원 | 63,720원 (월 환산 약 5,310원)  |
| Speaking AI | 7,900원 | 85,320원 (월 환산 약 7,110원)  |
| Writing AI  | 7,900원 | 85,320원 (월 환산 약 7,110원)  |
| AI Pro      | 9,900원 | 106,920원 (월 환산 약 8,910원) |

### 학생 추가

연간 결제 기간 중 학생을 추가하시면, 남은 기간만큼만 일할 계산하여 즉시 결제됩니다.

* 계산식: **(연간 가격 ÷ 365) × 잔여 일수 × 추가 학생 수**
* 연간 할인율은 그대로 유지되며, 잔여 기간만큼만 청구됩니다.
* 추가 결제 금액에 해당하는 크레딧도 함께 지급됩니다.

### 학생 삭제

연간 결제 기간 중 학생을 삭제하셔도 현재 결제 주기는 그대로 유지됩니다.

* 결제 주기 중에는 **환불이 발생하지 않습니다.**
* 비워진 자리(seat)는 **1년 이내에 다른 학생으로 교체**하실 수 있으며, 이 경우 추가 청구가 없습니다.
* 다음 갱신 시점에는 줄어든 학생 수를 기준으로 자동 갱신됩니다.

### 크레딧(CP)

* 연간 결제 시 1년간 사용하실 수 있는 크레딧이 **일괄 지급**됩니다.
* 연간 결제 기간 동안만 사용 가능하며, 기간이 만료되면 남은 크레딧은 소멸됩니다.

## 비즈니스 파트너 플랜

원아워를 단순한 비용이 아니라 학원의 새로운 수익 모델로 활용하고 싶은 원장님께는 비즈니스 파트너 플랜을 별도로 안내드립니다.

비즈니스 파트너 플랜은 학부모가 결제하고, 학원은 수익을 정산받는 구조입니다.

비즈니스 파트너 플랜의 경우 학생이 납부하는 요금 중 플랜 이용료를 기준으로 기본 크레딧이 제공됩니다.

</details>
