### 2. GitBook 가이드 페이지

## 콘텐츠 학습 과제

수업에 포함된 영상 또는 음원을 학생이 끝까지 시청하도록 과제로 부여하는 기능입니다.\
기존 과제(단어·문장·말하기 등)와 달리 점수가 아닌 \*\*진행률(%)\*\*로 결과가 표시됩니다.

***

### 어떤 수업에 낼 수 있나요?

영상 또는 음원이 포함된 수업에만 콘텐츠 시청 과제를 부여할 수 있습니다.

| 과제 부여 가능             | 과제 부여 불가   |
| -------------------- | ---------- |
| 유튜브·비메오 영상 수업        | PDF만 있는 수업 |
| MP4 영상 수업            | 이미지·텍스트 수업 |
| PDF + 영상(유튜브·비메오) 수업 | 비공개·삭제된 영상 |
| PDF + 음원 수업          |            |
| 음원 수업                |            |

{% hint style="info" %}
과제를 낼 수 없는 수업에서는 콘텐츠 학습 섹션 자체가 표시되지 않습니다.
{% endhint %}

***

### 과제 내는 방법 (선생님)

{% stepper %}
{% step %}
### 과제 선택

1. 과제 내기 화면에서 수업을 선택합니다.
2. 과제 정보 하단 **콘텐츠 학습** 섹션에서 **콘텐츠 시청** 칩을 클릭합니다.
3. 선택된 과제 목록에 '콘텐츠 시청'이 추가됩니다.
{% endstep %}

{% step %}
### 기간 및 옵션 설정

* 다른 과제(단어, 문장 등)와 함께 분할 과제로 설정할 경우, **콘텐츠 시청은 한번만** 옵션을 선택할 수 있습니다.
  * **ON**: 분할된 첫날에만 콘텐츠 시청 과제가 생성됩니다.
  * **OFF**: 분할된 날마다 콘텐츠 시청 과제가 생성됩니다.
* 콘텐츠 시청 과제만 단독으로는 분할 설정이 불가합니다.
{% endstep %}

{% step %}
### 클래스·학생 선택 및 완료

Step 3, 4. 클래스·학생 선택 및 완료 — 기존과 동일합니다.
{% endstep %}

### 학생 화면

#### 과제 목록

* 콘텐츠 과제는 **콘텐츠** 태그로 표시됩니다.
* 미시작 상태에서는 **과제 시작**, 진행 중에는 **이어서 하기** 버튼이 표시됩니다.
* 진행률(%)이 버튼 옆에 함께 표시됩니다.

#### 시청 화면

* 영상이 전체 화면으로 재생됩니다.
* **영상 앞으로 건너뛰기가 차단**됩니다. 클릭 시 "영상을 처음부터 끝까지 시청해야 해요" 안내가 표시됩니다.
* 다른 탭으로 이동하면 영상이 자동으로 일시정지됩니다. 탭으로 돌아오면 직접 재생 버튼을 눌러야 합니다.
* **진행률은 서버에 저장**되므로 중간에 나가도 이어서 볼 수 있습니다.

#### 완료 처리

1. 영상이 끝까지 재생되면 **시청 완료** 버튼이 나타납니다.
2. 버튼을 클릭하면 과제가 완료 처리됩니다.
3. 완료된 과제는 재진입할 수 없습니다. 영상을 다시 보려면 수업의 플레이 버튼을 이용하세요.

***

### 대시보드에서 확인하기

#### 캘린더

* 과제 카드에 **콘텐츠** 태그(브라운 컬러)가 추가됩니다.
* 진행 상태는 x(미시작) / △(진행 중) / O(완료)로 표시됩니다.

#### 과제 클릭 모달 — 학생별 현황

| 상태   | 표시 방식         |
| ---- | ------------- |
| 미시작  | -             |
| 진행 중 | 진행률% (노란색 배경) |
| 완료   | 100% (녹색 배경)  |

#### 개별 학생 과제 상세

* **진행률(%)**: 기존 "점수" 위치에 표시됩니다.
* **마지막 학습일**: 영상을 마지막으로 시청한 날짜입니다.
* **100점 처리 버튼**: 콘텐츠 시청 과제에는 제공되지 않습니다.

***

### 리포트에서 확인하기

개별 리포트에서 아래 항목을 확인할 수 있습니다.

| 항목      | 내용              |
| ------- | --------------- |
| 진행률(%)  | 시청 진행률 숫자       |
| 프로그레스 바 | 브라운 컬러로 진행률 시각화 |
| 완료 상태   | 완료 / 미완료        |
| 총 시청 시간 | 실제 재생된 누적 시간    |
| 완료 일자   | 완료한 경우에만 표시     |

{% hint style="info" %}
월간 리포트에는 반영되지 않습니다.
{% endhint %}

### 자주 묻는 질문

<details>

<summary>학생이 여러 기기에서 접속하면 어떻게 되나요?</summary>

항상 서버에 저장된 가장 높은 진행률 지점에서 이어서 재생됩니다. 두 기기에서 동시에 저장 요청이 오면 높은 진행률만 유지됩니다.

</details>

<details>

<summary>영상이 짧아도 동일하게 동작하나요?</summary>

네, 영상 길이와 관계없이 동일한 정책이 적용됩니다.

</details>

<details>

<summary>과제가 완료된 후 학생이 영상을 다시 볼 수 있나요?</summary>

완료된 과제로는 재진입이 불가합니다. 영상을 다시 보려면 수업 화면의 플레이 버튼을 이용해야 합니다.

</details>

<details>

<summary>네트워크 오류가 발생하면 진행률이 사라지나요?</summary>

진행률은 서버에 저장되므로 사라지지 않습니다. 오류 발생 시 재시도 버튼이 표시됩니다.

</details>
