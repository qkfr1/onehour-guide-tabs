# ranking\_guide

## 선생님용 가이드

### 기능 설명

학원에 소속된 전체 학생들의 학습 활동 순위를 한눈에 확인할 수 있는 기능입니다. 8가지 학습 항목별로 학생들의 성과를 파악하고 학습 현황을 모니터링할 수 있습니다.

**랭킹 항목 (총 8가지)**

| 항목    | 집계 기준     |
| ----- | --------- |
| 단어    | 맞은 단어 수   |
| 문장    | 맞은 문장 수   |
| 더빙    | 더빙한 문장 수  |
| 동시통역  | 동시통역 제출 수 |
| 단어 영작 | 영작한 문장 수  |
| 문장 영작 | 영작한 문장 수  |
| 에세이   | 에세이 제출 수  |
| 문제    | 맞은 문제 수   |

***

### 특징

* 학원 전체 학생의 순위를 항목별로 확인할 수 있습니다
* 과제와 연습 활동 데이터가 모두 포함되어 집계됩니다
* 월별로 최대 6개월 전까지 조회할 수 있습니다
* 각 랭킹 항목별로 개별 새로고침이 가능합니다
* 학습 활동이 없는 학생(데이터 0)은 랭킹에 표시되지 않습니다
* 점수가 동일한 경우, 가장 최근에 점수를 올린 학생이 상위에 위치합니다

***

### 접근 방법

상단 메뉴에서 **\[클래스]** → **\[랭킹]** 클릭

***

### FAQ

<details>

<summary>특정 학생이 랭킹에 보이지 않아요.</summary>

해당 월에 학습 활동(과제 또는 연습)을 하지 않은 학생은 랭킹에 표시되지 않습니다.

</details>

<details>

<summary>랭킹은 얼마나 자주 업데이트되나요?</summary>

학생이 학습을 완료하면 실시간으로 반영됩니다. 각 항목의 새로고침 버튼을 눌러 최신 데이터를 확인할 수 있습니다.

</details>

<details>

<summary>이전 달 랭킹도 확인할 수 있나요?</summary>

네, 화면 상단의 이전달/다음달 버튼으로 최대 6개월 전까지 조회할 수 있습니다.

</details>

<details>

<summary>동점인 학생들의 순위는 어떻게 결정되나요?</summary>

점수가 동일한 경우, 해당 항목에서 가장 최근에 점수를 획득한 학생이 상위에 위치합니다.

</details>

## 학생용 가이드

### 기능 설명

같은 학원 친구들과 학습 성과를 비교할 수 있는 기능입니다. 8가지 학습 항목별로 나의 순위와 상위권 학생들의 순위를 확인할 수 있습니다.

**랭킹 항목 (총 8가지)**

| 항목    | 집계 기준     |
| ----- | --------- |
| 단어    | 맞은 단어 수   |
| 문장    | 맞은 문장 수   |
| 더빙    | 더빙한 문장 수  |
| 동시통역  | 동시통역 제출 수 |
| 단어 영작 | 영작한 문장 수  |
| 문장 영작 | 영작한 문장 수  |
| 에세이   | 에세이 제출 수  |
| 문제    | 맞은 문제 수   |

***

### 특징

* 각 항목별 Top 5 학생과 나의 순위를 확인할 수 있습니다
* 내가 Top 5 밖일 경우, 나와 바로 위/아래 1명의 순위가 추가로 표시됩니다
* 나의 순위는 하이라이트로 강조되어 쉽게 찾을 수 있습니다
* 다른 학생의 이름은 개인정보 보호를 위해 마스킹 처리됩니다 (예: 송\*은, 테\*트)
* 내 이름은 마스킹 없이 전체가 표시됩니다
* 과제와 연습 활동 데이터가 모두 포함되어 집계됩니다
* 스피킹(더빙, 동시통역) 및 라이팅(영작, 에세이)은 다시 제출해도 모두 합산됩니다

***

### 접근 방법

웹

* 상단 메뉴에서 **\[랭킹]** 클릭

앱

* 하단 **\[학습 현황]** 탭 → **\[랭킹]** 선택

***

### FAQ

<details>

<summary>내 순위가 보이지 않아요.</summary>

해당 월에 학습 활동(과제 또는 연습)을 하지 않으면 랭킹에 표시되지 않습니다. 학습을 시작하면 랭킹에 등록됩니다.

</details>

<details>

<summary>다른 친구들 이름이 왜 가려져 있나요?</summary>

개인정보 보호를 위해 다른 학생의 이름은 일부가 마스킹 처리됩니다. 본인의 이름만 전체가 표시됩니다.

</details>

<details>

<summary>더빙이나 동시통역을 다시 하면 점수가 올라가나요?</summary>

네, 스피킹(더빙, 동시통역)과 라이팅(단어영작, 문장영작, 에세이) 항목은 다시 제출해도 모두 합산되어 점수에 반영됩니다.

</details>

<details>

<summary>이전 달 랭킹도 볼 수 있나요?</summary>

네, 화면 상단의 이전달/다음달 버튼으로 최대 6개월 전까지 확인할 수 있습니다.

</details>

<details>

<summary>친구와 점수가 같은데 순위가 다른 이유는 뭔가요?</summary>

점수가 동일한 경우, 해당 항목에서 더 최근에 점수를 획득한 학생이 상위에 위치합니다.

</details>
